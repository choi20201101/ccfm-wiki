# 루비알엔 앰플클렌저 — 얼굴광채/톤업 시장조사 (2026-04-28)

> 제품: 메라블 루비알엔 앰플클렌저 (RubyRN Ampoule Cleanser)
> 산출물: `canvas/04_glow_keyword_research.md`
> 사용한 플레이북: [[market-research-playbook]]

## 수집 규모
- 검색광고 keywordstool: 80 키워드
- 데이터랩 트렌드: 24개월 윈도우 (2024-03 ~ 2026-03)
- 다음카페 (Kakao API): 2,464건
- 네이트판 HTML: 593건
- YouTube Playwright + 댓글: 65건 중 **100만뷰+ 5건**, 50만+ 8건, 10만+ 13건

## 핵심 발견

### 1. 시장 추세 — PDRN(+27%) · 글루타치온패치(+423%) · 비타민B3(+94%) 동시 급등
일반 폼클렌저는 -27%대 감소했지만 "앰플/부스터형 광채 클렌저" 카테고리는 자체가 신생.

### 2. 결정적 인사이트 — "PDRN 손으로 바르면 겉돈다"
**유튜브 13만뷰 영상 댓글 [좋아요 104]**: "이게 아무리 좋은 PDRN이라도 그냥 손으로 바르면 입자가 커서 그런지 겉도는 느낌이 좀 있더라고요"

→ 루비알엔 카피 무기: "**손으로 바르는 앰플은 겉돌지만, 거품에 풀어 쓰는 클렌저는 다르다**" — 경쟁 카테고리(앰플)의 약점을 자기 기회로 전환.

### 3. 100만뷰 영상 후킹 패턴 (실측)
- 뭔몽 149만: **"⚠️제발 이거 쓰지마... 90%가 모르는"** — 부정 손실회피 + 비밀 호기심
- 현징이 146만: **"쌩얼도 예뻐지는"** + "내돈내산 파데프리" — 자기 권위 + 결과 약속
- 림온 137만: **"대부분 잘 모르는 찐 꿀팁"** — 비밀 + 위험회피
- 비아 136만: **"베이스 톤업과 지속력 팁"** — 기능형 검색 직타 + 댓글 [814] **"화장은 무조건 자연광이야"** = 자연광 무드의 진정성 합격선
- 톡톡꿀정보 109만: ⚠️ "**바세린과 이것 섞으면 기미 싹 사라집니다**" — "싹 사라집니다" 단어는 자사 사용 금지 (반려 위험), 단 "**A와 B 섞으면**" 호기심 갭 패턴은 차용

### 4. 네이트판 593건 정제 — 페인포인트 1차 발화 풀
직설 발화 빈도 1위:
- "얼굴이 칙칙하데요.....어떻게 하죠???"
- "얼굴이 누렇게 뜨고 피부가 칙칙해졌어요"
- "파데 했는데도 얼굴이 먼가 칙칙해"
- "사진 찍으면 나만 노란기 도는 그 톤"
- "거울 볼 때마다 거슬리던 그 결"

→ **자막 첫 줄 그대로 차용 가능**.

### 5. 7대 인사이트 매칭 결과
- 사회적 통념: 53건 (생리·호르몬·유전·나이)
- 사회적 증거: 30건 (3주만에/8일만에/완판/베스트셀러)
- 권위자: 55건 (피부과 의사 → ⚠️ 루비알엔은 절대 사용 금지, 대체: "임상연구센터 시험번호")
- 배경 상황: 77건 (스트레스 39 + 임신 27 + 야근 + 면접 + 결혼식)
- 장소: 181건 (올리브영 60 / 거울 앞 / 욕실 선반)
- 물건/성분: 367건 (콜라겐 117 / 세럼 69 / 앰플 61 / PDRN 7)
- 원초적 욕망: 274건 (탄력 124 / 동안 52 / 매끈 22 / 자신감 8)
