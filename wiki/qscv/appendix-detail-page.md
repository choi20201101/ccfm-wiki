# [QSCV Appendix] 상세페이지 점검 체크리스트

광고 유입 후 소비자가 **USP → Pain Point → 해결 근거**를 납득하고 구매로 이어지도록 하는 상세페이지 구조 체크리스트.

## 핵심 프레임
### 1. USP(Unique Selling Proposition) 명확화
- USP = 수많은 경쟁 속에서 우리를 선택해야 하는 이유
- 정의 필수: 어떤 욕구 해소? 우리만이 할 수 있는 이야기? 사회적·수치적 근거?
- 예
  - "명품 향수 조향사, 7시간 베이비파우더 향 바디로션"
  - "피부과 원장 개발, 바르자마자 탄력 170% 개선"
  - "4년 연구 5세대 다이어트 신소재, 18가지 체성분 개선"

### 2. Pain Point 구체화
- USP가 해결하는 욕구가 **불편 상황**으로 묘사되었는가
- 니즈 유형화

### 3. USP 상세페이지 두괄식
- 광고 USP가 상세 초입에서 **즉시 재확인**

### 4. 욕구·문제해결 흐름 반영
- 욕구 → Pain Point → 해결 과정 → 근거 → 설득 → CTA

### 5. 근거(증빙) 콘텐츠
- 임상·특허·수상·리뷰·Before&After

### 6. 구매 유도 장치
- 한정성 / 사회적 증거 / 리스크 해소 (AS·환불)
