# [QSCV Appendix] 퍼포먼스 캔버스 업데이트 및 재기획

집행 기간 후 성과 데이터로 **퍼포먼스 캔버스를 갱신**하고 다음 사이클을 재기획하는 루프.

## 핵심 프레임
### 1. 가설 검증 & 데이터 분석
- 지표: CTR, CPC, CVR, ROAS, VTR, CPV
- 유효 → 심화·확장 재설정
- 비유효 → 실패 원인 분석 후 개선

### 2. 캔버스 업데이트 항목
- **제품 본질**: USP 재검토, 리뷰 기반 욕구 반영
- **고객 정의**: 페르소나 재검토, 신규 페르소나 설정
- **경쟁사·시장 분석**: 월별 추이·시즌 반영, 레퍼런스 업데이트
- **근거 요소**: 성과 소재 간 유사성 확인
- **설득 요소**: 효율 높았던 서브 USP 업데이트
- **핵심 소구점**: 신규 소구점 도출 + 테스트 계획

### 3. 방향성 재기획
- 신규 가설, 타겟·소재 방향, KPI·실행 계획
- 디자인본부·미디어본부와 방향 공유

### 4. 신규 콘텐츠·캠페인 구성
- 영상 기획안 일정 / 제작 완료 시점 사전 논의
- 추가 A/B 테스트 전략

### 5. 광고주 공유·피드백
공유 포함 항목:
- 이전 방향 수립 사유
- 얻은 인사이트
- 새 방향·실행 계획 일정
- 가설 검증 리뷰 예정 시점
