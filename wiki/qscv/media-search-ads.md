# [QSCV] 검색광고 매체 운영 매뉴얼

네이버 검색광고(파워링크/쇼핑검색) + 구글 검색광고 공용. **키워드 유형화와 확장** 중심. 1046줄 / 6청크.

## 핵심 구조
### 1. 퍼포먼스 마케터를 위한 키워드 이해
- 경쟁사 대비 신뢰 확보 → 시장조사·키워드 발굴
- 기준: 리뷰, 사회적 통념, 반복되는 소비자 문제, 객관적 근거

### 2. 키워드 유형 분류

#### 2-1. 메인 키워드
- **브랜드 키워드**: 브랜드명 단독 / 브랜드+제품 / 브랜드+세부(가격·피부과·내돈내산·효과)
- **상품 키워드**: 제품군 일반 (아이크림/수분크림/레티놀크림). 뼈대 + 효능/타겟층/리뷰 확장
- **소비자 구매의도 키워드**: 고민 해결·구매 직전 (아이백크림, 레티놀아이크림, 눈가주름크림)

#### 2-2. 서브 키워드
- **감성 키워드**: 감정·분위기 자극 (다크서클완화 / 비건아이크림)
- **소비 타겟층 키워드**: 성별+연령 (40대 아이크림 / 남자아이크림)
- **경쟁사/제휴 키워드**
- **시즌/이슈 키워드**

### 3. 키워드 확장 실무
- 기본 + 효능 / 타겟층 / 리뷰 조합
- 성과 기반 재배분

### 4. 입찰·예산 운영
- 품질지수, 경쟁도, CPC 상한
- 시간대/요일/지역 조정

### 5. 랜딩 매칭
- 키워드 유형별 랜딩 분기: 브랜드 → 자사몰, 상품 → 카테고리, 구매의도 → 제품상세

### 6. 리포트·최적화
- CTR / 전환율 / 키워드별 ROAS
- 저성과 키워드 제외·상위 키워드 확장
