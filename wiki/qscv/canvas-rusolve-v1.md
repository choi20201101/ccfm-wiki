# 퍼포먼스 캔버스 v1 — 루솔브(Rusolve) 탈모

> **상태**: 초안 / 데이터 집행 전 **가설 베이스라인**. Part 7 캔버스 프레임에 따라 작성.
> **다음 액션**: 각 항목의 가설을 키워드 조사(네이버키워드도구·블랙키위·썸트렌드) + 경쟁사 VOC 리뷰 분석(아로나민·판시딜·마이녹시딜·닥터포헤어 등) + AI 심층리서치로 1회 검증 후 v2로 증분.

---

## 1. 제품의 본질 (Why it exists)
- **본질**: 탈모 초기·진행 단계 **예방·완화**를 위한 브랜드 (2026 런칭)
- **USP 가설** (복수 후보 → A/B 검증 대상):
  1. 성분 USP — (한국 임상/특허 성분 존재 시) 핵심 성분 중심
  2. 루틴 USP — 두피케어 + 보조제 통합 프로그램
  3. 가격/구독 USP — 지속 복용·두피케어 부담 → 구독 모델
- ⚠️ 확정 USP는 제품 스펙·임상·경쟁 구도 확인 후 고정 (현재는 공란으로 표시, 1차 리서치 후 기입)

## 2. 고객 정의 (Who)
### 인구통계 (가설)
- 1차 타겟: 28~38세 남성, 사무직/프리랜서, 가족력 있음, 정수리·M자 초기 신호 자각
- 2차 타겟: 22~27세 남성, 조기 탈모 민감층
- 3차 타겟: 30~40대 여성, 출산후·갱년기 탈모
### 욕구 (가설, 썸트렌드·커뮤니티 검증 필요)
- 표면 욕구: "더 빠지지 않게 하고 싶다", "숱이 많아 보이고 싶다"
- 심층 욕구: **젊어 보이고 싶다 / 자신감 회복 / 이성에게 매력적이고 싶다 / 노화 부정**
- 회피 욕구: 병원 처방(프로페시아) 부작용 두려움, 약 의존 저항감
### 페르소나 후보
- **박준호 (33세, IT 직장인, 미혼)**: 최근 정수리 사진 보고 충격. 병원 가긴 부담, 네이버·유튜브 검색. "자연 유래·부작용 없음" 키워드에 반응.
- **김예린 (35세, 워킹맘)**: 출산 후 앞머리 숱 감소. 탈모샴푸는 써봤지만 효과 의문. 리뷰·임상 근거에 민감.

## 3. 경쟁사 · 시장 분석 (가설 프레임)
- **직접 경쟁**: 닥터포헤어·그라펜·TS샴푸(샴푸축) / 판시딜·아로나민석세스(경구 OTC) / 마이녹시딜 토닉
- **대체재**: 병원 처방(프로페시아/아보다트), 모발이식, 가발·헤어픽서
- **시장 시그널**: 탈모 시장 YoY 성장, 2030대 조기탈모 증가, 여성 탈모 시장 확장
- **할 일**: 경쟁사 1~3위 쿠팡/스마트스토어 리뷰 AI 분석 → 긍정어/부정어 리스트업 → 우리 제품의 차별화 포인트 1~2개 도출

## 4. 근거 요소 (Evidence Stack — 수집 Todo)
| 유형 | 내용 | 상태 |
|------|------|------|
| 숫자 | 임상 효능 수치(모발 증가율/두께/탈락 감소 %) | ⏳ 제품팀 확인 |
| 사회적 권위 | 피부과/모발 전문의 자문, 대학병원 협업 | ⏳ 섭외 검토 |
| 통념 | "30대=탈모 신호 시작", "방치하면 돌이킬 수 없다" | ✅ 활용 가능 |
| 경쟁력 | 특허 성분 / 원료 농도 / 원료 원산지 | ⏳ 제품팀 확인 |
| 컨셉 | (예) "병원 가기 전에, 집에서 막는 골든타임 케어" | 🟡 후보 |
| 기능 포지셔닝 | (예) "12주 루틴 = 병원 진료 1회 비용" | 🟡 후보 |

## 5. 설득 요소 (Psychology Levers)
- **손실회피**: "지금 방치 → 3년 후 이식 비용 OOO만원" (Before/After 시각화)
- **권위**: 전문의 자문 / 공인 임상 / 학술 근거
- **사회적 증거**: 사용자 타임랩스 후기(12주차 비교컷), 누적 판매·재구매율
- **희소성·긴급성**: 런칭 한정 가격, 구독 쿠폰
- **상호성**: 무료 두피 진단 키트 / 샘플 트라이얼

## 6. 핵심 메시지 & 시각화
- **한 문장 후보**
  - (A) 손실회피형: "3년 뒤 후회하기 전에, 오늘부터 12주."
  - (B) 자존감형: "빠지는 걸 보는 대신, 되찾는 걸 보세요."
  - (C) 근거형: "임상 OO%, 전문의가 말하는 골든타임 케어."
- **비주얼 아이디어**
  - 1초컷 후킹: 정수리 줌인 → 충격 리액션 컷
  - B&A: 12주 타임랩스 (Before 어둡게 → 손실회피)
  - 숫자 그래픽: 임상 수치 + 기간
  - 페르소나 장면: 거울 앞 박준호 / 머리 말리는 김예린

## 7. 실행 플랜 (Launch Sprint)
### 매체 믹스 가설 (집행 전)
- **인지·해상도 확보**: Meta Advantage+ / GFA (리타겟 풀 빌드업)
- **구매 전환**: Google 쇼핑·브랜드 검색 / 네이버 브랜드 검색 (상표권 확보 필수)
- **바이럴**: 유튜브 숏츠·릴스 UGC·인플루언서 (Before/After 포맷)
### A/B 테스트 축 (3x3 매트릭스)
- 소재 축: USP-성분 / USP-루틴 / USP-가격
- 메시지 축: 손실회피(A) / 자존감(B) / 근거(C)
- 우선 테스트: `성분 × 손실회피` vs `루틴 × 자존감` vs `가격 × 근거`
### KPI (초기 런칭 벤치 가설)
- CTR > 1.2%, CPC < 1,000원, CVR > 1.5%, ROAS 1차 목표 200%
- VTR > 25% (숏폼 15초 기준)
### 랜딩·상세페이지 구조
- 문제제기(탈모 골든타임) → 해결(루솔브 메커니즘) → 근거(임상·전문의·후기) → 구매 유도(구독 할인)
- [[appendix-landing]] · [[appendix-detail-page]] 체크리스트 적용

## 8. 미검증 가정 & 리스크
- 제품 USP 미확정 → 임상·특허 근거 없으면 설득 카피 약화
- 광고 심의: **의료기기·기능성 인정 여부에 따라 표현 제한** (표시광고법 리스크) → 법무/RA 사전 검토 필수
- 여성 페르소나 추가 시 소재 별도 제작 필요 → 1차는 남성 타겟 집중 권장

## 9. 재기획 루프 (v2 진입 트리거)
집행 14일 후 `[[appendix-canvas-reupdate]]` 절차로 v2 생성:
1. 성과 지표 (CTR/CPC/CVR/ROAS/VTR) 가설 검증
2. 유효 소구점 → 심화·확장 / 비유효 → 원인 분석
3. 페르소나·USP·근거·설득 요소 업데이트
4. 신규 A/B 계획 수립 → 광고주(내부 BU) 공유
