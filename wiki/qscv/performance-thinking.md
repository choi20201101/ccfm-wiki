# [QSCV] 퍼포먼스 사고 확장 가이드

QSCV의 **사고 프레임 OS**. 마인드셋(5가치) → 퍼포먼스 기초 → 심리학 설득 → 문제해결 → 퍼포먼스 캔버스 → 콘텐츠 기획까지 전 과정의 사고도구를 정의. 1417줄 / 8개 청크로 가장 큰 문서.

## 목차 (8개 청크에 대응)
### Part 1. 마케팅 마인드셋 (5가치)
- **고객집착** — 경청, 지속 소통, 관심, 학습, 신뢰/유대
- **실행력** — 속도, 주간 A/B, 짧은 피드백 루프, 실패 공유
- **도전정신**
- **팀워크**
- **책임감**

### Part 2. 퍼포먼스 마케팅 기초
- 퍼포먼스 마케팅의 정의
- 중요 요소: 직접/간접 매출 이해
- 플랫폼·고객 이해 (모바일 스캔 패턴, 플랫폼별 행동 특성)
- 광고 세팅 개념: CBO / ABO / DCO

### Part 3. AI 개념 및 기본 활용
- 중요성
- 프롬프트 설계와 데이터 제공
- 마케팅 사례

### Part 4. 심리학적 접근 (무의식·인식·설득)
- **무의식**: 시선·후킹
- **인식**: 고객 상황·장면 공감
- **설득**: 결정타

### Part 5. 문제 도출과 해결 프로세스
- 문제 파악 방법
- 문제 해결 방법
- 실제 사례

### Part 6. 고객의 문화와 언어 이해
- 문화·언어 이해의 중요성
- 고객 문화 이해 방법론

### Part 7. 실행 단계별 가이드 — 퍼포먼스 캔버스
- 캔버스란?
- 주요 항목과 작성 방법
- 활용 사례

### Part 8. 콘텐츠 기획 및 분석 + 부록 용어
- 시장 조사 방법론
- 경쟁사 분석
- 콘텐츠 제작 전략
- 부록: 주요 마케팅 용어
