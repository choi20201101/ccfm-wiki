# [QSCV Appendix] 이미지 필수 기획 항목

이미지 소재 제작 전 **반드시 정의**해야 하는 항목 체크. 영상 기획 항목의 이미지 버전.

## 핵심 프레임
### 1. 광고 소재의 방향성
- 목적: 전환 / 참여 / 인지 중 어느 것인지 명시
- 예: "체취 고민 해결 중심 전환소재"

### 2. 핵심 USP · 페르소나
- 표현할 USP 1개 명시 (예: "12시간 체취 차단")
- 메인 타겟 페르소나 + 설정 근거

### 3. 스토리라인 구조
**문제 → 해결 → 근거 → 설득 → CTA**
- 적용 예시 (헤어라인 두피타투)
  - 문제: 정수리 넓어 스트레스, 시술 부담
  - 해결: 두피 착색 + 30초~5분 커버 / 2주 지속
  - 근거: 무자극 테스트 + 탈모 완화 기능성 성분
  - 설득: 제품 배경·타제품 지속력·커버력 비교·B&A
  - CTA: 3차 예약판매 49% + 샴푸 증정
