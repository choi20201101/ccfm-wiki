# [QSCV] META 매체 운영 매뉴얼

Facebook + Instagram 광고 실행 표준. AI 기반 자동화 · 숏폼 · 크리에이터 협업 · iOS14 이후 데이터 신뢰도 이슈까지 포함한 1024줄 매뉴얼.

## 핵심 구조
### 1. 매체 개요
- **소개**: 전 세계 최대 SNS, AI 기반 자동 최적화, 정교한 타겟팅
- **주요 특징**: AI 크리에이티브 최적화 / 숏폼 중심 / 개인정보 강화(iOS14) / 메시지·커뮤니티 / 크리에이터 협업
- **장점**: 쌍방향 소통, 정교한 타겟팅(맞춤/유사), 바이럴 확산, 온드미디어 결합, 다양한 포맷
- **단점**: 알고리즘 의존 예측불가, 계정정지/검수 이슈(뷰티/건기식/다이어트 취약), iOS14 이후 추적 신뢰도 저하

### 2. 광고 지면 & 포맷
- 이미지 / 카드뉴스(캐러셀 2~10장) / 컬렉션 / 동영상 / 스토리 / 릴스

### 3. 광고 집행 플로우
- 픽셀 + Conversions API 이중 연동 권장 (iOS14 대응)
- GA4 / 호스팅사 UTM 병행 분석으로 누락 최소화

### 4. 캠페인 유형 & 목적
- 인지도 / 트래픽 / 참여 / 잠재고객 / 전환 / 카탈로그 판매 / 매장 방문

### 5. 타겟팅 전략
- 맞춤 타겟 / 유사 타겟 / 관심사·행동 기반
- Advantage+ 활용 원칙

### 6. 소재 운영 원칙
- 초기 10장 제작 (디자인본부 표준)
- 저성과 소재 교체 주기

### 7. 리포트·성과 지표
- 주요 지표: CTR, CPM, CPC, CVR, ROAS, 빈도
- iOS 기여기간 설정

### 8~ 세부 운영, 정책 위반 대응, 트러블슈팅
