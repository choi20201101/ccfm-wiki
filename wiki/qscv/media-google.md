# [QSCV] Google 매체 운영 매뉴얼

Google Ads (검색 / 디스플레이 / YouTube / Gmail / 파트너) 운영 표준. 1497줄 / 8청크 — 매체 매뉴얼 중 최장.

## 핵심 구조
### 1. 매체 개요
- **소개**: 풀퍼널(인지→전환), 전 세계 90% 이용자 도달, 텍스트/이미지/동영상 포맷 일체
- **특징**: YouTube 포함 강력한 지면, 정교한 타겟팅, 목표 중심 머신러닝
- **장점**: 퍼스트파티 기반 리타겟/유사/맞춤형 잠재, GA 연동 행동 분석, 고도화 머신러닝
- **단점**: 엄격한 정책(뷰티·건기식·의료 랜딩까지 검토, 비승인 사유 불명), 복잡한 캠페인 구조, 외부 채널(스스·쿠팡) 전환 추적 불가, 예산 제어 한계

### 2. 매체 선택 가이드
- 업종·제품·목적별 선택 기준

### 3. 캠페인 유형
- 검색 / 디스플레이(GDN) / 실적 최대화(PMax) / 디맨드젠 / 비디오 / 쇼핑 / 앱

### 4. 타겟팅·입찰
- 맞춤형 잠재고객 / 유사 / 리마케팅
- 입찰 전략: 타겟 CPA, 타겟 ROAS, 전환수 최대화 등

### 5. 소재 표준
- 초기 15장 (리사이징 포함) — 디자인본부 기준
- 반응형 검색/디스플레이 (RSA/RDA) 에셋 구성

### 6. GA4 / 추적
- 전환 설정, 기여 모델, UTM 규칙

### 7. 리포트·최적화 루틴
- 머신러닝 재학습 피하기: 변경 주기·예산 변동 폭 관리

### 8. 정책·검수 대응
- 비승인 시 공통 사유 & 우회 체크
