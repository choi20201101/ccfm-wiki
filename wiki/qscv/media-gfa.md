# [QSCV] GFA (네이버 성과형 DA) 매체 운영 매뉴얼

네이버 성과형 디스플레이(GFA) 운영 표준. 수동 개입 비중 높은 매체 특성을 전제로 한 1290줄 / 7청크.

## 핵심 구조
### 1. 매체 개요
- **소개**: 네이버 프리미엄 지면(메인/지식인/블로그/카페/뉴스) 기반 CPC/CPM 성과형 DA
- **특징**: 자동 최적화 약함 → **수동 개입 전제**, 세분화된 데이터 확인 용이, 검색 세그먼트 기반 의도 타겟팅
- **장점**: 정교한 타겟팅 조합, 네이버 탐색 흐름 자연 노출, 초보자도 세팅 가능, 실시간 수정 운영
- **단점**: 소재 영향력 극대(같은 타겟·예산이라도 소재로 결과 갈림), 테스트 기간 길어짐, 머신러닝 빈약(네이버 내부 학습만), 수동 입찰 필요, 소재 검수 까다로움

### 2. 운영 원칙
- "사람이 직접 소재 선택·교체"가 META/Google 대비 중요
- 전환 캠페인은 자동 입찰 가능하나 그룹 예산 수동 조정 필요

### 3. 캠페인 유형
- 트래픽 / 전환 / 동영상 / 쇼핑 등

### 4. 소재 표준
- 초기 **30장** (리사이징 포함) — 디자인본부 기준 (META/Google 대비 최다)
- 소재 ON/OFF 빠르게 돌리는 운영 전제

### 5. 타겟팅
- 성별/연령/지역/관심사/디바이스/OS 조합
- 검색 기반 세그먼트: 강한 구매 의도 반영

### 6. 지면별 전략
- 메인 / 카페 / 지식인 / 블로그 / 뉴스 / 쇼핑

### 7. 검수 대응
- 사람 검수 기준 — 타 매체 대비 엄격
