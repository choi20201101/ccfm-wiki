# /gptim — GPT 광고 소재 배치 생성 가이드

> **분류**: domains / DA-creative · 자동화
> **최종 업데이트**: 2026-04-28
> **연관**: `da-creative.md`, `marketing-automation.md`, `usp-performance-canvas-research.md`, `tacit/creative-patterns.md`

## 1. 한 줄 정의

ChatGPT(비공식 백엔드, god-tibo-imagen) 으로 한국 퍼포먼스 광고 소재(피드/카드뉴스/포스터)를 N장 일괄 생성하는 스킬. 사용자에게 6질문 → 매트릭스 → 스모크 5장 → 풀배치.

## 2. 왜 만들었나

- 디자이너 의존 줄이고 A/B 테스트 양산 속도 ↑
- 메라블 루비알엔 앰플클렌저 100장 케이스 검증 후 일반화
- 한국 GFA/카카오/네이버 매체 심의 룰 사전 차단 (반려시트 학습)
- ChatGPT 계정 한도 안에서 API 키 없이 작동 (~/.codex/auth.json 재사용)

## 3. 입력 (반드시 6질문으로 받기)

| # | 항목 | 예시 | 비고 |
|---|------|------|------|
| 1 | 브랜드 / 제품 풀네임 | 메라블 루비알엔 앰플클렌저 150ml | 모든 카피에 1회 노출 강제 |
| 2 | 소재 크기 / 비율 | 1:1 / 4:5 / 9:16 / 16:9 | 기본 1:1 (피드) |
| 3 | 생성 장수 | 30 / 50 / 80 / 100 | 1장당 ~140초 |
| 4 | 시드 폴더 (4종) | face/, pd/, best/, canvas/ | 절대/상대 경로 |
| 5 | 금지 카피·반려 사유 | 키워드 텍스트 OR 반려시트 파일 | xlsx/json/md 모두 OK |
| 6 | 핵심 후킹 (선택) | 핑크광·임상수치·번들·14일 챌린지 | 강조 축 |

## 4. 단계 (BDH)

### BOB — Spec
- 8 후킹축 × 12 비주얼씬 × 카피풀 → N개 변형
- 모든 프롬프트에 브랜드 배지·서브카피·금지사항·얼굴시드 80% 규칙 강제

### DD — 실행
1. **step-00 환경 점검**: `~/.codex/auth.json`, npm 패키지, out/ 폴더
2. **step-01 자료 수집**: canvas/*.md + 반려시트 → USP/금지어 추출
3. **step-02 매트릭스**: `scripts/build-variants.mjs` → variants.json N개
4. **step-03 스모크 5장**: `node scripts/batch.mjs --start 1 --end 5`
5. **step-04 풀배치**: 사용자 컨펌 후 `--start 6 --end N` 백그라운드

### HARNESS — 강제 규칙

| 코드 | 규칙 |
|------|------|
| H1 | **얼굴 시드 80% 유사·완전 다른 인물** 프롬프트 강제 (딥페이크 방지) |
| H2 | 출력 `out/M<id>_<hook>.png` + `out/_progress.jsonl` |
| H3 | 동일 계정 1세션 → 순차 호출. 401→종료, 429→8s+1회 재시도 |
| H4 | 한국 GFA 심의 금지어 풀 (싹/제거/완치/약국/피부과/N일이면 등) |
| H5 | 보고 포맷: `✅ N장 완료 / 후킹 분포 / 평균 시간` |
| H6 | 생성 PNG 메인 컨텍스트 재로드 금지 (확인 요청 시에만) |
| H7 | 패키지가 401 일관 발생 시 백엔드 변경 가능성 보고 |

## 7. 한국 GFA/매체 반려 패턴 (학습 결과)

### 7-1. 단어 차원 차단

- **효과 단정**: 싹/쓱/사라져/청소/정화/제거/삭제/완치/치료/탈출/해결/흡착/녹여/직빵/깨끗/탱탱/매끈/없애
- **시간 단정**: N일이면/N초만에/10년 어려진 (단 "N일 챌린지" 허용)
- **권위 오인**: 약국/피부과/병원/의원/클리닉/원장/약사/의사/한의사/교수
- **지역 식별**: 강남/청담/압구정/명동/홍대 (지역 후킹은 비유 우회)
- **단정형**: 이거 하나로/한 번에/1등/최고/미쳤어요/장담
- **타사**: 카카오/인스타/카페 로고

### 7-2. 이미지 차원 차단

- B&A 비교 컷
- 피부 모공 확대
- 의료 오인 소품 (의사복/처방전/엑스레이/약국 간판)
- AI 신체 기형 (손가락·얼굴)
- 인체 변형/블러 왜곡 (딥페이크 의심)

## 8. 시드 얼굴 처리 규칙 (H1) — 중요

> "첨부 시드 얼굴의 분위기·헤어·메이크업·체형은 80% 유사 참고하되, 얼굴 자체는 완전히 다른 새로운 가상 인물을 생성. 식별 가능한 특징(눈·코·입·윤곽) 시드와 닮지 않게. 딥페이크/실존인물 합성 절대 금지."

이 문장을 모든 face seed 프롬프트에 강제 삽입.

## 9. 성능 / 비용

- **1장당 시간**: 80~230초 (평균 ~140초). 모델 reasoning + SSE 스트림 길이 변동
- **80장 총**: ~3시간
- **동시 호출**: 1개 세션 = 순차만. 병렬 시 429 확정
- **인증**: ChatGPT 계정 한도 차감 (Plus/Pro 이미지 생성 한도)

## 13. 알려진 제약 / 위험

- **백엔드 비공식 의존**: ChatGPT 백엔드 변경 시 401/4xx 가능. 패키지 버전 모니터링.
- **계정 정책 위반 가능성**: 사용 전 사용자 동의 필요.
- **한글 텍스트 렌더링**: 짧은 한글(8자/줄 이하)은 또렷, 장문은 흐려질 수 있음 → 메인 카피 짧게.
- **시드 이미지 크기**: 7MB 이상이면 base64 페이로드 부담. 누끼는 단일 시드(pd1) 권장.
