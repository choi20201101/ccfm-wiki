# 바이럴 패턴 (최재명 대표)

> 출처: i-boss 201건. 소스: [[wiki/sources/src-iboss-choi-jaemyeong.md]]

## 바이럴 본질

### 바이럴 = 좋아요/댓글/공유/저장의 전염
- 관찰: 그중 댓글·공유가 최강 전파 도구
- 조건: 판매자 관점 버리고 "소비자가 남에게 공유하고 싶은 욕망" 중심 기획
- 외적 보상(할인)보다 내적 보상(재미·실용·이타심·감성)이 자발적 공유 유도
- confidence: high

### 컨테이저스 6법칙 + 2
- 사회적 화폐·계기성·감정·대중성·실용성·이야기성
- 그중 "실용성 + 사회적 화폐"가 가장 강력
- 1.8만원 큐레이션 콘텐츠 → 180회 공유(공유당 100원) 검증

### 바이럴 4요소 균형
- 데이터 + 본능 + 트렌드 + 사회적 통념
- 데이터만으로는 안 됨 — 인간 6가지 본능(생존·소속·지위·호기심·두려움·쾌락) 동반

## 오프라인 → 온라인 바이럴

### 줄선 모습 = 리뷰보다 강력한 신뢰 신호
- 기대감 상승 + 실패 두려움 해소 동시
- 파격 혜택(사르르 꽃등심 100원)으로 실제 줄 형성 → 다각도 촬영 → 전채널 배포
- 가격경쟁을 벗어난 프리미엄 브랜드화 수단

### 진짜 목표는 단골
- 대세감은 일시적, 진짜 목표는 신뢰 기반 재방문율
- 혜택 기획→시간/인원 제한→광고→촬영→공유 5단계 루틴

### 경쟁심 + 낮은 진입장벽 = 다층 참여
- 가오나시 코스튬(정육점 초통령)으로 아이들 관심 → 가위바위보 쿠폰 → 보호자 전환
- 숏폼 조회수 50만 돌파 순간 매출 2배 급등
- 이벤트는 신규 획득용, 제품 품질이 재구매 결정 — 투트랙 필수

## 숏폼 바이럴 법칙

### 3초 내 끝까지 볼지 결정
- 다수 시청자는 3초 내 이탈 판단
- Before&After는 2~3초 내 직관 노출 (규제 회피 위해 실험·연출로 우회)
- 카피를 5~10자 단위로 쪼개 장면 교차 편집 → 몰입·호기심

### 숫자 + 순서의 위력
- 제목·썸네일에 숫자 삽입 → 임팩트·가독성·클릭률
- 번호식 순서 체계가 무의식적으로 다음 내용 궁금증 유발
- "새로 배웠다/더 알고 싶다" 감정이 남으면 구독·저장·공유 폭증 (17.3만 조회 사례)

### 상황극보다 정직한 일상
- 일하는 모습·청소 장면 같은 "정직한 일상"이 고객의 성실함 감지 유발
- 같은 영상도 대본(스토리)에 따라 결과 달라짐 — 공감·감정 자극이 매출 연결
- 60분 투자로 2천만원 매출 릴스 — 평소 일상 영상 축적 후 편집

## 유기 도달 레버리지

### 10개 개인계정 셀프 공유 = 폭탄 심지에 불
- 페북은 유기 도달을 먼저 적용 → 댓글·공유·좋아요로 리치 무료 확대
- 5만원으로 90만명 도달, 3초 조회당 0.1원 — 터지는 콘텐츠는 광고 효율도 극대화

### 인스타 명함 이벤트 + 틱톡 자동 인증 (마리모 사례)
- 월 5천 → 5억 성장, 바이럴 루프 설계
- Q&A·리뷰 기반 SKU 확장(5종 → 100종)

### 스탕달 신드롬 바이럴
- 심장 뛰는 감동 경험이 자발적 확산
- 모그라미 팬티 등 감정 극대화 사례

## 사회적 통념 편승

### 유행어·문화 코드 접목
- "마라탕 분자모자 팬티" / "골프 전용 향수" / "핑거루트=다이어트"
- 타겟 마음속 'A=B' 각인을 제품과 연결 (유행 종료 후 급락 주의)

### 77년 전통·지역 특산 등 문화 요소
- 누구나 인정하는 문화 결합 시 강력한 매출

### 전단지의 성공 공식
- 고기 사진 크게 + 사람 얼굴 포함 + 텍스트 최소 + 기대 카피
- 전단지엔 반드시 "보고 왔음" 입증 혜택 명시 (측정 위해)

## [2026-04-28] 100만뷰+ 클렌징/톤업 영상 후킹 패턴 5선 (실측)
> 출처: [[sources/src-rubyrn-cleanser-glow-research-2026-04-28]] (Playwright 크롤 65건 중)

### 패턴 1: ⚠️ "제발 이거 쓰지마..." + "90%가 모르는"
- 관찰: 뭔몽 149만뷰 "⚠️제발 이거 쓰지마... 90%가 모르는 클렌징 방법" — **부정 손실회피 + 비밀 호기심 동시**
- 조건: 뷰티/스킨케어 카테고리, 카테고리 권위 채널 (구독자 1만+)
- confidence: medium (1회 100만+, 다른 i-boss 사례와 결 일치)

### 패턴 2: "대부분 잘 모르는 [카테고리] 찐 꿀팁"
- 관찰: 림온 137만뷰. **"대부분 잘 모르는"** 비밀 + **"트러블 안 나는"** 위험회피 + **"찐 꿀팁"** 약속 = 3중 후킹
- 조건: 뷰티/생활 정보 영상, 정직 어법 채널
- confidence: high (림온 137만 + 80만 두 영상 모두 적중)

### 패턴 3: "쌩얼도 예뻐지는 / 내돈내산 파데프리"
- 관찰: 현징이 146만뷰. "쌩얼도 예뻐지는" + "내돈내산 파데프리" — **자기 권위(내돈) + 결과 약속**
- 조건: 뷰티 일상 채널, 학생/청년 톤
- confidence: medium

### 패턴 4: 댓글 [814] "자연광이 진실 판정 기준"
- 관찰: 비아 bia 136만뷰 영상에서 댓글 좋아요 814개의 **"화장은 무조건 자연광이야"** — 시청자는 자연광에서의 진정성을 판정 기준으로 삼음
- 조건: 톤업·광채·B/A 영상의 합격선
- 적용: 광고 이미지·영상에 **자연광 모티프 (창가 햇살, 모닝 라이트)** 명시 필수
- confidence: high (단일 댓글이지만 814 라이크 = 시청자 공통 인식)

### 패턴 5: "엄마/친구가 모르고 발랐다가 놀라더라" 3인칭 목격담
- 관찰: 현징이 영상 댓글 [231] "우리 엄마가 바를꺼없다고 내꺼 발랐는데 ... 엄마 바른거 보고 놀래서 소리지름"
- 조건: 직접 B/A 비교가 규제로 막혔을 때, 가족·친구 반응으로 우회
- 적용: GFA 반려 회피 + 시청자 공감 동시 (이미 [[creative-patterns]] §B/A 12가지 우회 §3인칭 목격담과 동일 결)
- confidence: high (자체 + 외부 viral 사례 일치)

## [2026-04-28] "PDRN 흡수 의심" — 통념 깨기 후킹 (Negative Framing)
- 관찰: 광고없는 PDRN TOP3 (뷰케터 13만뷰) 영상 댓글 [104] **"PDRN이라도 그냥 손으로 바르면 입자가 커서 그런지 겉도는 느낌"**
- 시사: 소비자는 "비싼 PDRN 앰플 사는데 흡수 안 되는 듯한" 의심을 갖고 있음
- 적용: 클렌저/마스크 카테고리는 **"손으로 바르면 겉돌던 PDRN, 거품/제형으로 풀어 쓰면 다르다"** 메시지로 앰플 카테고리의 약점을 자기 기회로 전환
- 조건: PDRN 함유 클렌저·마스크·토너 카테고리
- confidence: medium (1회 viral 댓글, 다른 PDRN 영상에서도 비슷한 결 댓글 산발 등장)
