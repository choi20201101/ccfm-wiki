# 심리·설득 원칙 (최재명 대표)

> 출처: i-boss 201건 + 40개 심리법칙 4부작. 소스: [[wiki/sources/src-iboss-choi-jaemyeong.md]]

## 원초적 욕구 계층

### 5대 원초 욕구 기반
- 관찰: 식욕·안전욕·수면욕·성욕·인정욕이 모든 제품의 뿌리
- 욕구 → 욕망 진화로 새 산업 탄생(먹방 유튜버·명품·성형)
- 적용: "없던 것을 만들 수 없다" — 기존 욕구의 현대적 재포장

### 3중 자극 공식
- 도파민(쾌감) + 공포(미래 불안) + 무의식(습관) 동시 작동 시 효과 극대화
- 예: "지금 안 사면 피부과 월 100만 원" 식 미래 손실 구체화
- confidence: high (DA 카피 사전 시리즈 검증)

### 인간 6대 본능 (바이럴 기반)
- 생존·소속·지위·호기심·두려움·쾌락
- 데이터만으론 바이럴 안 됨 — 본능을 건드려야 반응
- 바이럴 공식: 데이터 + 본능 + 트렌드 + 사회적 통념

## 설득 9요소 (반복 등장)

1. **상호성**: 선제적 호의(무료 샘플·자가진단 PDF) → 호혜 전환율 상승
2. **희소성**: 한정수량은 "그럴듯한 사유" 있을 때 극대화
3. **권위**: "의사" 아닌 "성형외과 20년 의사가 지방 재배치 기반 개발" — 구체성=권위
4. **일관성**: 에펠탑 효과 — 동일 USP 반복 노출로 호감 형성
5. **사회적 증거**: 3명 이상 후기·인파 사진으로 CTR 2배
6. **호감**: 후광 효과 — 의사·PT코치 직업 신뢰 (월 1억→30억 사례)
7. **숫자**: 정량 수치 포함 카피가 CTR 압도. "80% 재구매율"보다 "10명 중 8명"
8. **공포**: 소크라테스 효과 — 공포로 고객이 스스로 답에 도달
9. **쾌락**: 도파민 보상 구조

## 무의식·통념 활용

### 무의식 스위치 ON → 인식 강화 → 설득
- 관찰: 3단 순차 배치. CTR 1% → 3~4%
- 조건: 무의식 욕구 건드림 → 공감 → 설득 요소 3개 이상

### 사회적 통념 편승
- "분자모자 팬티" = 마라탕 유행 편승
- "골프 전용 향수·팬티" = 골프 신드롬 편승 (유행 종료 후 급락)
- "핑거루트 = 다이어트" 통념 형성 → 일 1억+
- 원칙: 타겟 마음속 'A=B' 각인을 찾아 제품과 연결

### 학습된 무의식 (뇌 화학적 습관)
- 인간 행동의 50%+가 습관 — 반복 타겟 카피가 최고 반응
- "장어=정력" 같은 연상 단어 연결이 설명보다 효과적

## 카피 심리학

### 고객이 듣고 싶은 말로 전환
- "내가 하고 싶은 말 → 상대방이 듣고 싶은 말"
- SNS 썸네일에서 근본 욕구(적은 노력·경제적 성공) 직접 표현
- 40만원 상품 릴스 하나로 24시간 완판 사례

### 처지 명시가 CTR 폭발
- "좁쌀여드름 있는 사람" NG → "딸이 사춘기 여드름으로 고생하는 엄마" OK
- "40대 맘까페에서 난리난" 같은 나이+모임 지정이 결정적

### 모호함 제거
- "최신형" NG → "1분 뒤 땀나는 발열조끼" OK
- 장문 나열 금지, 기능+타겟팅된 결과로 함축

### 초등 4학년 수준 어휘
- 상세페이지 카피 20~25자 간결, 1분 내 사로잡기
- B2C 전환율 분기점은 "지친 소비자 뇌 배려하는 단순함"

## 리뷰·신뢰 심리

### 리뷰 신뢰성 신호
- 자세한 설명, "나" 단어 최소화, 동사 절제로 가짜 티 제거
- 5점 만점은 의심, 4.2~4.7이 오히려 신뢰도 높음
- 전문가+일반 후기+구체 수치+관련 집단 언급으로 다층화

### 불만 고객 = 충성 고객 전환
- 대다수는 말없이 떠남 → 불만 표현 고객은 "고마운 사람"
- "덕분에 개선할 수 있다" 반복 감사로 뿌듯함 유도
- 문제 상품 무상 교체는 LTV 관점의 전략적 투자

### 자이가르니크 효과
- 완성도 높은 광고보다 미완성 진정성이 신뢰 상승
- 영세업체의 솔직함이 강점 — 발주실수 솔직 공유로 응원방문 유도

## 가격·결정 심리

### 작은 결정 누적
- 큰 구매를 월 납입·3선택지로 쪼개면 결정 저항 감소
- 미끼 효과: 2+1, A/B/C(B 최적), 장바구니 단계 추가 할인

### 50% 할인 = 혁신 인식
- 기준 가격보다 50% 이상 저렴 + 품질 유지 = "혁신"
- 불경기에도 매출 터지는 제품의 공통점

### 객단가가 CVR보다 현실적
- 객단가 1만 → 3만 올리면 필요 전환율 17% → 5%로 급감
- 세트·할인 명분 변경으로 "비합리적 선택" 의도적 유도 → 객단가 2배
