# AI 에이전트 빌딩, 이 10가지 해봤으면 당신은 중급 일껄요?

## 핵심 인사이트
- CLI 환경 습득의 중요성 — "체감 속도 차이 10배 이상"으로 GUI 대화창 대비 파일 읽기/수정이 직접 자동화되어 정확도와 작업량이 급증
- 스펙 중심 설계 필수 — 프롬프트 반복 입력 대신 설계 문서 30분 작성이 수정 삽질 3시간을 절감하는 비용편익 구조
- 토큰 소비 경험치 — 월 100~200달러 요금제를 일주일 내 소진해본 경험이 "진짜" AI 활용과 장난감 수준의 분기점
- 검증 서브에이전트의 효율 — 독립적 검증 프로세스를 거친 결과물이 검증 없는 것 대비 "3-4배 좋아짐"
- MECE 분해 기법 — 큰 개념을 겹침 없이 단계적으로 쪼개야 "땜빵 코드의 늪"을 피할 수 있음
- 기술 스펙 최소 이해 — 개발 불가능해도 React, API, 서버리스 같은 용어 이해로 AI 지휘력 향상
- 도구 한계의 실전 발견 — 제미나이는 API 호출보다 대화창에서 이미지 생성 품질이 우수한 비공식 특성 발견
