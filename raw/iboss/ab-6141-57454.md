# 마케터가 마케팅 핵심 이해하는 지식 40부작 시리즈 [15/40] - 광고 데이터 통찰하는 방법 및 개선 아이디어 도출

## 핵심 인사이트
- "숫자로 어떤 것을 보고 어떻게 봐야 통찰할 수 있는가"가 핵심 - 광고 데이터를 단순히 수치로 해석하지 말고 고객 행동의 의도를 찾아야 함
- 가설 검증 프로세스가 핵심: 회원가입 이탈율 90% → 원인을 추정 → 2~3일간 개선 후 데이터 확인 → 이탈율 40% 감소 사례
- CPC 100원이 제품 USP를 담고 있다면 상세페이지 문제일 가능성이 높음 - 썸네일 개선만으로 "이탈율 30% 이상 감소, 전환율 2배 상승" 가능
- 경쟁사 벤치마킹보다 자사 데이터 추세 분석이 중요 - 같은 상품도 톤이 다르면 데이터가 달라짐
- "데이터 분석은 심플하게, 개선안은 고객관점으로, 검증은 데이터 값으로" - 세 가지 원칙 준수
- 객단가 상승은 가격 인상이 아니라 옵션 구성으로 해결 - 3개 옵션의 가격차를 2.5배 벌려 중간값 구매 유도
- 데이터는 고객 심리를 완전히 드러낼 수 없음 - 지표로 모든 고객을 이해했다고 착각하지 말 것
