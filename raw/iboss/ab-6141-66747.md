# 100억짜리 자료 "퍼포먼스" 캔버스 활용법

## 핵심 인사이트
- 기획의 본질: "콘텐츠의 가장 중요한 것은 기획"이며, 제품의 본질적 존재 이유를 명확히 정의해야 모든 마케팅 전략이 시작된다
- 고객 정의 3단계: 인구통계학 → 욕구와 욕망 → 페르소나 순서로 체계적 분석 필요. 네이버 검색키워드도구와 AI 활용으로 4050대 남성, 대면 업무 다중인 타겟 발굴 가능
- 리뷰 데이터 활용: 경쟁사 상위 제품 리뷰를 GPT로 분석하면 "향기, 세정력, 지속력" 같은 핵심 욕구 키워드 추출 가능하며, 이는 콘텐츠 소재로 직접 활용된다
- 근거 6가지 요소: 숫자적 근거(임상) + 사회적 근거(1위, 만족도) + 사회적통념(고령화) + 경쟁력 + 컨셉 + 기능적 포지셔닝으로 구성되어야 한다
- 컨셉 전환의 힘: "그냥 향기 좋은 바디워시" → "은은한 살냄새용 바디워시"처럼 같은 제품을 재포지셔닝하면 시장에서 완전히 다르게 인식된다
- 고통지수 마인드맵: 바디워시 > 냄새 제거 > 액취증으로 확장하면 "액취증 전문가 개발 참여" 같은 강력한 권위 메시지 구성 가능
- 설득의 7가지 심리학: 상호성, 희소성, 권위, 일관성, 사회적증거, 호감요소, 숫자적근거가 모두 포함되어야 구매 결정에 이른다
- 메시지-이미지 3가지 테스트: 섹시한 중년 남성 이미지 + 거품 표현 + 액상 색감으로 나누어 테스트하면 성과 좋은 포인트 발굴 후 최적화 가능
