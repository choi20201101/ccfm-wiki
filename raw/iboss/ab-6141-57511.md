# 마케터가 마케팅 핵심 이해하는 지식 40부작 시리즈 [21/40]

## 핵심 인사이트
- 썸네일 변경만으로 2% 이상의 전환율 상승 - 누끼 사진이 정석이라는 고정관념을 벗고 모델, 사용 장면 등으로 실험하면 데이터 기반 최적화 가능
- 가격 인상 후에도 전환율 유지 사례 존재 - "기존의 좋았던 사례들을 참고하면서 테스트해볼 필요가 있습니다"
- 세트 구성 전략으로 객단가 2배 상승 - 여성패션에서 펌프스+슬링백 조합 시 "전환율이 1.5배 오르며 객단가도 2배 상승"
- 페르소나 변경으로 제품 포지셔닝 완전 달라짐 - "의사선생님이 만든 남자속옷"과 "당면보다 얇은 남자속옷"은 같은 제품이지만 고객 인식 차이 존재
- 보습에서 향기로 컨셉 전환 시 매출 성장 - 원초적 욕구 자극이 "매출 성장과 광고 효율에 직접적인 영향"
- 3~4일 데이터 수집 후 변경 전후 비교 필수 - 가설만으로 끝내지 말고 실제 측정을 통해 신뢰성 확보
- 단점도 마케팅 포인트로 승화 가능 - 꿀사과의 경우 농사 오류를 "마치 꿀이 들어있는 것처럼 표현"해 장점으로 전환
