# 돈 벌게 해주는 공감되는 컨텐츠를 파세요 5탄

## 핵심 인사이트
- CPC 전체 지표로 "컨텐츠에 사람들이 얼마나 잘 반응하는가" 간접 측정
- 아웃바운드 클릭과 랜딩페이지 조회 비용 차이가 크면 사이트 성능 문제/과도한 클릭 유도 확인
- "랜딩조회비용이 사이트 유입 숫자" - 픽셀 기반 실제 유입 추적
- 페이스북 기여기간은 조회 1일, 클릭 28일 기준 간접 기여도 측정
- CPM이 높다면 특정 고객층/한정 상품 특성 때문일 수 있음
- 체류시간 평균값에서 상위 25% 유저 행동 예측하여 시나리오 수립
- 이탈율 80%는 100명 중 80명이 아무 행동 없이 떠났다는 의미
- 공감 요소 기반 날카로운 리뷰 배치로 구매 전환율 상승
- GA-UTM 활용한 개별 콘텐츠 트래킹으로 성과 최적화
- 모든 고객이 아닌 구매 가능성 높은 타겟 설득에 집중해야 수익성 향상
