# 구글 애즈 검색광고 기능 한번에 이해하기 : 2년간 60억써본 인사이트 4탄-1

## 핵심 인사이트
- 구성 이해: 페이스북과 동일하게 캠페인 > 그룹 > 소재 구조로 구성되어 있으며, "구성이 똑같습니다"라는 점을 파악하면 세팅이 훨씬 쉬워진다
- 입찰 전략: 타겟 CPA/ROAS 설정 시 기존 데이터 기반으로 10~20% 범위 내에서 조절해야 하며, 급격한 목표 변경은 머신러닝을 방해한다
- 키워드 매칭: 정확한 노출을 위해서는 [가방] 형태로 일치검색을 사용하고, 확장검색은 초기 데이터 수집 후 별도 그룹으로 운영하는 방식 추천
- 초기 세팅: 전환 목표는 장바구니, 회원가입, 결제시작, 구매완료 등 구매 여정에 맞게 다중 설정하여 학습 데이터를 확보한다
- 소재 최적화: "최대한 많은 이미지와 문구를 넣어서 동적광고가 잘 학습되도록" 하는 것이 동적 소재 성과의 핵심
- 검색 네트워크 제외: 저자는 기본적으로 네트워크 지면을 제외하고 검색 결과에만 노출시켜 성과를 집중시키는 방식 선호
- 캠페인 목적 선택: 판매 목적의 캠페인에서 검색, 실적최대화, 디스플레이, 쇼핑, 동영상 채널을 모두 활용하되, 스크립트 데이터가 쌓인 후 활성화
