# 무의식, 의식, 설득 3가지만 알면 100억 매출 뚝딱?

## 핵심 인사이트
- 무의식 전략의 이중 구조: 욕구 기반 무의식(예쁜 여성, 본능적 반응)과 교육·학습 기반 무의식(의사의 신뢰도)을 구분하여 타겟별로 적용하면 CTR을 3-4배 개선 가능
- 구체적 사례—착한구두: "통굽"이라는 단어 자체가 고객의 무의식에 편함으로 자동 연결되어, 이미지 각도와 상품 조합 테스트로 경쟁사 1% 대비 3-4% CTR 달성
- 광고 콘텐츠 3단계 구조: 문제제기(무의식) → 문제해결(인식) → 설득의 심리학 요소 9가지(권위, 희소성, 사회적 증거 등) 단계적 적용
- 설득의 심리학 필수 요소 5가지: 권위, 통념, 숫자 근거, 희소성, 신뢰도—이 중 3개 이상 포함 시 초강력 콘텐츠 생성
- 숫자 표현의 심리효과: "300명 중 3분의1 암 치료 성공" vs "매년 100명 생명 구조"—같은 수치도 긍정 프레임이 먼저 오면 더 강력한 인상 형성
- 마인드맵을 통한 통념 발굴: 시장조사 키워드, 고객 리뷰 VOC, 문화적 특성을 연결하여 새로운 포지셔닝 기회 창출(예: 전골국시, 김치찌개쌈밥)
