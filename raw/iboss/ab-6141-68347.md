# AI로 사실적인 인물 치트키 + 나노바나나 멱살잡는 씨드림4.0 가이드

## 핵심 인사이트
- "완벽함을 버려야 진짜가 된다" - 모공, 주근깨, 잔주름 같은 결함을 의도적으로 추가하면 AI 이미지의 현실감이 극적으로 증가
- 프롬프트에서 "사실적인", "고퀄리티" 같은 표현은 오히려 인위적인 결과를 만들어 역효과 발생
- 배경은 도시/자연 같은 추상적 표현 대신 "콘크리트 균열, 낡은 벽돌, 녹이 슨 에어컨실외기" 같은 구체적 질감으로 묘사
- 노출비(밝은/어두운 부분 대비)를 2:1(자연스러움)부터 16:1(극적)까지 조정하면 분위기를 완전히 컨트롤 가능
- 생성 후 GPT에 JSON 분석 요청 후 개선점 반영하는 3~5회 반복 루프로 클라이언트도 실제 촬영으로 착각할 수준 달성
- 네거티브 프롬프트보다 긍정적 디테일 묘사가 효과적 ("~하지 말 것" X → "~한 모습" O)
