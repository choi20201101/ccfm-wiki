# 구글 애즈 동영상 캠페인 기능 이해하기: 2년간 60억써본 인사이트 4탄-2

## 핵심 인사이트
- 전환 목표 설정 시 지면(트루뷰, 아웃스트림, 인스트림) 선택 불가능 — "머신러닝에게 맡기는 방식이 좋다고 판단"하여 자동 조절로 변경됨
- 초과게재 현상 주의: 일일 예산 설정 시 초기 캠페인은 최대 2배까지 청구 가능하므로 조기 승인 타이밍 조절 필수
- 캠페인 구조: "유형별로 1개~2개 정도가 적당"하며, 소재별 다분화는 초과게재와 예산 낭비 유발
- 사이트 링크 확장 기능: 4개 이상 설정 시 CTR 개선 효과 명확하며 고객에게 제품 정보 전달력 증대
- 제품 피드(쇼핑 확장): 페이스북 카달로그처럼 동영상 하단에 상품 노출 — CTR과 CVR 동시 개선 검증됨
- 타겟 ROAS는 보조수단: ROAS 100%인 소재를 머신러닝으로 200% 달성 불가능하므로, 충분히 학습된 데이터(140~150%) 기반 160%로 점진적 상향 추천
- 콘텐츠 제외 기능: 민감한 카테고리 설정으로 성인/혐오 컨텐츠 채널 노출 차단 가능
