# 죽은 상권에서 월매출 2억 멱살잡고 끌어올린 사례

## 핵심 인사이트
- 배후세대 의존도 극복 - 상권의 고객층이 재개발로 1,000세대가 사라졌을 때도 월매출 2억 달성 가능
- 배달앱 플랫폼 활용 - "쿠팡이츠/배민스토어" 등록 후 "재주문고객+신규고객이 쌓여 추가 수익이 계속 쌓입니다"
- 카카오톡 채널의 위력 - 1년간 1,000명 모았으며, 카카오톡으로 주문한 고객은 계속 카카오톡으로만 재주문
- 네이버 플레이스 최적화 - 지역명+정육점 검색 시 1-2순위 노출이 선물세트 주문으로 이어지는 효과
- 수수료 고려한 가격 책정 - "배달팁을 모두 계산해 실제 마진을 고려하여 판매가를 설정"해야 함
