# 데이터쇼핑 하지 않고 매출 10배 성장시키는 10가지 꿀팁

## 핵심 인사이트
- 자사 데이터는 "결과"일 뿐 원인을 알려주지 않음 — 지난달 잘 팔렸다고 다음달도 같은 성과를 기대하면 위험
- "왜 샀는지" 모르면 재현 불가능 — 전환율이 올랐을 때 그 이유를 파악해야만 전략 수정 가능
- 샘플 부족 상태에서 데이터 기반 의사결정은 운에 기대는 것과 같음 — 최소 구매자 500명 이상의 데이터 필요
- 고객은 일관성이 없음 — 같은 사람이 같은 상황에서도 다르게 행동하므로 과거 행동으로 미래 예측은 실패 가능성 높음
- 문화 > 트렌드 > 유행의 레벨 이해 필수 — 각각 지속시간과 영향력이 다르므로 구분해서 대응해야 함
- 본질적 욕구는 절대 변하지 않음 — "매력적으로 보이고 싶다" 같은 기본 욕구를 간과하면 아무리 좋은 기획도 안 팔림
- 패턴은 숫자 나타나기 전에 정성적 신호로 먼저 옴 — SNS 피드, 인플루언서 언급, 커뮤니티 고민이 데이터 변화보다 먼저 나타남
- 패턴으로 가설 세우고 데이터로 검증하는 사이클 필수 — 소규모 테스트 → 데이터 확인 → 스케일 또는 폐기
- 숫자 보는 시간을 줄이고 사람 관찰 시간을 늘려야 함 — 리뷰 읽기, SNS 피드 분석, 고객 직접 인터뷰가 GA보다 중요
