# 구글디스커버리 CPC 100원 ROAS 300% 꿀팁 : 2년간 60억 써본 인사이트 5탄 -1

## 핵심 인사이트
- 초기 계정 모수 학습 필수: "충분히 계정의 모수가 쌓인 상태에 디스커버리 운영"해야 성과 기대 가능
- 단일 캠페인 집중 전략: 여러 캠페인보다는 1개 캠페인으로 집중 운영이 더 효과적
- 초기 예산은 5~7만원 시작: "광고비는 최소 5~7만원으로 시작해서 점진적으로 늘리는 것"이 성과 향상 도움
- 3가지 타겟 조합 운영: 넓은 타겟, 맞춤 타겟, 유사 타겟의 조합으로 최적 성과 달성
- 다양한 이미지 에셋 필수: "이미지 에셋은 최대한 다양하게 구성"하고 저조 에셋은 제외
- 클릭베이트 대응 체계: 구글 지원팀 활용으로 반려 사항 지속 해결 필요
- 방향성별 콘텐츠 묶음: 피부톤, 주름, 기미 등 같은 방향성끼리 콘텐츠 그룹화
- 쇼핑 머천트 센터 연동: 확장 에셋 기능 작동을 위해 필수적 점검 항목
