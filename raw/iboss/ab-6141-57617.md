# 마케터는 다차원적 사고를 통해서 상상력을 키워야합니다

## 핵심 인사이트
- 가설과 상상력이 광고의 핵심: "마케팅 광고의 핵심 역량은 가설과 상상력"이며, 단순한 기술적 운영이 아님
- 신규 고객 획득이 근본 목표: 모든 광고의 궁극적 목적은 신규 고객 확보이고 지속적 광고가 필수
- 제한된 예산 속 선별적 검증 필요: 수십, 수백 가지 경우의 수를 모두 테스트할 수 없으므로 전략적 선택이 필요
- 정확한 진단이 치료보다 중요: "기침이 난다고 무조건 감기가 아니"므로 데이터 분석으로 문제의 정확한 지점을 파악해야 함
- 단계별 데이터 추적 방법론: 장바구니→결제→회원가입 등 각 페이지 이탈율을 비교하여 문제점 특정
- 이상 데이터 패턴 캐치: "700명 유입, 650명 이탈, 15명만 구매"처럼 월별/동월대비 비정상 수치 포착이 핵심
- 경험과 해석의 개인화: 제시된 공식은 정답이 아닌 풀이 과정이므로 각자 재해석하여 자신만의 방법론 구축 필요
