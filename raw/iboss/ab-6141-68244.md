# AI 모델 이거 진짜 사람아니에요? 모델비 0원…!

## 핵심 인사이트
- AI 모델 생성 기술로 "모델료 0원의 시대"가 도래했으며, 특히 뷰티 모델 같은 고비용 인력을 대체 가능
- 이미지 레퍼런스 활용이 핵심: 레퍼런스 없을 때보다 넣었을 때 "거의 동일한 인물이 나온다"는 결과 차이 발생
- Midjourney 등으로 미리 생성한 레퍼런스 이미지를 활용하면 원하는 결과물을 더 정확하게 제어 가능
- AI 생성 이미지의 피부 질감과 세부사항이 실제 인물과 구분 어려울 정도로 발전했다고 주장
- 나노바나나 기능 활용으로 생성된 모델 이미지와 제품 합성 작업이 가능해짐
- 1크레딧으로 이미지를 4K 화질로 업그레이드할 수 있는 기능 제공
- 초상권 위반 주의: 기존 모델 사진 활용 시 법적 문제 발생 가능성 경고
- AI 툴도 "레이어 생각하고 적재적소에 써야" 하며, 단계별로 결과물을 설계해야 원하는 최종 결과 도출 가능
