# 미드저니를 쓰면 모델비, 촬영비 1000만원 아낍니다 미쳤습니다

## 핵심 인사이트
- 이미지 생성 AI 품질 차이: "달리3의 경우...미드저니가 뭔기 이미지 느낌으로 임팩트가 더 확실합니다" — 동일 프롬프트 기준 미드저니 품질이 100배 이상 우수
- 줌인/줌아웃 기능: 얼굴만 나오는 이미지를 상체까지 확대하거나 특정 부분만 재생성 가능하여 촬영 없이 다양한 컷 제작 가능
- 부분 수정 기능: 손가락이 어색한 부분만 선택해서 수정 가능하며 포즈 변경도 프롬프트로 간편하게 조정
- 모델 생성 비용 절감: 실제 모델 촬영 없이 "원하는 모델의 얼굴을 만들고 행동까지 부여하고 장면이나 배경까지 변경" 가능
- 의류 변경 기능: 기본 의류를 빨간색 드레스로 변경하거나 정장 스타일로 변환 가능 — 같은 모델로 다양한 상품 이미지 제작
- 프롬프트 자동 생성 도구: ChatGPT GPTS 중 "미드저니 프롬프트 마스터" 활용으로 복잡한 프롬프트 작성 시간 단축
- AI + AI + 사람 조합: "AI > AI 로 시킬 수 있다"는 점 — 여러 AI 도구 3~4개를 연계하면 최종 결과물 품질 향상
- 정책 제약: 성인자료 등 정책 위반 콘텐츠는 생성 거부되지만 "정상적인 작업"으로는 쇼핑몰 모델 생성 등에 활용 가능
