# 이미지_필수_기획_항목_Appendix

이미지 필수 기획 항목 Appendix
## 광고 소재의 방향성
해당 이미지 소재가 달성해야 하는 광고 목적 명확화
광고 운영 목표 점검 : 제품 판매, 브랜드 인지도, 프로모션 참여 등
예) “체취 고민 해결 중심의 전환소재” / “브랜드 인지도 증진용 참여소재”
## 핵심 USP 및 페르소나 정보
이번 소재에서 반드시 표현해야 할 USP
예) “12시간 체취 차단”
메인 타겟 페르소나
USP 기반으로 메인타겟을 잡고, 상세 페르소나 설정
페르소나를 어떤 이유로 설정하였는지 근거, 이유 필요
연령/성별/라이프스타일 등 구체적인 상황 적용
예) “체취 고민 있는 30~50대 남성 직장인”
## 스토리라인 구조 제안
기본 플로우
문제 → 해결 → 근거 → 설득 → CTA 구조 설계
문제 : 고객이 공감할 수 있는 문제 또는 불편함 제시
해결 : 문제 해결에 이후 기대되는 상황 제시
근거 : 문제 해결이 될 수 밖에 없는 근거 제시
설득 : 타인의 경험을 통한 신뢰 요소 제시
CTA : 고객이 얻을 수 있는 즉각적인 이점 제시
적용 예시
문제 제기 (체취 고민 상황) → 해결(특허 성분) → 근거(임상 수치, 후기가 입증) → 설득(사용 후기, 신뢰성 강화) → CTA(구매 혜택 안내)
[헤어라인 두피타투]
문제 제기 : 정수리 라인이 넓어 거울보는게 스트레스인 남성, 시술은 부담스러움
해결 : 두피에 착색되고, 단기간(30초, 5분 등) 쉽고 빠른 커버로 2주동안 정수리 커버 유지
근거 : 피부 무자극 테스트와 탈모 증상 완화 기능성 성분이 함유
설득 : 제품 탄생 배경 설명, 타제품과 지속력, 커버력 비교, 전후 이미지로 설득
CTA : 3차 물량 완판으로 인한 예약판매 기간동안 49% 할인, 삼푸 무료 증정
