## 1. 마케팅 마인드셋
마케팅 마인드셋은 마케팅사업부의 핵심가치 5가지를 기반으로 합니다. 이 핵심가치들을 항상 중요시하며 일하고 있는지 점검해야 합니다.
### 고객집착
#### 중요성
퍼포먼스 마케팅은 고객이 원하는 것 또는 고객의 성과를 높이는 모든 것을 빠르게 테스트하고 결과에 따라 개선하는 일이 핵심입니다.
단순 광고 세팅 반복만으로는 효율 개선에 한계가 생깁니다.
정확한 소통과 문제파악을 위해서는 고객에게 집착해야 하며, 이를 통해 정확한 문제 해결을 위한 원인을 파악할 수 있습니다.
#### 고객 집착 기본
경청을 통해 고객의 숨은 니즈·문제 인지
지속적인 소통으로 불안감 없이 진행상황 공유
관심 가지기로 먼저 움직여 고객 비즈니스를 업데이트하고 고객의 고객들의 관심사 파악
학습과 성장으로 문제 해결 역량을 계속 키우기
신뢰와 유대감을 형성해 "함께 성장"하는 관계 구축
#### 고객 집착의 기본 소통법
목표·니즈 먼저 파악 → "구체적인" 해결책·효과를 제시(데이터 근거 포함)
팀 전체가 방문 고객(문의)이 들어오면 빠른 응대 (전화·메일·DM 모두)
전문성·신뢰: 구구절절이 아닌, 단도직입 + 수치·객관적 자료로 근거 제시
고객이 "지금까지 비효율" 상태였음을 느끼게 하되, "우리한테 맡기면 즉시 개선"이라는 이득 강조 및 실행 플랜과 데드라인 제시
중요: 불만/질문이 생기면 혼자 넘기지 말고, 바로 담당자/팀 공유 & 함께 해결
이렇게 실행하면, "내가 (고객) 이 회사에 맡기면 득이고, 안 맡기면 손해"라는 느낌을 고객이 분명하게 받게 됩니다. 이는 곧 고객집착의 핵심 목표: 고객이 원하는 성과와 가치를 정확히 짚고, 우리의 역량을 단도직입적으로 제시함으로써 즉각적 신뢰와 만족을 얻는 것입니다.
### 실행력
#### 중요성
퍼포먼스 마케팅은 다양한 아이디어와 전략이 있어도, 얼마나 빠르게 실행하여 성과를 확인·개선하는지가 결정적입니다.
시장·광고 환경이 급속도로 바뀌는 상황에서, 실행이 늦거나 망설이다 보면 경쟁우위를 놓치거나 비효율이 커질 수 있습니다.
"치트키" 없이 가설 → 실행 → 데이터 분석 → 보완을 짧은 주기로 반복할 때 성과가 크게 개선됩니다.
#### 실행력 기본
속도 중시: 회의에서 나온 아이디어나 가설은 빠른 예산 셋팅+A/B 테스트로 곧바로 검증
단계별 피드백: 주 단위(또는 더 짧은 주기)로 결과 리뷰, 안 되는 건 즉시 중단 or 수정
짧은 보고 & 의사결정 루프: 불필요하게 설명이 길어지지 않고, 바로 실행-결과-피드백 순환
체력·내구성: 광고 시즌/프로모션 시 과부하가 걸리는 경우도 많으므로, 충분한 체력·멘탈 관리
실패 경험 축적: 실패를 드러내는 걸 두려워하지 말고, "왜 안 됐는지"를 공유하며 팀의 학습 자산으로 쌓기
#### 실행력의 기본 실행(소통)법
주간(Weekly) A/B 테스트 & 리포트
매주 1~2개 핵심 변수를 바꿔 즉시 실험
N일 후 CTR·ROAS 등 지표 분석 → 잘된 건 확대, 안 된 건 폐기 or 수정
팀원 간 "즉시 협의"
담당 부서·파트너(디자이너, 영상 디자이너 등)와 상시 커뮤니케이션 체계 구축
"이 카피 바꿀까요?" 등 작은 변경도 신속하게 합의 후 적용
중간보고→개선안
실험 진행 중 또는 후 실험의 기간은 리더와 설정한 기간 엄수, 중간 성과 보고 후 행동 계획 결정
"정성적 피드백(리뷰·댓글·고객목소리)"과 수치적 결과(전환율, ROAS) 함께 살피기
실행 계획에 '데드라인' 명시
"이번 주 금요일까지 카피 3가지 버전 준비 & 토요일~다음 수요일 테스트"처럼, 작고 명확한 기한 설정
데드라인 없으면 실행 지연→효율 저하
무조건 데드라인 안으로 실행, 데드라인 안에 실행 못하는 일 발생 시 해당 실무 담당자와 PM 리더의 책임으로 부여
이렇게 실행하면:
확실한 루틴으로 지속적 실험이 이뤄지면서, 잘되는 쪽은 빠르게 강화 → 매출/전환 효율 지속 상승
실패 사례도 빠르게 확인·폐기하므로 광고비 낭비 최소화, 시행착오에서 빨리 배우는 조직 문화 형성에 기여
결과적으로 고객(광고주) 입장에서도 "이 회사는 말이 아닌 '실행 + 데이터'로 증명"한다는 신뢰가 쌓임
실행력은 퍼포먼스 마케팅 성공의 절대 조건: "빠른 테스트→즉각 개선"이 핵심
목표 달성을 위한 짧은 주기(주간/격주) A/B 테스트, 데드라인 기반의 협업, 투명한 피드백 체계를 견고히 하면 성과는 자연스럽게 올라감
### 도전정신
#### 중요성
새로운 매체·트렌드가 끊임없이 나타나는 디지털 마케팅 환경에서, 기존 방식만 고수하면 효율 하락과 성장 정체가 올 수 있음
실패 위험이 있더라도 "한 번도 안 해본 채널·포맷"을 시도해야 폭발적 성과를 얻을 기회가 생김
#### 도전정신 기본
새 아이디어·매체에 열린 자세: "안 해봤다 → 안 한다" 금지. 테스트 예산으로 먼저 소규모 시행
실패 허용 & 학습: 모든 시도가 성공하진 않음 → 실패해도 빠르게 이유 분석, 팀 공유
데이터+직관 병행: 해외 및 국내 사례, 시장 트렌드 조사 후 가능성 있으면 바로 실험
조직 학습 자산화: 성공/실패 모두 기록해두어, 유사한 다음 시도에 참고
실패는 과정: 실패라 생각하지 말고 새로운 도전이였다면 과정이라고 생각하고 빠르게 복기 후 학습
#### 도전정신의 실행(소통)법
월 1회 이상 신규 채널/컨셉 테스트
예: 틱톡·유튜브 숏츠 등 새 포맷 광고 N주간 집행
CTR·전환율 등 성과 추적 → 확대 or 중단 결정
팀 공유 & 책임 명확화
"신규 프로젝트" 전담자 지정, 실패해도 책임 추궁 아닌 학습 초점
정기 리뷰
1주 후 간단한 회의: "신규 시도가 어떤 결과?" "무엇을 배우고 어떤 개선점이 있나?"
실험 예산 분리
전체 예산 중 일정 %를 '혁신 테스트 전용'으로 책정 → 부담 없이 도전
광고주에게 미리 소통해 부담 없이 혁신할 수 있는 방안 제한 후 도전
이렇게 실행하면:
시장 선점 & 폭발적 성과
기존에 잘 되는 채널에서 새로운 도전과 실행 반복해 경쟁사가 내지 못하는 효율 극대화
경쟁사보다 먼저 신규 채널에서 광고 → 저렴한 CPC/CPM, 빠른 고객 확보
팀의 성장 & 실패 두려움 해소
다양한 시도 속에서 무엇이든 시도-학습 문화 형성
고객(광고주) 만족도↑
"늘 앞서 나가는" 대행사·마케터라는 인식 → 신뢰와 재계약으로 이어짐
도전정신은 "예산 일부를 과감히 투자하여 기존 채널과 새 채널·컨셉의 실험을 주기적으로 시험하고, 실패해도 즉시 원인 파악→학습"하는 기회를 만들어줌
남들이 CTR 1.5% 나오면 3% 나오게 만드는 방법을 끝없이 고민하고 도전
이를 통해 혁신적 성과와 조직 역량의 상승이 동시에 가능해짐
### 팀워크
#### 중요성
온라인 마케팅은 마케팅 기획 → 디자인 또는 영상 컨텐츠 제작 → 매체 세팅 → 데이터 분석 → CS/리뷰 관리(광고주 파트너쉽 협업) 등 다학제적 협업이 필수
한 파트(광고 세팅)만 잘해도, 디자인, 영상 컨텐츠 제작자와 소통 불량이면 전체 성과 하락이 되고 개선도 불가능
팀 전체가 "한 프로젝트, 한 목표"로 일관성 있게 움직여야 퍼포먼스 최대화 가능
#### 팀워크 기본
공유/협업 프로세스
프로젝트 일정·성과 지표·과업분담 등 투명하게 공유
"누가 무엇을 언제까지" 할지 명확히 문서화
서로의 역량·강점 이해
예: A는 카피/카메라 감각, B는 데이터 분석, C는 CS·고객대응 강점
강점을 살려 일을 분담하고, 약점 부문은 협력
수평적·즉각적인 커뮤니케이션
잘못된 점은 지위/직급 상관없이 즉각 피드백 주고받기
"이건 제 담당이 아니에요"식 떠넘기기 절대 지양
#### 팀워크의 실행(소통)법
주간(Weekly) 스탠드업 미팅
10분 내외로 "이번 주 목표, 각자 진행상황, 장애물" 간단히 공유
각 파트에서 필요한 협조사항을 빠르게 전달
성과·지표 투명 공유
매체별 CTR/CPA/ROAS 등 주기적(주/월) 리포트 → 팀 전체가 인지할 수 있도록 모든 마케터, 디자이너, 영상디자이너 데이터 숙지
디자인 파트도 해당 성과(이미지 실험에 따른 CTR 등) 인지해 개선 반영
문제 발생 시 즉시 협의
예: 광고 소재 오탈자, CS 고객 불만 → 관련 부서 책임 추궁이 아닌, 바로 실무자 및 각 조직 PM 리더와 소통
"원인 파악 → 책임 추궁보다는 해결책" 초점
강점 분담 + 크로스체크
콘셉트 기획 시, 퍼포먼스 캔버스 중심으로 마케터와 디자이너가 함께 도출한 UPS와 근거와 통념을 바탕으로 디자이너와 영상디자이너 기획서 작성 진행
마케터도 기획서를 작성, 마케터의 가장 중요한 포인트는 고객사 컨펌과 소통
항상 문제가 발생할 수 있는 컨텐츠 부분은 마케터, 디자이너 크로스 검수
이렇게 실행하면:
작업 효율 극대화:
중복 작업, 떠넘기기, 누락 등 최소화 → 더 빠른 캠페인 론칭, 품질 높은 광고 소재
오류 리스크 ↓:
여러 파트가 상호 검수, "문제 발생 시 즉각 협의"로 빠른 대응 가능
팀 시너지로 성과 상승:
한 파트가 성과를 내면 다른 파트에도 정보가 공유되고, 전체 수준 향상
고객 만족:
일관적이고 전문적인 협업 프로세스로 고객에게 "체계적 대행사/팀" 인식 → 신뢰↑, 재계약·평판 향상
팀워크란 결국 "정보 투명 공유 + 역할 분담 + 즉각 협의" 세 박자가 맞아야 함
이를 통해 빠른 실행력·낮은 오류율·높은 고객 만족도로 이어지고, 장기적 퍼포먼스 성공에 결정적 역할을 함
### 책임감
#### 중요성
광고비를 직접 사용하고, 매출·성과에 큰 영향을 미치는 퍼포먼스 마케팅 특성상 사소한 실수(트레킹 누락, 랜딩 URL 오류 등)도 큰 손해로 이어질 수 있음
책임감 없이 "어쩔 수 없어요" 식으로 넘어가면, 고객 신뢰 저하 → 장기 거래·명성에 악영향
광고 컨텐츠에 대한 성과 개선도 퍼포먼스 캔버스를 통해서 기획한 중심으로 실무 마케터와 디자이너가 두명이 책임을 다해서 중심을 잡고 움직여야 성과가 발생함
#### 책임감 기본
오류·리스크 사전 방지 노력
미리 매체 설정, 광고 소재, 트래킹 코드를 체계적으로 점검
컨텐츠에 대한 주의 사항 요청 사항에 대한 부분에 대한 점검
문제 발생 시 즉각 보고·해결
숨기거나 미루는 대신, 누구나 "내 일"처럼 책임감 있게 나서서 처리
결과(성과·데이터)에 대한 자기 주도적 해석
"어쩔 수 없었다"가 아니라, 내가 개선점을 고민·제시하고 리드
기한·마감 준수
고객과 약속한 일정·성과 보고 시점 지키기 (작은 약속이라도 필수)
기한 준수를 하지 못했을 때 치명적, 기간이 더 필요하다면 미리 필히 몇시간 혹은 몇일전 소통을 통해 조율 필요
#### 책임감의 실행(소통)법
오류·실수 발생 시 절차
"문제 원인 파악 → 담당 팀/담당자와 공동 대응 → 빠른 해결 → 사후 재발 방지안"
문서(노션 및 슬랙)에 기록해 추후 동일 실수 방지, 반복적인 부분은 체크리스트화 해서 반복관리
지표 & 목표치 관리
무조건 매체·환경 탓이 아니라, 목표치 달성을 위해 어떻게 바꿀지 스스로 생각·실행
실제 성과(전환율, 매출, CPC 등)와 목표 간 차이가 크면, 팀/고객에게 즉각 설명+대안 제시
컨텐츠에 대한 다양한 책임감을 지니고 반복적으로 성과가 안나오는 컨텐츠 책임지고 제거 후 성과 나오는 컨텐츠 신규 발굴
작은 약속도 어기지 않기
미팅 일정, 보고 날짜, 디자인 샘플 제출 기한 등 일관된 데드라인 준수
변수가 생기면 즉시 알리고 새 일정을 재합의
이렇게 실행하면:
오류·리스크 최소화
매주·매일 체크리스트 운영, 작은 문제도 커지기 전 조기 발견
고객(광고주) 신뢰 확립
"문제 생겨도 이 팀은 숨기지 않고 바로 해결책 준다"
반복 거래·장기 계약으로 이어짐
팀 내부 책임 의식 강한 문화
각 담당자가 자신의 파트(광고 세팅·디자인·데이터 등)를 끝까지 책임 → 프로정신 강화
성과 향상 + 오류 비용 절감
책임감 있는 실행→문제 대응 속도 빨라져 결과적으로 성과 지표도 안정적·상승 추세
책임감은 "광고비·성과·결과물에 대한 철저한 관리와 작은 오류도 즉각 해결"하는 태도에서 비롯
이를 위해 투명한 예산 관리, 문제 발생 시 즉각 보고·조치, 데드라인 준수를 잘 지키면, 신뢰도 높아지고 장기적 성과가 안정적으로 유지됩니다.
## 2. 퍼포먼스 마케팅 기초
### 퍼포먼스 마케팅이란?
#### 정의
투자 대비 성과(ROAS, 전환율, 클릭 등)와 같은 수치적 지표를 기준으로, 광고/마케팅 캠페인을 계속 측정·분석·개선하는 마케팅 방식입니다.
#### 목표
매체 별 정확한 ROI(투자대비효과)를 파악해, 효율을 극대화하는 것입니다.
전통적 마케팅(브랜딩, 노출 중심)과 달리, 퍼포먼스 마케팅은 직접 측정 가능한 지표에 집중합니다.
### 퍼포먼스 마케팅 중요 요소
#### 가설
퍼포먼스 마케팅의 가장 많이 실무에서 반복적으로 쓰게 될 용어인 가설에 대한 이해가 필요합니다.
가설은 A를 실행하면 내가 원하는 결과 A를 기획으로 정리하는 것입니다.
가설은 현재 있는 단계에서 다음 성과의 목표로 성장하는 방향 설정이 필요합니다.
효율 개선을 위해 A/B 테스트나 적용할 요소들을 넣어 데이터의 변화를 검증하기 전 잡는 기획입니다.
##### 좋은 가설을 잡는 순서
문제 또는 목표 명확화
해결하려는 수치가 무엇인가? CTR, CVR, 방문자수, 객단가 등
해결이 되려면 어떤 것이 우선순위로 되어야 하는가? 측정
측정의 방법:
자사 데이터 기반
고객의 목소리 VOC
시장 경쟁사 리서치
경험(직관) 또는 브레인스토밍
설정된 목표의 측정하여 가설을 잡을 기준을 명확하게 만들었다면 가설을 문장화하고 측정 지표를 정의해서 실험방법을 정리, 고객사와 소통해 실험 준비에 대한 일정을 잡습니다.
##### 좋은 가설의 예시
변수가 1개로 통제됩니다.
CTR 체크를 위해서 이미지 변화만 통제합니다. 통제하는 요소가 이미지의 모델의 사진을 다양하게 써서 CTR에 변화를 추적합니다.
객단가가 3만원인데, 세트구성으로 4.5만원까지 개선 시켜봅니다. 세트 구성을 통해서 3단계 옵션 구성 3/6/9만원 옵션으로 가장 많이 구매하는 옵션의 비율을 실험 추적합니다.
제품의 카피에서 반복적으로 고객들의 전환이 잘 나오는 카피요소가 무엇인지 체크하며 추적합니다. 키워드나 문장 묘사하는 내용을 추적합니다.
광고와 리뷰의 통일을 통해서 그리고 리뷰의 내용에 따라 전환율이 변화하는지 체크 추적해봅니다.
가설은 "(무엇을 어떻게 바꾸면) (어떤 지표)가 (어떻게 개선될) 것이다"라는 형태로 명확히 작성해야 합니다. 카피·세팅·이미지·객단가·전환율 등 요소마다 다양한 시나리오를 설정하고, 데이터 기반 A/B 테스트로 성과를 측정함으로써 실제 퍼포먼스 향상을 이끌어낼 수 있습니다.
성과가 무조건 나오겠지라는 가설을 잡는 게 아니라 성과를 만들어내는 과정을 계속해서 만들어내서 단 1cm라도 고객사의 성장을 이끌어낼 수 있는 가설이 필요합니다.
저조하거나 성과가 나오고 있는 광고주의 상황을 개선하거나 더 성과를 높이기 위해서는 무조건 가설을 잡는 연습을 하고 그 계획을 빠르게 실행하는 것이 가장 높은 고객집착을 통해 만족을 만드는 방법입니다.
#### 실험
가설을 잡은 것을 실험하는 것이 퍼포먼스의 가장 빠른 결과를 만들어내는 방법입니다.
가설 잡은 방법을 실험하는 것이 곧 실행입니다.
KPI 설정을 통해서 실험의 목표를 구체화합니다.
##### 좋은 실험의 방법 예시
좋은 실험은 우선 가설에 잡힌 방향을 체크해 빠르게 실행하는 것입니다.
1회 실험당 1가지~2가지 이하의 핵심 변수만 변경해서 실험합니다.
고객사와 실험 결과를 추적할 때 7일간 진행 여부를 전달합니다(기간은 고객사와 설정)(얼마 유입 100명 유입 후 데이터 측정도 가능).
실험 기간 내 데일리 단위로 수치 변화에 대해서 소통하고 실험을 기간 동안 계속 지속할지 팔로업하면서 현상에 대한 분석과 지표 공유를 통해서 실험에 대한 의사결정을 소통합니다.
단순히 실험 종료 방향에 대한 공유가 아닌 광고주가 이 실험을 계속 지속해야하는 이유에 대해서 데이터 기반으로 설득이 필요합니다.
옵티마이즈 기능을 활용해서 실험하거나 없다면, 적용 전후의 데이터를 명확하게, 주 단위, 일 단위, 요일 단위로 비교합니다.
결과 도출 → 즉시 적용 → 학습기록 → 다음 실험 로직으로 실행합니다.
실험을 정교하게 설계·실행하면, 퍼포먼스 마케팅에서 객관적 근거로 성과를 끌어올릴 수 있습니다. 작은 실패들도 체계적으로 학습하여, 점진적이면서도 지속적인 퍼포먼스 최적화에 도달하게 됩니다.
##### 결과
실험의 결과를 기록하여 고객에게 빠르게 공유하고 결과에 대한 피드백으로 이어질 수 있도록 소통합니다.
결과는 주관적 의견이 들어가지 않고 무조건 객관적 데이터 기준으로 전달합니다.
#### 피드백(분석)
분석은 데이터를 절대로 해석하려 하지 않는 것에서부터 시작합니다.
데이터는 우리가 가설로 잡는 결과의 후행 지표(행동 후 결과물)입니다.
데이터가 우리가 가설 잡은 목표를 달성했다면 "성공", 달성하지 못했다면 "실패"로 확인하는 용도로 활용합니다.
CTR, CPM, CPC, 전환수, 전환율, ROAS, CPA 객단가 각 퍼널이벤트 장바구니, 결제시작, 회원가입, 이탈률, 체류시간 등 핵심 지표의 변화로 확인합니다.
우리가 잡은 목표 가설에 대한 부분을 적용했는 것이 유의미한 데이터 변화로 이어진 것인지에 대해서 정리합니다.
##### 좋은 분석의 방법 예시
분석의 핵심은 데이터 각 변화의 유의미함이 있는지 확인하고 그 변화에 따른 다음 가설을 잡는 용도로 사용합니다.
전환율 1% 개선이 되었습니다. 이번에 의도했던 리뷰를 상단에 몇가지를 작성해서 올렸더니 개선되어 리뷰를 통해서 더 좋은 전환율이 나오는 리뷰는 무엇인지 0일 단위로 보겠습니다.
회원가입을 유도하고 혜택을 강조했더니 장바구니 이탈자가 줄어들고 회원가입수가 늘어났습니다. 회원가입을 더 강화하기 위해 고객들에게 회원가입 혜택의 한정수량 기준을 잡아 더 강화겠습니다.
이미지를 뒷굽을 강조하는 사진 중에 45도 각도로 보여지고 배경이 복잡하지 않고 누끼 회색 배경이 CTR 3% 이상 보이는 것으로 보여, 비슷한 45도 사진들을 오늘 바로 세팅해서 CTR 검증을 해 CTR 3% 나오는지 두번 점검을 통해서 가설에 대한 성공을 추가 점검하겠습니다.
결국 어떤 현상이 발생을 하기 위해서 잡을 가설에 대한 결과를 데이터로 읽고 실패의 경우 다른 가설을 투입하고, 성공이라면 그 가설을 더 강화하는 방향으로 광고주에게 제안하는 것이 필요합니다.
대안책 없이 데이터만 공유하고 실패했습니다라는 공유는 결국 고객사가 의문을 가질 수 밖에 없는 결과를 만들 수 있습니다.
##### 통제
가설에 대한 통제를 필수로 해야 실제로 그 가설을 통해서 결과가 변화하는 것을 알 수 있습니다.
위 내용에서도 반복적으로 통제해야한다고 나오는 이유는 초보자나 실험을 많이 해보지 않는 사람들은 여러가설을 겹치는 실수를 많이 하게 됩니다.
변수를 통제한다는 의미는 1가지 이상에 변수를 넣는 순간부터 분석하기가 어려워지고 여러 변수를 측정하려면, A B C 안으로 3가지 랜딩을 조합해 컨텐츠를 실험하거나, 매체의 CBO 기능과 DCO 기능으로 테스트해 AB테스트 기능을 활용해 최대한 변수 통제가 가능한 기능들을 활용해서 실행이 필요합니다.
### 직접 매출과 간접 매출의 이해
#### 중요성
광고 어드민 매출은 광고의 측정기준, 즉 광고 어드민 기준을 믿다보면 광고에서는 매출액과 ROAS가 높으나 실제 자사몰이나 광고를 유입시킨 어드민에서의 구매 매출이나 구매 건수와 차이가 발생합니다.
발생하는 이유는 광고 매체 어드민의 경우 추적기간이 1일~28일까지 다양함(더 긴 매체도 있음). 추적기간이란? 광고를 보고 난 뒤에 해당 고객이 28일 이내 구매하면 광고 어드민에 해당 광고의 기여로 판단되어 결과가 발생하는 현상이 발생합니다.
왜? 이 부분이 문제인가? 28일안에 다른 여러가지 광고를 보고 구매했을 가능성, 이미 구매하고 난 뒤 노출되어도 동일한 결과가 발생합니다. 즉 나중에 다른 검색광고를 보고 구매했음에도 이 광고도 기여했다고 판단해 광고 매체 어드민은 광고 성과로 측정해서 광고주 고객에게 안내되는 오류 발생합니다.
실제 광고를 보고 구매한 고객들을 측정하는 도구가 필요합니다. GA 또는 카페24 접속통계와 같은 애널리틱스 툴을 활용합니다.
#### 직접 매출 / 간접 매출의 기준
어드민 매출 기준 1일 / 7일 / 28일, 조회와 클릭 기준 확인
GA 또는 접속통계 UE 태그 또는 UTM 태그 기준 확인
UE 또는 UTM 도구 활용 측정된 매출은 어떤 기준 일까? 해당 광고의 링크를 클릭하면 해당 이름표가 측정된 상태에서 구매하는 방식, 광고를 보고 있는 상태 혹은 보고 바로 클릭한 링크를 통해서 구매하는 형태, 직접전환에 가깝다고 표현합니다.
간접전환의 경우 위 클릭 후 1일 이내 구매, 클릭 후 7일 이내 구매, 조회 후 28일 후 구매, 컨텐츠를 반응하는 방식의 기준에 맞춰 구매가 실제로 일어난 고객의 데이터와 매칭되어 표기되는 방식입니다.

##### 매체 퍼널
1단계: 페이지뷰(사이트 방문)
2단계: 콘텐츠뷰(콘텐츠 조회, 상세페이지 조회)
3단계: 장바구니(장바구니 담기)
4단계: 회원가입(회원가입 완료)
5단계: 결제시작(구매 결제 시작 페이지)
6단계: 구매완료(실제 구매 완료)
매체 퍼널의 경우 매체 단위에 머신러닝에 사용되며 마케팅 퍼널에서와 다르게 각 이벤트가 발생하는 방식이나 내용에 대해서 이해하는 과정이 필요합니다.
각 요소가 작동해서 머신러닝이 결국 가장 높은단계 6단계에 가까운 고객을 더 높은 가치의 고객이라고 판단하기 때문에 광고 단위에서 콘텐츠를 6단계에 가까운 고객부터 여러번 보여주는 것에 대해서 이해가 필요합니다.
그래서 매체 퍼널 단계에서 1~2단계는 크게 의미있는 액션은 아니며, 3단계부터 유의미한 액션으로 판단하고 실행합니다.
최종 목표는 구매전환이므로 구매 전환이 나올 수 있는 좋은 컨텐츠를 만드는 것에 가장 집중하는 것이 전체 매출에 가장 큰 영향을 주는 지름길입니다.

## 3. 플랫폼과 고객 이해
### 플랫폼 이해: 중요성
채널별 사용자 환경
페이스북, 인스타그램, 네이버, 구글, 틱톡 등은 광고 노출 지면이나 광고 포맷이 다릅니다.
어떤 플랫폼에서는 이미지·동영상을 주로 소비하고, 다른 플랫폼에서는 텍스트·검색 기반으로 정보를 찾습니다.
사용자 심리/맥락
예: 네이버 검색 광고는 고객이 이미 고민·욕구를 가진 상태("구매 의도"가 높음)
반면 SNS 피드형 광고는 "가벼운 스크롤" 상태(인지/브랜딩 단계)에 가까움
광고 세팅 방식
페이스북 Ads: 머신러닝 최적화(전환 캠페인, 리타겟팅 등)
네이버 GFA: 다양한 언론사 지면 배너 + 검색 연계
구글 Ads: 검색용 키워드 최적화 + GDN(디스플레이) + 유튜브 등 멀티
각 플랫폼이 지원하는 타겟팅 옵션, 과금 방식이 달라 효율도 달라집니다.
알고리즘 변화 & 업그레이드
플랫폼은 주기적으로 광고 정책, 노출 알고리즘, UI를 업데이트합니다. 이를 모르면 광고 효율이 떨어집니다.
브랜드/제품 적합도
특정 제품/서비스가 어울리는 플랫폼이 있습니다(패션·뷰티는 인스타/틱톡, B2B/기술자료는 네이버 검색/유튜브 등).
플랫폼 이해 없이 무턱대고 같은 광고 메시지를 전 채널에 돌려봐야 효율이 낮습니다.
#### 결론
플랫폼 이해
각 매체 특성(사용자 목적, 광고 형태, 타겟 옵션, 비용 구조)을 알고 전략을 세워야 합니다.
"어떤 플랫폼이 우리 고객의 구매 여정 어느 단계(인지·고려·구매)에 유리한지" 파악해 맞춤형 메시지를 투입해야 합니다.
### 모바일 화면 스캔 패턴
#### 전체적인 모바일 화면 스캔(Scanning) 패턴
##### F-패턴, Z-패턴
F-패턴:
원래 웹(데스크톱) 환경에서 많이 언급. 왼쪽 상단→수평 스캔→아래 스크롤→다시 왼쪽(제목·핵심 포인트) 식으로 "F" 형태로 시선이 움직인다는 연구 결과
모바일에서도 상단(타이틀/메인비주얼)→수평(화면 폭이 좁아 수평 스캔은 짧음)→아래로 빠르게 스크롤, 눈에 띄는 지점(버튼·이미지)에서 멈춤
Z-패턴:
상단 왼쪽→상단 오른쪽→중간/아래쪽 왼쪽→오른쪽 식의 "Z" 형태 시선 이동
모바일에선 화면이 좁기 때문에 Z-패턴이 단순화되어 "타이틀→중간→하단 버튼" 정도로 관찰되기도 함
##### 스크롤의 속도 & 멈춤 지점
대부분 사용자, 처음 2~3초 동안 화면 상단을 대충 보고, 빠르게 스크롤
시선을 멈추는 지점:
굵은 타이틀/섹션 헤더
이미지·영상 썸네일
버튼(CTA)가 시각적으로 도드라지는 부분
"할인/무료" 등 자극적 단어
### 플랫폼별 고객 행동 특성
#### SNS (인스타그램, 페이스북, 틱톡 등)
피드 스크롤
유저는 정적 이미지/텍스트가 아니라 동영상·스토리를 빠르게 넘겨봄
1~2초 내 "보고 싶은 내용" 아니면 바로 스킵
시선 집중 요소
모바일 화면에서 상단(계정명, 프로필 이미지)도 시선을 끌지만, 컨텐츠 본문은 스크롤로 빠르게 지나침
첫 화면에 보이는 이미지·영상이 매우 중요 → 1~2초 안에 눈길 확 잡아야 함
탭/스크롤 패턴
좋아요·댓글 아이콘(화면 하단 정렬)이 익숙, 그 외 링크는 상단/본문 링크를 터치("링크 인 바이오" 문구, 스토리 스와이프 등)
자막·텍스트가 짧고 명확해야 스크롤 멈출 확률↑
#### 검색(네이버, 구글)
검색 결과 페이지(모바일 SERP)
최상단(광고+쇼핑탭+지식iN 등)이 시야에 먼저 들어옴, 2~3건만 보고 뒤로 스크롤
"세로 스크롤" 하면서 답변·리뷰를 확인. 광고임이 명시되어 있으면, 클릭률이 상대적으로 낮아질 수도 있음
탭(쇼핑/VIEW/이미지):
네이버 모바일에선 상단 카테고리 탭(통합·VIEW·쇼핑 등)을 가볍게 탭해 다른 결과 확인
"쇼핑" 탭에서 상품리스트 → 이미지·가격·리뷰 수 중심으로 비교
클릭 & 즉시 이탈:
모바일 속도가 느리거나, 페이지 첫 화면이 혼란스러우면, 1초 내 뒤로가기 발생
"첫 화면에 핵심 메시지/이득" 제시가 중요
#### 커뮤니티/카페(여러 커뮤니티 카페, 디시인사이드 등)
텍스트 중심
글·댓글 중심이라, "스크린샷, 캡처 이미지" 등이 첨부되지 않으면 스크롤 빠름
최대한 굵은 글자 or 핵심어 강조로 시선을 붙잡아야 함
댓글 흐름
모바일에선 게시물 본문보다, 댓글·대댓글로 '핵심 정보'를 찾으려는 경향
광고/홍보 티가 나면 반감이 커져, 이탈이나 비판 댓글 발생 가능
#### 모바일 화면(아이트래킹)
상단(탑 바)
첫 영역(배너, 계정명, 검색창, 프로필사진) → 브랜드/계정인지 파악. SNS/쇼핑몰 상단 메뉴(카테고리, 로고)도 주목
중앙부(메인 콘텐츠)
이미지는 중앙 배치가 일반적, 색/애니메이션/움직이는 요소가 있으면 시선이 멈춤
텍스트는 2~3줄 이내로 간단히 요약이 효과적
하단(Bottom CTA / Navigation)
모바일 UI 상, 하단 바에 장바구니/구매버튼/메뉴 배치하면 터치 편리(한 손 조작)
CTA(구매, 신청, 문의 등) 버튼은 화면 하단 고정 시 전환↑ ("Sticky button")
오른쪽 상단 아이콘(장바구니, 메시지 등)
손이 닿기 조금 불편할 수도 있으나, 익숙해진 패턴(쇼핑몰 앱)
너무 중요한 기능은 화면 아래쪽에 배치하는 게 일반적
#### 고객 행동의 활용
광고/컨텐츠 배치
상단에 "핵심 USP·혜택" 요약, 시선을 붙잡는 비주얼 배치
하단에 주요 CTA(구매·문의) → 바로 눌러볼 수 있게
메인 카피
모바일에선 "짧고 명확한 키워드"가 효과적 (스크롤 빨라 긴 문장 잘 안 봄)
혜택·공급 종료 시점('오늘까지 할인', '무료배송')을 초반에 강조
안정적 로딩 속도
모바일 첫 로딩 3초 넘어가면 사용자 이탈 많아짐(뒤로가기)
이미지 용량 최적화, 불필요 스크립트 최소화
장바구니/결제 단순화
옵션 선택절차 줄이고, 간편결제(카카오, 네이버, 페이코 등) 지원
중간중간 팝업(쿠폰/이벤트)이 너무 많으면 오히려 이탈유발
리뷰·Q&A 접근성↑
많은 구매자들은 스크롤 빠르게 내려서 리뷰/문의 먼저 본다
상세페이지 중간~하단에 "후기 요약/별점" 보여주고, 클릭 시 전체 리뷰로 이동
#### 결론
플랫폼별 행동 패턴
SNS=피드 스크롤 속도 빠름, 시각적 임팩트 필수
검색=의도형, 상위 노출+짧은 정보 + 신뢰(리뷰·평점)
이커머스=상세페이지 상단 3~4초 내 관심 유도 + 빠른 결제
모바일 시각 패턴
상단 → 빠른 스크롤 → 하단 CTA
중간에 이미지·굵은 글자 등 시선 멈출 포인트 필요
전략적 설계
고객이 어디를 먼저 보고 어디서 이탈하는지 알면, UI·카피·CTA 위치를 최적화할 수 있다
플랫폼 특성 & 고객 흐름 맞춰 핵심 메시지를 배치하고, 구매 과정 최소화
이로써 고객이 플랫폼·모바일 화면에서 어떻게 행동하는지 파악하고, 그에 맞춰 광고/컨텐츠/구매 프로세스를 설계하면 퍼포먼스 마케팅에서 훨씬 효율적인 전환·매출 향상을 기대할 수 있습니다.
## 4. 광고 세팅 개념
광고 세팅에서 가장 중요한 개념은 캠페인, 그룹, 소재 3가지입니다. 모든 광고는 이 3가지로 이뤄져 있습니다.
이 3가지의 구조가 있다는 것과 이걸 어떻게 활용할 것인지가 중요합니다.
모든 광고에는 CBO, ABO, DCO가 있습니다. DCO는 매체별로 없는 곳도 있기도 합니다. 다만 이 요소들을 활용하는 방법을 기본적으로 이해하고 있으면 어떤 매체를 하던 기본 이해 바탕으로 세팅하기 때문에 모든 매체를 실험할 때 의도를 부여하고 움직일 수 있습니다.
### 예산 최적화
#### 왜 해야 하는지?
ABO, CBO, DCO는 광고를 효율적으로 운영하기 위한 세팅 방식들입니다.
머신러닝이 성과 좋은 광고 소재(또는 그룹)에 집중 투자하도록 돕거나, 여러 개의 소재·타겟·예산을 자동 분배하여 광고 효율을 극대화할 수 있습니다.
다양한 가설(소재 A vs B, 타겟 X vs Y)을 동시에 테스트하면서 성과를 빠르게 검증하고 최적화하기 위한 필수 기능입니다.
#### 기본
CBO(Campaign Budget Optimization)
캠페인 단위로 일/전체 예산을 설정 → 머신러닝이 각 그룹(타겟·소재)에 성과 기반으로 자동 분배
예산이 성과 좋은 그룹에 가중, 빠른 최적화
예시로는 논타깃 vs 관심사타깃, USP1 vs USP2 등 2~3그룹을 한 캠페인에 넣고, 광고비는 캠페인 레벨에서 통합 관리
ABO(Adset Budget Optimization)
광고그룹(Adset) 단위로 예산 별도 설정
각 그룹에 균등 예산을 투입해 정확한 A/B 비교 가능
사용의 예) 소구점(카피, 이미지)이나 타겟을 1~2개씩 명확히 구분해 테스트할 때
DCO(Dynamic Creative Optimization)
여러 크리에이티브 요소(이미지·영상·카피·버튼 등)를 한 번에 업로드 → 플랫폼이 자동 조합 & 성과 높은 조합을 우선 노출
다양한 소재 조합을 짧은 기간에 동시 테스트 가능, 최적화 속도↑
사용예: "카피 3종 + 이미지 3종 + 버튼 2종"을 한꺼번에 세팅해 머신러닝으로 최고의 매칭 탐색
#### 방법
광고 구조 설계
캠페인(목표) → 그룹(타겟/소재 분류) → 소재(카피·이미지·영상)
CBO: 캠페인 레벨 예산으로 머신러닝이 그룹별 소진 결정
ABO: 그룹별 예산 배분, 각 그룹이 균등하게 테스트
DCO: 소재 여러 개(이미지, 카피 등)를 동적 조합
가설 수립 & 변수 통제
"논타깃 vs 관심사타깃, 전환율은 어느 쪽이 높을까?" → CBO로 그룹 2개 운용
"소구점 A vs B, 클릭률은?" → ABO 그룹별 예산 균등 배분 후 CTR 비교
"카피/이미지 조합 중 최적은?" → DCO 적용해 자동 조합 & 성과 모니터링
머신러닝 학습을 고려
7일에 전환 50건(메타 기준) 등 플랫폼별 최적화 가이드 준수
예산이 너무 분산되면 학습 안되므로, 소규모 그룹(2~3개) or 소재 집중이 권장
정기 모니터링
클릭률, CPC, CPA, ROAS 등 지표를 주·단위로 확인
성과 없는 그룹/소재는 Off or 수정, 잘 되는 건 확대(증액)
#### 예시
##### A. CBO 활용 예시
캠페인: "전환 목표" / 일 예산 10만 원
그룹:
논타깃 그룹
관심사 A('패션' 관심)
관심사 B('뷰티' 관심)
결과: 머신러닝이 CPA가 가장 낮은 그룹(예: 관심사 A)에 예산의 70% 집중, 나머지는 30% 소진
의도: "가장 퍼포먼스 좋은 타겟"에 자연스럽게 예산이 몰리도록 → 팀은 수시로 CTR·CPA 관찰, 효율 좋으면 그대로 유지 or 추가 증액
##### B. ABO 활용 예시
캠페인: "트래픽 유도" / 캠페인 예산: 그룹별 5만 원씩
그룹:
카피 A + 이미지1
카피 B + 이미지2
결과: 두 그룹 모두 동일 예산 5만 원씩 소진 → 1주 뒤 CTR / CPC / 전환율 비교 → "B"가 더 효율 좋으면 B로 전체 통합
의도: 특정 변수(소구점·디자인 등)를 정확히 A/B 테스트하기 위함
##### C. DCO 활용 예시
캠페인: "전환 목표", 일 예산 8만 원
그룹: 1개, DCO로 이미지 3장 + 카피 3개 + 버튼 2종 업로드
결과: 머신러닝이 "이미지2 + 카피3 + 버튼(구매하기)" 조합에서 전환↑라 판단 → 그 조합 노출이 많아짐
의도: 소재 조합 테스트를 간편히 하여, 가장 성과 좋은 조합을 자동 선별
#### 결과
자동 최적화
CBO: 성과 좋으면 그룹에 예산 집중
ABO: 균등 예산으로 정확한 비교 후 결정
DCO: 여러 소재 조합 중 성공 조합 발굴
테스트 효율
한정된 기간 & 비용으로 여러 변수(타겟, 카피, 이미지)를 동시에 검증 가능
성과 향상
최적화된 설정(머신러닝)으로 광고비 대비 최대 전환이나 매출을 노릴 수 있음
학습 및 재활용
결과 데이터("어떤 타겟/조합이 성과 좋았다")를 다음 캠페인에도 활용 → 지속 성과 향상
#### 결론
광고 세팅(CBO, ABO, DCO)은 "결국 성과를 빠르게 찾아내고, 비효율 요소를 줄이는" 과정입니다.
각 방식의 차이점과 장단점을 정확히 이해한 뒤, 목표·가설에 맞춰 적절히 선택/혼합하는 게 핵심입니다.
머신러닝에만 맡기지 말고, 마케터, 디자이너, 영상디자이너가 세운 가설(타겟·소구점·소재 등)을 통제하여 제대로 된 테스트가 이뤄지도록 유도해야 합니다.
이렇게 최적화 과정을 반복하면, 더 높은 전환 효율과 광고비 절감 효과를 동시에 얻을 수 있습니다.
### 다양한 직무에서의 활용 방법
#### 마케터 관점: 왜 이해해야 하는가?
전략적 의사결정
어떤 캠페인 예산 최적화 방식을 택할지(ABO vs CBO vs DCO)는 광고 성과와 테스트 속도에 직결됩니다.
마케터가 세팅 로직(머신러닝이 어떻게 예산을 분배하고 어떤 소구점이 성과를 내는지)을 알아야 전반적 전략을 구상할 수 있습니다.
가설 수립 & 테스트 주도
마케터는 "소구점 A vs B" 혹은 "타겟1 vs 타겟2" 등을 가설로 잡고 ABO/CBO/DCO를 통해 체계적으로 실험합니다.
이를 모르면 무분별하게 소재를 투입하거나, 잘못된 예산 배분으로 충분한 데이터 수집이 안 돼 정확한 결론을 못 냅니다.
성과 분석 & 확장
(광고)실험 결과를 보고, 성과 좋은 그룹·소재에 즉시 예산 확대·중단 결정 등을 책임지고 의사결정해야 합니다.
마케터가 구조를 이해하면, "어떤 디자이너가 만든 소재가 왜 잘 됐나"를 데이터로 피드백해 전체 운영 효율을 높입니다.
#### 디자이너 관점: 왜 이해해야 하는가?
소재(크리에이티브) 제작 방향 정립
ABO/CBO/DCO 중 어떤 방식을 쓰는지에 따라 소재 요구사항이 달라집니다.
예: DCO면 "카피 3종, 이미지 3종 등 다양한 조합"이 필요한지, ABO면 "그룹별로 확실히 구분되는 콘셉트"가 필요한지 등.
테스트 효율 극대화
디자이너가 광고 세팅 방식을 몰라, 무작정 한 종류 디자인만 만든다면, 머신러닝이 다양한 조합을 시도할 기회를 놓침.
반면 디자이너가 ABO/CBO/DCO 구조를 알면, 가장 빠르게 학습할 수 있도록 "디자인 요소를 어떻게 다양화·분화"할지 준비 가능.
팀 협업 & 피드백
데이터 결과(CTR·전환율)로 "어떤 디자인이 잘 먹혔다 or 안 먹혔다"가 나오면, 콘텐츠를 직접적으로 개선하는 주체는 디자이너입니다.
구조 이해가 있어야, "ABO 그룹 A 실패 → 어떤 부분이 실패 요소였는지" 문제를 재해석하고, 다음 디자인 수정이 가능합니다.
#### 영상 디자이너 관점: 왜 이해해야 하는가?
영상 소재가 갈리는 광고 구조 이해
예를 들어 CBO에서 여러 영상물을 동시에 올리면, 머신러닝이 가장 전환율 높은 '영상+카피' 조합에 집중 노출합니다.
영상 디자이너가 이 로직을 알면, "짧은 버전 vs 긴 버전", "타이틀 텍스트형 vs 내레이션형" 등 분화된 소스를 효율적으로 준비할 수 있습니다.
DCO(동적 크리에이티브)에서의 영상 조합
DCO가 텍스트·버튼 뿐 아니라 영상도 여러 파트로 자동 조합하는 경우가 있습니다.
영상 디자이너는 장면 분리(인트로, 메인, 클로징), 짧은 클립(5~10초) 등 기획을 단계별로 나누어 제공해야 합니다.
성과 데이터 기반 반복 개선
영상의 후킹 부분이 스크롤 스톱(멈추게) 역할을 충분히 못하면 CTR이 낮을 것입니다.
후킹과 영상의 내용 실험 결과(영상 A가 전환 2배...)가 나오면, 영상 디자이너가 "무엇이 사람을 멈춰 세웠나?" 분석 & 다음 영상에 반영할 수 있어야 합니다.
#### 결론
직무별 시너지
마케터가 "가설 및 전체 전략"을 잡고, ABO/CBO/DCO 선택 + 예산 분배, 지표 모니터링 → 피드백 주도
디자이너(그래픽·이미지)는 각 그룹·소재가 명확히 구분(소구점, 메시지, 디자인 톤)되도록 준비 → 머신러닝이 테스트 쉽게 진행
영상 디자이너는 "어떤 타겟/콘셉트"인지 알면, 각자 다른 영상 컷/길이로 소재 다양성을 제공 → DCO나 CBO 환경에서 효율적 분배
이해하지 못하면:
마케터가 혼자 기획해도 디자이너가 의미를 모르면 소재 중복 혹은 방향 불일치 → 실험 불가
디자이너/영상디자이너가 "어차피 광고는 마케터가 세팅"이라 생각하면, 소재가 단조로워 머신러닝 이점 못 살림 → 성과 저하
이해한다면:
세 사람이 동일 기획 의도 하에 가설→실험→데이터→피드백 순환 구조를 짧은 주기로 돌릴 수 있음
결론적으로 광고 성과 극대화 & 빠른 최적화 달성
이로써 마케터, 디자이너, 영상 디자이너가 각각 왜 ABO/CBO/DCO를 이해해야 하는지 명확합니다. 협업 시 서로가 어떤 형식의 소재·분화·실험이 필요한지 같이 기획하여, 최적화 과정을 신속하고 효과적으로 진행할 수 있게 됩니다.

## 5. AI 개념 및 기본 활용
### AI 활용의 중요성
#### 생산성 극대화
단시간에 방대한 작업(카피, 리뷰, 번역, 코드 등)을 처리 → 사람 혼자서는 몇 시간/며칠 걸릴 일을 AI가 수분/수십 분 안에 해냄
반복적/루틴한 작업을 대폭 줄여, 창의·전략 업무에 더 집중할 수 있음
#### 전문 지식/데이터의 즉각 재생산
AI가 이미 학습된 거대 지식 + 우리가 제공한 데이터를 자동 조합해서 새로운 아이디어나 결과물을 생성
전문가 수준의 "카피·글쓰기·데이터 분석"을 짧은 시간 안에 "보조자"로 활용 가능
#### 광범위 적용 가능성
마케팅, 디자인, 영상, 고객지원, 데이터 분석 등 거의 모든 직무 영역에 일부 도입 가능
특정 카테고리(예: 뷰티 리뷰, 이커머스 제품 설명)에 최적화된 AI를 학습시키면, 해당 분야의 업무 효율이 크게 향상
#### 미래 경쟁력 확보
AI 활용이 당연해진 시대에, AI를 모르면 결과물의 속도·퀄리티에서 뒤처질 위험
향후 더 진화된 AI(GPT-5 등) 등장 시, 지금부터 활용 습관을 들인 사람들은 급격한 변곡점에도 빠르게 적응 가능
### 기본 개념
#### AI의 활용 원리
이미 거대한 데이터(텍스트, 이미지 등)를 학습한 모델이 사용자 명령(프롬프트)를 해석 → 최적 답변을 산출
예) GPT류(텍스트), 미드저니·스테이블디퓨전(이미지), Text-to-speech(음성) 등
#### 핵심 요소
프롬프트(Prompt): AI에게 일을 시키는 구체적 명령·조건
데이터: AI에 공급되는 예시·지식
결과물: AI가 출력한 텍스트·이미지·영상 등
이 3박자를 잘 다루어야 원하는 결과를 얻음
#### 장단점
장점: 빠른 속도, 대용량 작업, 다양한 창의적 산출 가능
단점: "할루시네이션(거짓/허구 정보)" 문제, 사실 검증 필요, 사용자의 명령(프롬프트)가 정확해야 성능 극대화
### 프롬프트 설계와 데이터 제공
#### 프롬프트 설계
목적/역할/조건을 구체적으로 적는다. (예: "나는 마케터, 너는 마케팅 전문 카피라이터. 제품은 2030여성용 뷰티크림. 할인 키워드 넣어줘" 등)
예시·샘플 데이터(잘된 카피, 글 스타일)를 AI에게 보여주면 AI가 그 패턴을 학습해 더 적합한 결과를 만듦
#### 데이터/지식 제공
AI가 이미 어느 정도 기본 지식을 갖고 있지만, 이커머스 제품 리뷰나 내부 통계 등 구체 자료를 추가로 주면 산출물 품질↑
자료는 텍스트(리뷰, 고객문의, 상품설명 등) 형태로 제공
#### 검수(Iterate)
첫 결과물을 보고, 부족한 점(사실오류·어색한 표현) 지적 → AI에게 수정 요청
반복적으로 대화하며 최적 결과에 도달 (사람→AI→사람→AI 순환)
#### 실무 적용
카피, 블로그 글, 광고용 이미지·영상 제작, 데이터 분석, 기획서 초안 등 다양한 목적으로 활용
필요한 부분만 사람이 최종 다듬어 배포하면 시간 대폭 절감
### AI를 활용한 마케팅 사례
#### 이커머스 상품 카피 생성
마케터: "20대 여성용 립스틱 홍보 카피 작성해줘. 톤&매너는 발랄, 문장 3개 이내, 할인 키워드 꼭 포함."
AI: "지금 20% 할인! 당신의 데일리 립스틱" 등 10개 버전 한 번에 제시
사람: 마음에 드는 2~3개를 고르고, 최종적으로 문구 일부 수정
문구 수정 시 GPT > 클로드 활용 시 더 사람이 말한 듯한 문구로 대량 수정 및 다양한 생산 추가 가능 (AI to AI)
#### 리뷰 분석 & 인사이트
데이터: 자사몰 고객 리뷰 2000개 txt로 정리
마케터: GPT에 해당 텍스트 일부 or 주요 문장들 chunk로 feed → "부정적 키워드, 긍정 포인트, 자주 언급된 문구 요약해줘" 명령
AI: "부정: 배송 지연(20%), 반품 처리(10%)... 긍정: 품질(50%)" 식으로 패턴 요약 & 해결책 제안
활용: 제품 개선·광고 메시지('빠른 배송') 강화
글자수 일정 이상의 리뷰 필터 후 추출
특정 재미있는 예시 리뷰를 AI에게 학습, 경쟁사 리뷰 속에 광고 소재용 재미있는 리뷰 기준에 맞게 리뷰 추출
#### 이미지 합성·배경 생성 (광고 배너)
디자이너: 미드저니/스테이블디퓨전 등 이미지 생성형 AI에 "여름 해변 느낌 배경 + 제품(선크림) 강조" 프롬프트 입력
AI: 몇 초 안에 썸네일 후보 4~8장 생성
사람: 그중 1~2장 선택, 포토샵 약간 보정, 텍스트 오버레이 후 광고 배너 완성
디테일한 부분은 디자이너가 컴피 UI 활용해 디벨롭
딥페이크 모델 활용해 모델생성, 생성된 모델 데이터 기반 컴피 UI 활용 디테일한 모델 이미지 생성
### 결과
작업 속도 대폭 향상
하루 걸릴 광고 카피+이미지 기획을 1~2시간 만에 끝내거나, 방대한 리뷰 분석을 수십 분 만에 처리 가능
아이디어 다양성
사람 혼자 구상하기 어려운 창의적 문장·이미지·영상 초안이 쏟아짐
다수 시도→베스트 골라 다듬어 "퀄리티+실험 속도" 둘 다 상승
성공률 개선
반복적 실험(A/B) 때 AI가 다수 시안을 빠르게 생성 → 전환율 높은 버전 찾기 쉬워짐
### 결론
AI는 만능이 아님, "사람의 기획/검수 + AI의 무한 생성·분석 능력" 결합이 핵심
프롬프트(명령) 설계 및 필요한 데이터 준비가 결과물 품질 좌우
이커머스, 마케팅, 디자인, 데이터 분석, 기획 등 어디든 "반복적이고 창의 요구되는 작업"에 AI 접목하면 엄청난 시간 단축과 성과 향상을 경험 가능
시대 흐름상 AI 활용 능력이 "업무 경쟁력"이 되는 시기가 이미 도래했으며, 빠르게 습관화해야 변곡점에 뒤처지지 않음
이렇게 요약하자면, AI 개념과 활용을 잘 이해하면 업무 효율이 높아지고, 다양한 아이디어를 즉시 시도해볼 수 있어 결과물의 수준과 속도 모두 향상시킬 수 있습니다.
## 6. 심리학적 접근: 무의식, 인식, 설득
### 전체 개요
콘텐츠 기본 구조
문제제기 → 문제해결 → 해결 근거
광고나 상세페이지 등, 사람을 설득해 행동(구매·문의 등)을 유도하는 모든 컨텐츠는 이 3단계 구조를 동일하게 갖습니다.
3대 요소
무의식: 시선·후킹을 유발 (사람이 본능적으로 끌리는 욕구나 학습된 반사적 반응)
인식: 고객의 상황·장면을 공감, "아 이건 내 이야기네" 하도록 유도
설득: 사회적 지위·숫자적 근거·통념 등으로 "이 제품이 정말 믿을만하다"라고 납득시키기
### 무의식 파트
핵심 역할: 고객이 "아무 생각 없이" 광고를 스치다가도 '멈추게' 만드는 첫 단계
#### 무의식의 두 축
욕구 기반
기초 생리(먹고·자고·싸고)·성욕 등
예: 섹시·멋짐·맛있음·편안함 등 "본능적 끌림"을 자극하면 CTR(클릭률) 상승
학습·교육 기반
"의사 = 신뢰", "약국 = 안전", "권위자 = 믿음" 등 사회에서 오랜 기간 학습된 가치
예: "강남 약국 입점" → '검증된 제품'이라는 무의식 작동
#### 적용 가이드
후킹 이미지/카피를 먼저 찾기
욕구 기반: 예쁜 모델, 통굽 구두(편해 보임), 뱃살·여드름 등 '고통 장면' 등
학습 기반: 의사·약사·박사·연구소·명품 브랜드 등 "권위·신뢰" 상징
테스트 후 선택
여러 후킹안(이미지·키워드)을 제작해 CTR 시험. 무의식적으로 가장 많이 눌리는 쪽을 우선 채택
### 인식 파트
핵심 역할: "아, 이거 나 문제랑 비슷하네?"라고 느끼며 광고를 계속 보게 만드는 단계
#### 인식 요소
고객 상황
"여드름에 고민하는 고등학생", "통굽 구두 원하는 직장인 여성", "뱃살 고민 중년 남성" 등
실제 생활·경험·장면을 구체적으로 표현
장면 연출 & 카피
"수업 중 거울 볼 때마다 트러블 눈에 띄어 자존감 낮아지는 10대"
"출근길 발 아픈 구두 신느라 일하기 힘든 XX세대 여성"
이렇게 구체적·생생하게 묘사하면, 고객이 "내 이야기"라고 느끼게 됨
#### 적용 가이드
후킹(무의식) 후 '나도 이런 상황이야' 공감 단계
예: "단 10일이면 여드름이 사라져요" (카피) + "강남 약국 입점 인증샷" (이미지)
경험치·리뷰 인용
"직장인 여성 52% 하루 8시간 서 있거나 이동 → 구두 불편" 식 구체적 통계
### 설득 파트
핵심 역할: 구매·상담 등 '행동'으로 이어지도록 "근거 + 이유"를 제시
#### 9대 설득 요소 (주요 예시)
권위(사회적 지위)
의사·박사·교수·연구소 협업 등
"유명 XX 연구소 인증", "현직 의사가 추천"
숫자적 근거
임상결과 %·구체적 판매량·리뷰 갯수 등
"7일에 여드름 93% 개선, 2만 개 판매 돌파"
사회적 통념
"중국당면 = 얇고 탱글→ 초얇 팬티 연상", "골프 = 품격 있는 취미→ 프리미엄 느낌"
대중적 인식·통념을 끌어와 제품 포지션
희소성
"한정수량", "지금만 할인" → 서두르게 함
상호성
샘플·무료체험 등 "먼저 베풀었다" → 고객이 '은혜 보답' 심리로 결제
신뢰·진실성
제3자 리뷰·"내돈내산" 리뷰·체험단보다 더 자발적 후기 등
(나머지 일관성·호감요소·사회적 증거 등도 비슷한 원리로 "고객이 믿고 사고 싶게 만드는 근거"를 제공)
#### 적용 가이드
무의식·인식으로 끌어온 고객에게 마지막 '구매 확신' 심어주는 핵심
예: "의사 협회 인증, 7일에 93%가 좋아졌다는 실제 임상, XX약국 1320개 판매, 현재 1000세트만 할인 진행 중"
전환 페이지(상세페이지·영상 후반부 등)에 집중 배치
문제해결 근거를 폭발적으로 제시
### 최종 적용 흐름
(1) 무의식 (후킹)
본능(욕구) or 학습(권위) 자극
예: 예쁜/멋진 모델, 박사·의사, 유명 브랜드 시각적 요소
(2) 인식 (공감)
구체적 상황·장면 제시
예: "여드름이 심해 고민인 10대", "발 아픈 구두 신는 30대 여성"
(3) 설득 (결정)
숫자·권위·희소성 등 명확 근거
예: "강남 약국 판매 1위", "누적 2만 명 사용, 재구매율 80%"
### 요약 체크리스트
무의식 후킹
모델·이미지·색감·관습 등으로 즉각 관심을 끌 요소가 있는가?
'권위'나 '본능적 욕구'를 즉각 자극하는 포인트 배치했는가?
인식(공감) 단계
타겟 상황 묘사가 구체적인가?
"이건 내 문제와 같다"는 대사가 보이는가?
설득(결정) 단계
권위, 숫자, 희소성, 통념 등 9대 설득요소 중 최소 3가지 이상 반영했는가?
구매가 이루어지도록 이벤트·조건·사회적증거·리뷰 등이 잘 제시돼 있는가?
랜딩·후속 페이지
무의식+인식+설득이 잘 연결되도록, '이야기 흐름'이 순서 맞게 전개되는가?
최종 근거(리뷰·누적판매·연구결과 등) 제시 후 구매버튼이 배치됐는가?
### 결론
광고·상세페이지·SNS 콘텐츠 모두 "무의식(후킹)→인식(공감)→설득(근거)" 구조로 만들 때 전환률이 높아집니다.
무의식에서 "시선"을 잡고, 인식으로 "아, 내 얘기구나"를 느끼게 한 뒤, 설득(숫자·권위·희소성·통념 등)으로 "이유"를 납득시키면 구매로 이어질 확률이 크게 상승합니다.
특히 무의식 요소(본능적 욕구, 학습된 권위·키워드)와 설득 요소(권위·통념·숫자근거)가 함께 적용되면 불황에도 매출을 만들어낼 가능성이 높아집니다.
### 추가 자료
[아이보스] 무의식, 인식, 설득 3가지만 알면 100억 매출 뚝딱
## 7. 문제 도출과 해결 프로세스
### 왜 필요한가?
정확한 문제 파악 없이 해결책만 찾으면 낭비
코끼리의 '꼬리만 보고 밧줄'이라거나, '다리만 보고 기둥'이라고 생각하는 오류처럼, 전체를 보지 못하면 헛된 해결책으로 시간을 허비하게 됩니다.
실질적인 원인 발견 → 효율적인 해결책
퍼포먼스 마케팅, 상품 기획, 운영 등 어떤 업무이든, 문제의 '본질'을 다각도로 분석해야 근본 해결이 가능합니다.
팀원·조직 간 협업 용이
문제 구조를 정리(Why 트리, MECE 등)해두면, 팀원끼리 "어디가 문제가 됐는지, 어떤 파트가 해결해야 하는지"를 명확히 공유할 수 있습니다.

가설 검증(실험/데이터 확인)
해결책 도출 & 실행
문제 파악 단계에서, MECE(상호 중복·누락 없이 쪼개기), Why 트리(원인 탐색) 같은 프레임워크가 유용합니다.
해결책 단계에서는 How 트리(어떻게 해결?)를 세워서 구체적 실행방안을 도출합니다.
### 문제 파악 방법
#### 다차원적 사고
코끼리 비유처럼, 한 부분(광고 세팅·컨텐츠·외부 환경...)만 보는 게 아니라 전체 맥락을 확인합니다.
"광고 효율이 낮다" → 외부적(시장, 매체, 바이럴 이슈), 내부적(디자인, CS, 리뷰 관리), 통제불가(시즌, 법률, 환경) 등으로 구분합니다.
#### 구체적 정보 수집
고객 리뷰, 시장 흐름(경쟁사·트렌드), 내부 CS 기록, 실제 광고 세팅 상태 등 객관적 자료를 수집합니다.
감정·주관보다 데이터 중심으로 문제점을 찾습니다.
#### MECE / Why 트리
"왜 효율이 낮을까?" → 세부 원인들을 가지치기 (왜? A / B / C)
각 하위 원인 중 더 세분화할 부분("광고 세팅" → 타겟, 입찰방식, 소재...), "컨텐츠" → 리뷰·카피·영상 등
### 문제 해결 방법
#### 하늘·비·우산 프레임워크
하늘(사실): 현재 객관적 상황("구름이 끼고 있다 → 자사 점유율 감소")
비(해석): 사실을 보고 "비가 올 것 같다 → 시장에서 경쟁사가 약점인 친환경 제품이 떠올랐다"
우산(행동): 우리가 취해야 할 구체적 행동("우산을 챙기자 → 친환경 상품 기획·마케팅 전개")
#### How 트리
왜 트리에 이어, "그럼 어떻게?"를 가지치기합니다.
예: 리뷰 문제 → (1) 바이럴(내부·외부 리뷰 확대), (2) 고객 리뷰 이벤트, (3) CS 매뉴얼 개선
#### 우선순위 선정 & 실행
해결책이 여러 개일 경우, 영향도 높은 방법 먼저 시도 → 성과 검증
기획→실행→복기→개선(짧은 주기로 반복)
### 실제 사례를 통한 이해
예시 A: 광고 효율이 낮다
문제 파악
(외부) 광고매체 이슈, 바이럴 부족, 시장 구도 변화
(내부) 디자인·컨텐츠·CS·물류, etc.
(통제불가) 시즌·법규 등
Why 트리로 세분화 → "광고 세팅 불안정 + 고객 리뷰 부정 + 계절적 비수기"가 근본원인이라고 판단
해결책(How)
광고 세팅: 타겟 재설정/AB테스트, 입찰방식 변경
고객 리뷰: 리마인드 이벤트, 리뷰 관리 솔루션 도입, 정성적 분석으로 메시지 개선
비수기: 세일 프로모션, 새로운 USP 소구로 시선 끌기
결과
2주 후 CTR/전환율 향상, 불만 리뷰 줄어듦
예시 B: 기미 화장품이 여름에만 잘 팔린다
문제 파악
(Why) 여름 자외선 시즌이 끝나면 검색량 급락
내부적으로 "컨텐츠 강화 필요" 판단했지만, 시즌 문제(통제불가)가 핵심
해결책(How)
가을/겨울엔 "잡티·미백" 다른 소구점으로 상품명·광고 메시지 바꾸기
라인업 확장(보습+기미 겸용) → 시즌 상관없이 사용 가능 컨셉
결과
가을에도 어느 정도 구매 유지, 연중 마케팅 가능
예시 C: 리뷰 관리 문제
문제 파악
부정리뷰 많은데 적절 대응 안 함 → 전환율 저하
리뷰 자체가 적은 편 → 사회적 증거(다수후기) 부족
해결책(How)
(바이럴) 우수후기 이벤트, 체험단·인플루언서 리뷰 늘리기
(고객리뷰) SNS·커뮤니티도 모니터링, FAQ 정리 & CS 매뉴얼
(시스템) 별도 자동 리뷰수집 툴 & 리뷰 자동분류(AI)로 빠른 대응
결과
긍정적 리뷰 비중↑, 부정리뷰 빠른 해결 → 구매전환 향상
### 결과
문제 "전체"를 볼 수 있는 시각 확보
코끼리의 부분만이 아니라, 시장·고객·제품·내부 프로세스를 아우르며 원인을 구조화
정확한 우선순위 & 실질 해결책
문제 요인별 MECE 분류 → 해결책(How)도 단계·책임부서별로 명확해져, 실행력↑
재발 방지 & 반복 학습
문제-해결 과정 문서화 → 다음 유사 문제 시 신속 대처
시행착오가 줄어들어 효율적 운영
### 결론
문제 도출(파악)과 문제 해결은 "1) 구조화와 2) 다각적 시선"이 핵심입니다.
MECE / Why-How 트리 / 하늘·비·우산 등의 프레임워크로 원인을 세분화하고, 행동(우산)을 구체적으로 정의하면 실행의 정확도가 높아집니다.
특히 마케팅·이커머스에서는 "광고 효율"만 보지 말고 외부(시장·매체), 내부(콘텐츠, 기획, 컨셉 잡기), 통제불가(시즌·환경) 등으로 넓게 살펴야 진짜 문제를 찾을 수 있습니다.
결국, 다차원적으로 문제를 바라보고, 실행→검증→개선을 빠르게 반복하면 성과를 극대화할 수 있습니다.
## 8. 고객의 문화와 언어 이해
### 문화와 언어 이해의 중요성
동일한 언어권이라도 나이·지역·환경이 다르면 문화적 차이가 큼
예) 20대가 40대 고객에게 맞는 메시지를 쓰기 위해선, 40대의 문화와 언어적 특징을 반드시 파악해야 함
인문학적 접근을 통해 사람(고객)의 심리·습관·행동 방식을 깊이 이해할 수 있음
단순 설문보다 "그들이 무엇을 먹고, 보고, 말하는지" 문화와 언어적 뉘앙스를 파악해야 함
소비자 행동은 "무의식+심리+문화+학습 경험"이 복합 작용
숫자로만 설명할 수 없는 "비합리적" 의사결정이 많음 → 문화·언어의 맥락까지 챙겨야 제대로 된 타겟팅·카피가 가능
### 고객 문화 이해를 위한 방법론
#### 핵심 개념 8가지 (인문학·심리학 관점)
사람의 심리는 대부분 비슷하게 작동
깊이 파고들면, 인간의 욕구(먹고·자고·싸는 등) 같은 원초적 측면에서 공통분모가 많음
여러 사람들의 페르소나를 만나보기
직접 인터뷰·리서치·리뷰 관찰로 그들의 말투·행동·환경까지 파악하기
인문학은 우리의 언어와 밀접
고객의 문화·언어 표현 방식을 이해해야 설득력 있는 메시지 가능
인문학 이해 → 심리학 이해 쉬워짐
인문학은 '인간 전체'에 대한 학문이므로, 사람이 왜 그렇게 행동하는지 직관적으로 파악 가능
인문학(문화·역사·철학) = 결국 고객 이해
특정 연령·지역·직업·가치관에 따라 구매 포인트가 다름
인간의 문화에 관심 = 가장 중요한 마케팅 포인트
'그들은 어떤 문화·커뮤니티에서 어떤 코드를 쓰는가?'가 구매 행태에 직결됨
인간 탐구 = 심리학으로 발현
반복적 패턴, 집단적 행동(베르테르 효과 등)을 이해하면, 효과적인 마케팅 기획 가능
인간도 결국 동물
원초적 본능(본능·습관·무의식) → 불합리·비합리 결정 많음 → 감성·문화·습관 공략 필요
### 실행 단계별 가이드
#### 사전 조사(리서치)
목표 고객(타겟 페르소나) 설정
연령, 성별, 직업, 라이프스타일, 주 사용 플랫폼, 주 가치관 등
예) "40대 기혼 여성" → 구체화: "40대 수도권·자녀 2명·직장 경험 있음·네이버 카페 활동 多"
문화·환경 파악
해당 고객층이 주로 활동하는 커뮤니티·SNS·카페 파악 (맘카페, 특정 취미 동호회 등)
그들이 자주 쓰는 용어·밈·줄임말 등 수집
예) 20대 MZ 소비자의 경우 "인스타 감성" "OOTD" "~stagram" 등 신조어 파악
리뷰·댓글 정밀 분석
경쟁사·유사 상품의 리뷰(긍정/부정), 커뮤니티의 사용 후기 등에서 반복되는 단어·문장 정리
해당 고객층의 불만·열광 포인트 및 "그들만의 은어" 확인
1차 가설 설정
"이들은 ○○ 문제에 가장 공감이 크다"
"언어 표현은 ○○형 말투(예: 친근한 반말, 존칭 등)를 선호한다"
"브랜드 신뢰를 주려면 ○○(전문가, 특정 이미지)를 활용해야 한다"
#### 직접 소통(인터뷰·FGI·실험 캠페인)
짧은 전화 인터뷰/실험적 설문
타겟층의 주요 불편사항, 선호 표현, 자주 쓰는 어투 질문
"요즘 ~를 뭐라고 부르세요?" 처럼 문화어휘·밈 등에 대한 FGI(Focus Group Interview)
가설 검증용 '소규모 광고 캠페인'
A/B 테스트: 동일 제품이지만 카피를 20대 언어ver. VS 40대 언어ver. 로 나눠서 CTR·CVR 비교
짧은 기간이라도 실제 성과를 보면, 어떤 톤 앤 매너가 고객에게 통하는지 데이터로 확인
굳이 20대 언어 40대 언어 테스트는 지양, 어차피 40대 언어가 잘 먹히긴 함. 다만 20대 언어에서도 필요한 포인트가 있고 필요한 가설이라면 진행
결과 분석 & 추가 보완
"40대가 더 반응 좋은 카피는 '○○' 표현을 쓰는 것이고, 20대에게는 다소 촌스럽게 보였다"
"가설 중 일부만 맞았음 → 새롭게 발견한 언어 요소를 정리"
#### 콘텐츠·랜딩 제작 시 적용
메인 메시지(Headline) → 고객의 언어·맥락 반영
예) "힘든 다이어트? 저희는 '아줌마 살빼기' 프로젝트로 쉽고 빠르게 만든다!"
'아줌마 살'이라는 다소 직설적 표현이 타겟(40대)에게 거부감이 없는지, 직접 검증 필요
예) "캠핑 가기 전, [바베큐 대용량 패키지]가 최적!" 처럼 문화를 직접 언급
비주얼·이미지에서 그들이 익숙해하는 요소
페르소나가 가장 많이 들르는 장소, 드레스코드, 일상 소품 등을 이미지로 활용해 "바로 내 모습"이라 느끼게 만들기
예) 5060 남성이라면 중장년들이 자주 쓰는 색감·환경(골프장, 낚시터, 등산) vs 20대 여성은 카페·핫플레이스 인테리어
후킹 & 설득 파트에서 '문화·언어' 극대화
설명 없이도 그들끼리는 '당연히 알아듣는 표현'을 활용하면 "내 얘기네!" 공감↑
숫자·근거 제시 시에도 그 문화에서 신뢰하는 '전문가·매체' 인용 (예: 맘카페 언급, 특정 유명 플랫폼 리뷰 인용 등)
랜딩 상세 내용
"왜 이 문제가 생기고, 왜 우리가 해결 가능?" → 해당 타겟층이 이해하기 쉬운 비유·예시 넣기
"쓰면 이런 변화가 온다" → 그들의 실제 라이프스타일 변화(차 트렁크 정리, 아이 등하원, 직장 동료와 점심 등)로 구체화
#### 모니터링·지속 개선
주기적 댓글·리뷰 모니터링
계속 새로운 밈·표현이 생기기도 하고, 트렌드가 빠르게 변함
부정 리뷰에서 "이 표현이 불쾌하다" 같은 피드백 나오면 바로 수정·보완
새로운 이슈·문화 코드 캐치
타겟층이 열광하는 새 OTT 드라마, SNS 챌린지, 뉴스 이슈 등을 빠르게 파악
"문화=소비 행동"이 곧장 연결될 수 있으므로, 적용 가능한지 '가설 & 실험' 진행
팀 내부 공유
마케터·디자이너·영상 디자이너·경영관리팀이 "고객 관점"을 동일하게 갖도록
인사이트 노트를 팀 내 공유 (언어 표현, 최근 트렌드, 광고 성과 지표 등)
### 유의 사항 & TIP
고객마다 언어·문화가 확연히 달라짐
하나의 페르소나에만 갇히지 말고, 최소 2~3개 유형 이상으로 나눠서 관찰
시대적 흐름 & 계절별 이슈 고려
불경기 심해지면 "가성비·실용" 강조 언어가 통함, 호황기엔 "가심비·프리미엄" 언어가 반응↑
인문학적 해석은 절대적인 과학이 아님
정성적 분석(사람·문화) + 정량적 분석(광고 지표·데이터)을 함께 접목해야 안정적인 결과
너무 과한 '밈·줄임말'은 역효과
40대·50대에게 10~20대가 쓰는 과도한 신조어는 불편감 초래할 수 있음
반대로 20대에게는 30~40대가 쓰는 올드한 표현이 고루하게 느껴질 수 있음 → 적절한 균형 필요
### 예시 적용 시나리오
사례: 40대 여성용 다이어트 식품 광고 기획
사전 조사
맘카페·키즈맘포럼 분석 → "아줌마 뱃살" "아이 돌봄 때문에 운동 못 함" "별별 다이어트 시도해봄" 등 자주 언급
대체로 존칭보다는 반말·친근체를 쓰지만, 너무 유행어 쓰면 촌스럽다고 여김
직접 소통
짧은 설문조사: "가장 힘든 부분?" → 운동 시간 부족·식단 지키기 어려움
A/B 테스트:
"우리 상품, 7일이면 2kg 감량" 카피 (공식적 어투)
"7일 만에 -2kg, 애 셋 키우는 엄마도 가능" 카피 (현실감+반말 섞인 어투)
결과: 2)에서 CTR과 전환↑
콘텐츠 구성
헤드라인: "아줌마 살!? 애 키우면서도 뱃살 뺐어요!"
메인 비주얼: 아이·주부 일상 속 음식 조절 모습 & 편안한 홈트 장면
설득 파트: 맘카페 인증·수치("만 명 중 89%가 만족") & 전문가(영양사·체형관리사) 한마디
모니터링·개선
한 달 뒤 댓글 모니터 → "제품 효과보다 엄마 살빼기라는 말이 좀... 불편" 일부 피드백
해당 표현 수정 → "엄마의 숨은 라인을 찾아줄 간편 다이어트"로 변경
### 결론
고객의 문화와 언어 이해는 단순히 "어떤 단어를 써야 클릭이 잘 나오는지"를 넘어, 그들이 살아온 환경·문화·역사·철학적 기반을 폭넓게 이해하는 과정입니다.

 이를 위해 인문학과 심리학적 관점을 접목하여, 실제로 인터뷰·커뮤니티 리서치·가설 검증 광고 등을 수행해야 합니다.
마케터뿐 아니라 디자이너, 영상 제작자, 경영관리팀 모두가 같은 고객 이해 프레임을 가져야 일관성 있는 메시지가 만들어지고, 그 결과 불황이든 호황이든 탄탄한 고객 반응을 이끌어낼 수 있습니다.
핵심 요약
"문화·언어적 맥락"을 모르면, 고객에게 '말이 안 통하는 광고'가 되기 쉽다.
인문학적 접근(사람·환경·문화·역사·철학) → 고객 심리(무의식+학습) 연결 →
 실제 카피·비주얼·메시지에 투영 → A/B 테스트·지속 모니터링 → 반복 개선.

## 9. 퍼포먼스 캔버스
### 퍼포먼스 캔버스란?
[IMG]
#### 정의
마케팅 기획의 시작점이자, 마케터·디자이너·영상 디자이너 등 팀원들이 한 장의 시트(캔버스) 위에서 제품/고객/차별점/근거/메시지를 종합적으로 정리·공유하는 도구입니다.

기획 과정에서 방향을 잃지 않도록 가이드해주고, 팀 간 소통을 강화해 성과를 내기 위한 베이스맵 역할을 합니다.
#### 전제 지식
퍼포먼스 캔버스를 제대로 작성하기 전, 아래 개념들이 정립되어 있으면 더 효과적입니다:

무의식·인식·설득(심리학적 관점)
문제도출 > 문제해결 프로세스
마케팅 기본 용어
고객의 문화와 언어 이해

### 퍼포먼스 캔버스의 작성 방법: 단계별 가이드
아래는 남성 바디워시 예시를 따라가는 흐름으로 설명합니다.
##### 제품의 본질적 이유 파악
"이 제품은 어떤 문제를 해결하기 위해 태어났는가?"
예) 남성 바디워시: "몸 냄새 제거+피부 청결 유지"가 본질
경쟁사·시장조사로 문제 정의 확인
시장에는 이미 '샤워젤' '남성용 바디워시' 등이 존재함 → 그럼에도 불구하고 "향 지속력"이나 "각질·냄새" 해결이 부족하다는 점이 본질을 강화할 근거가 될 수 있음
결과물: "고객의 (특정 냄새/청결) 고통을 해결해주기 위해 태어났다" 정도로 명확히 기술
##### 고객(타겟) 정의
인구통계학(Demographics)
연령/성별: 예) "주로 30~50대 남성, 그중 40대가 핵심"
직업/환경: 대면 업무 많아 땀·냄새 고민, 사무직이지만 하루 종일 셔츠 착용 등
검색·SNS 행태: 검색엔진(네이버), 맘카페보다는 남성 커뮤니티(예: 뽐뿌, 스포츠 동호회 등)
욕구와 욕망
데이터 수집: 경쟁사(우르O스 등) 리뷰, 키워드 도구, SNS 댓글 등에서 "냄새" "향" "잔향 지속"이 반복적 언급
심층 욕구:
청결(기본 생리적 욕구)
사회적 인정('냄새=비호감' 방지)
성적 매력(향기로 인해 자신감↑)
결론: "남성들이 향/냄새 문제로 고민하고, 좋은 향이 나면 자신감과 사회적 매력이 올라갈 것이라 믿는다."
페르소나
예시:
이름: "김상훈(45세, 영업직, 대면 미팅 잦음)"
상황: "한여름에도 정장 입어야 하니 땀·암내로 고민, 바디워시 써봐도 향이 금방 날아가서 아쉬움"
기대 효과: "퇴근 후까지 상쾌한 냄새 유지, 동료·거래처와의 대면에 자신감"
##### 6가지 근거 요소
숫자적 근거
"냄새 93% 감소 임상 결과"
"7일 사용 후 설문, 87%가 '훈남 향기' 느꼈다" 등 구체 수치
사회적 근거
예) "전문가(피부과 의사) 자문 / 유명 남성 뷰티 크리에이터 언급"
사회적 통념
"중년 남성의 체취는 나이가 들수록 강해진다"
"청결·향 관리는 요즘 필수"
경쟁력 (제·기술·원가·전통·커뮤니티 등)
"특허 받은 냄새 중화 성분" or "50년 전통 화장품 기업이 개발"
컨셉
"액취증 전문가가 만든 '진짜 냄새차단' 고기능 바디워시"
기능·프리미엄·전문성 등 한두 문장으로 요약
기능적 특화 포지셔닝
예) "땀·암내 방지에 특화된 바디워시" / "과도한 체취 걱정 없는 바디솔루션"
##### 심리학적 설득 요소
앞서 도출한 근거 요소를 설득 심리학에 맞춰 콘텐츠(광고·랜딩)에 녹이는 단계
예시 적용(중요 포인트만 발췌):
상호성: "무료체험분 50ml 증정, 써보고 결정하세요!"
희소성: "초도 한정 1,000세트, 품절 시 재입고 불투명"
권위: "피부과 의사·3대 학회 논문 기반"
사회적 증거: "벌써 5,000명 이상의 40대 직장인 후기"
호감/연대감: "같은 고민을 가진 40대 영업사원, 1주 만에 냄새 불안↓"
숫자적 근거: "냄새 감소 93%, 향 지속 12시간 임상 완료"
##### 최종 핵심 메시지 설정
핵심 메시지/문장
예) "체취 고민 많은 3040 남성도 하루 종일 상쾌하게! 특허 성분으로 12시간 냄새 차단"
주요 비주얼 아이디어
(1) 잘생긴 중년 남성이 흰 셔츠 입고 부드럽게 웃는 장면 (후킹)
(2) 거품·액상 씬 연출 → '진짜 강력 세정' 이미지
(3) 숫자(93%, 12시간) 강조 그래픽
실행 계획
A/B 테스트: 위 이미지·카피로 2~3개 광고그룹 만들고, 어느 쪽이 CTR·CVR 높은지 비교
랜딩페이지: "문제 제기(냄새) → 해결책(특허 성분) → 근거(임상·전문가) → 구매 유도" 구조
CRM/리마케팅: 트라이얼 신청자에게 "연장 구매 할인+후기 인증 이벤트" 등 상호성·호감 요소 반복
### 활용 사례
#### 실제로 퍼포먼스 캔버스를 채우는 흐름 요약

### 유의 사항 & TIP
자료·데이터 기반
검색량·연관 키워드·리뷰 분석·A/B 테스트 결과 등 "객관적 근거"를 모아 캔버스에 기입
단순히 '한 번 작성' 후 끝이 아님
캠페인 진행하면서 신설 리뷰·댓글·광고 성과를 반영 → "퍼포먼스 캔버스"를 계속 업데이트
마케터-디자이너 공동 작업
마케터는 데이터·카피 아이디어를, 디자이너(혹은 영상팀)는 시각화·영상 구도를 서로 협의
문제-해결-설득 구조를 잊지 말 것
"문제 제기(냄새 고민)" → "해결안(제품 본질, 기능, 근거)" → "설득(심리학 요인, 권위, 증거)"
실체 없는 컨셉
광고주가 "임상·특허 없음" 등 근거가 전혀 없다면, 우선 소규모 테스트로 가설 검증 후, 실제 권위 확보(간단 임상·인플루언서 후기 등)를 진행하도록 제안

## 10. 마케팅 콘텐츠 기획 및 분석
### 시장조사 방법론
중요성
검색량과 인구통계 분석은 타겟 고객을 객관적으로 이해하고, 효과적인 마케팅 전략 수립을 위한 가장 기본적인 단계입니다. 검색량은 소비자의 실제 관심도와 수요를 반영하며, 인구통계학적 정보는 타겟 고객의 명확한 특성을 보여주어 효율적인 광고 타겟팅 및 콘텐츠 기획에 필수적입니다.
프로세스
주요 키워드 검색량 조사
네이버 키워드 도구, 블랙키위, 썸트렌드 등 데이터 분석 도구 활용
메인 키워드 및 연관 서브 키워드의 검색량 체크
키워드별 월별 검색량, 경쟁률, 연령별 검색비율을 파악
키워드별 인구통계학적 특징 분석
키워드 도구에서 연령대, 성별, 관심사 등의 세부 정보를 확인
주요 키워드와 서브 키워드를 통해 검색자의 주요 연령층과 성별을 파악
제품과 관련된 주요 고객의 특징을 명확히 정리
조사 결과를 통해 타겟 고객 정의 및 세분화
검색량 및 인구통계 데이터를 바탕으로 고객 프로필 작성
주력 고객층의 연령, 성별, 직업군, 소비 성향 등을 구체화하여 타겟 설정
유의사항 & TIP
검색량이 높은 키워드가 반드시 구매 전환으로 이어지지 않을 수 있으므로, 구매력과 연관된 키워드 위주로 추가 분석이 필요
키워드 조사는 주기적으로 반복하여 최신 데이터를 유지하는 것이 중요
연관 키워드와 서브 키워드까지 폭넓게 조사해야 고객의 실제 관심사를 정확히 반영 가능
예시
제품: 남성 전용 바디워시
키워드 조사 결과:
주요 키워드: "남성 바디워시", "바디워시 추천"
주요 연령층: 40~50대 (전체 검색자의 60%)
성별: 남성 중심 (약 70%)
인구통계학적 특징 정의:
40~50대 남성, 사무직 또는 비즈니스맨, 대면 미팅이 많으며 체취 관리에 관심이 많은 타겟으로 설정
#### 검색량 분석: 썸트렌드
중요성
썸트렌드는 소비자의 실제 의견과 감정을 객관적으로 파악할 수 있는 유용한 도구입니다. 특히 긍정어와 부정어 분석을 통해 고객이 제품이나 브랜드에 느끼는 만족 요소 및 불만 요소를 정확하게 이해할 수 있으며, 이는 마케팅 전략 개선과 콘텐츠 기획의 중요한 자료로 활용됩니다.
프로세스
주요 키워드 설정
제품 또는 브랜드의 핵심 키워드를 썸트렌드 검색창에 입력
연관 키워드, 상품명, 경쟁사 제품 키워드까지 폭넓게 설정
예) 바디워시, 바디워시 추천, 남자 바디워시, 남성 바디워시, 경쟁사 바디워시 이름 등
키워드 기반 긍정어·부정어 분석
썸트렌드의 "연관어 분석" 및 "감성 분석" 기능 사용
소비자가 자주 언급하는 긍정어와 부정어를 리스트업
긍정어: 고객 만족 포인트 (예: 향 좋다, 부드럽다, 촉촉하다 등)
부정어: 고객의 페인 포인트 (예: 자극적이다, 끈적이다, 향이 약하다 등)
분석 결과 기반 고객 니즈 정리
긍정어·부정어를 통해 고객의 핵심 요구사항과 불편 사항 도출
긍정어를 기반으로 마케팅 전략 및 콘텐츠를 강화할 요소 발굴
부정어를 기반으로 개선점 및 경쟁 우위 요소 발굴
유의사항 & TIP
긍정어나 부정어가 맥락 없이 나타날 수 있으므로, 단어가 사용된 맥락을 반드시 확인하여 분석
정기적으로 분석을 반복하여 소비자의 인식 변화를 추적 및 대응
데이터 샘플이 클수록 더욱 신뢰할 수 있는 분석 결과 도출 가능
예시
썸트렌드 키워드 분석 결과
긍정어: "향 좋다", "잔향이 오래간다", "부드럽다", "거품 풍성"
부정어: "향이 금방 사라진다", "피부 자극", "미끄럽다"
고객 니즈 파악 결과
향의 지속력이 고객의 주요 니즈로 나타남
피부 자극이 적고 촉촉하며 부드러운 사용감에 대한 선호가 높음
향 지속력 및 저자극성을 강조한 마케팅 콘텐츠 제작 필요
#### AI 활용 근거 자료 조사
중요성

목적 및 조사 키워드 정의
조사할 근거 자료의 목적 명확화 (예: 고객 니즈, 제품 차별성, 경쟁사 분석 등)

GPT 심층리서치 기능으로 검색기능을 열어 할루시네이션 최소화 후 시장조사
AI 분석 결과를 통해 근거 자료 확보 및 정리
AI 분석 결과를 활용해 다음의 자료를 도출:
고객의 주요 긍정·부정 키워드 (고객 만족 포인트와 불만 포인트)

유의사항 & TIP

AI 분석 결과를 지속적으로 업데이트하고 추적하여 최신 소비자 트렌드를 반영

어떤 경쟁사를 발굴하고, 어떤 지표(검색량, SNS 반응 등)를 기준으로 '잘되는' 사례를 수집할지 결정
경쟁사/사례 후보 탐색:
AI 검색(퍼플렉시티, ChatGPT 등) + 네이버 쇼핑 + 구글/메타 광고 등 복수 경로 활용
검색량·반응 지표로 상위 후보 추리기:
예) 최근 2~3개월 간 브랜드 검색량 증가율, SNS 좋아요·댓글수 등
실제 광고·콘텐츠 분석:
광고 소재(이미지·영상·카피), 랜딩페이지 구조, 바이럴/커뮤니티 운영 등 세부 항목 살펴보기
인사이트 정리 & 적용:
잘되는 소재/패턴을 모방 & 변주 (콘텐츠 기획, 랜딩, 광고세팅 등)
경쟁사·참고 쇼핑몰 파악 방법
방법 1: 퍼플렉시티 등 AI 툴 이용
AI에게 '카테고리+광고방식+기업 규모' 등으로 탐색
예) "국내 '바디워시' 판매 기업 중 최근 3개월 검색량이 급증한 브랜드 5곳 찾아줘."
검색된 브랜드별 '검색량', 최근 증가 추이 확인
검색량이 1만 이상 or 최근 2~3개월간 +XXX% 급증
인스타그램·구글 등에서 광고 확인
관심사 설정(앞서 AI가 언급한 키워드 클릭), 광고 노출 → '좋아요·댓글·공유' 많은 콘텐츠 수집
자사몰·SNS 검증
'왜 성장했을까?' → 내부 페이지(권위·근거요소, 후기, 이벤트 등) 살펴보기
증가 시점과 메인 광고 소재/캠페인 연결 짓기
팁: 퍼플렉시티는 "검색 시 프롬프트"를 자세히 쓰는 게 중요.
 예) "한국 내 중소기업 규모, D2C로 운영, 6개월 새 검색량 2배 이상 증가, 화장품·바디케어 카테고리"
방법 2: 네이버 쇼핑에서 직접 검색
핵심 키워드 입력
예) "미백크림", "바디워시", "헤어토닉"
파워링크(상단 광고) 노출 브랜드명 메모
이 브랜드명으로 다시 브랜드 검색량 파악 (네이버 키워드도구 이용)
최근 1~2개월 '급상승' 여부 체크
쇼핑탭 → 브랜드명 찾아보기
판매량·리뷰수·별점 등으로 상품 순위 대략 확인
"이 브랜드 최근 2달 만에 판매량/리뷰 폭증" → 광고 소재·바이럴이 잘 돌아가는 것으로 추정
팁: "검색량 1만 이상 & 급증"은 보통 최근 공격적인 광고 or 콘텐츠가 성과 냈다는 시그널.
방법 3: 소셜미디어 계정별 관심사 학습
메타(페이스북·인스타), 유튜브, 구글에 각각 '테스트용' 계정 or 프로필별로 관심사 유도
예) A프로필: 화장품·뷰티 관련 광고 주로 클릭, B프로필: 헬스·다이어트 광고 주로 클릭
실제 광고 노출 지켜보기
좋아요·댓글·공유 많은 콘텐츠 or 랜딩 클릭 → 내부 흐름(가격·후기·구매방법) 등 파악
광고주행동 추적
"이 브랜드 인스타 광고→랜딩→구매 프로세스" 실제 체험하면서 픽셀에 내 정보 담기도록 행동
DM/카카오문의, 후기작성 이벤트, 후속CRM(문자, 알림톡) 등도 체크
브랜드 검색량이 확 증가했거나 주목받고 있다면
그 시점 광고·이벤트·콘텐츠를 수집 → 무엇이 차별화 포인트인지 확인
중요: 라이브러리(페이스북광고 라이브러리 등)만 보면 좋아요·댓글수가 표시 안 되므로 실제 노출 광고를 유저 입장에서 클릭해 반응 지표를 확인해야 합니다.
##### 필수 조사 항목
브랜드 검색량
네이버 키워드 도구, 구글 트렌드, 퍼플렉시티
월 평균 1만 이상 & 최근 증가 폭(예: +50%↑)
SNS 반응 지표
좋아요·댓글·공유 수
댓글 내용(실제 구매·관심이 있는지, 이벤트성인지)
랜딩 페이지
권위 강조, 임상·수치, 고객후기, 이벤트, 희소성, 할인 등 설득 요소가 어느 정도로 배치되어있는지?
광고 소재 특징
(이미지) 사람 얼굴(모델) vs 제품 컷 비율, (영상) 길이·구성, (카피) 숫자·임팩트 표현 등
바이럴 채널
블로그, 카페, 유튜브, 틱톡 등에서 어떤 메시지·후기를 주로 띄우는지?
##### 케이스 수집 방법
N회 이상 기준은 보통 월 1만 or 5천 등, 업계 규모 따라 다름
최근 1~2개월 새 검색량 급등 & SNS 반응도 높은 브랜드 중 (예) 상위 3곳 고르기
수집할 항목:
메인 광고 소재 (캡처·링크 보관, 어떤 카피/비주얼인지)
좋아요·댓글수 통한 실제 성과 나오는 광고 콘텐츠 녹화, 캡쳐
랜딩 페이지 구조 (스크린샷, 핵심 메시지)
바이럴 채널 (블로그/유튜브 등 주된 점유 채널, 컨텐츠 포맷)
가격·이벤트 정책 (한정수량, 타임세일, 증정품 등)
실제 예: "A브랜드는 1~2월 검색량 2천→3월 1만으로 폭증, 인스타 광고 영상 '역대급 할인' 캠페인에 좋아요 5천개, 댓글 300개 달림. 랜딩페이지에서 '30명 한정 50% 특가' 호소력이 가장 컸음."
##### 인사이트 도출 방법
강조 요소 vs 우리의 상품
경쟁사 광고에 "임상 93% 개선"을 제일 세게 넣음 → 우리도 숫자적 근거가 있나?
만약 없다면, 근거 보강(신뢰자료·연구결과) or 다른 설득 포인트 모색
랜딩/콘텐츠 구조 비교
경쟁사: "문제제기→해결책→권위자료→한정수량→리뷰" 구조
우리: 권위자료·숫자적 증거 없는데도 장황한 설명만? → 즉시 개선 필요
SNS 댓글 분위기
'가격 대비 품질 최고', '신뢰하는 셀럽 협찬', '이번 달만 파격 할인이 크다' 등 구매 동기 파악
비교표 작성
경쟁사1~3, 브랜드 검색량, SNS 반응도, 랜딩 페이지 주요 강점
옆 칸에는 "우리 제품 적용 방안" 기입
#### 실무 적용 프로세스
경쟁사 상품 직접 구매
실제 구매 프로세스·고객 응대·리뷰 작성 과정 체험 → 사후 이메일, 카톡, DM까지
어떤 CRM을 어떻게 보내는지 모니터링
이벤트나 할인율 추이
시점별 할인폭이 달라지는지, 어떤 식으로 긴박감을 주는지 기록
소셜 리스닝
네이버 카페, 맘카페, 주부커뮤니티 등에서 "경쟁사 브랜드명 + 후기" 검색 → 단순 광고글인지, 자발적 후기인지 구분
시점 파악
"언제부터" 검색량이 오르고, 어떤 광고 소재가 당시 노출됐나 → 성과 배경 추적
우리 제품·브랜드와의 차별점
경쟁사 카피("임상 95%") vs 우리 ("임상 80%" or 없음) → 차별이 어려우면 다른 설득 방안 고민
#### 결론
경쟁사나 잘되는 브랜드는 대부분 최근 2~3개월간 검색량, SNS 반응이 급상승했고, 광고 소재나 랜딩페이지에 분명한 USP(차별 포인트)와 설득장치(숫자·권위·할인·희소성 등)가 명확히 존재합니다.
단순히 페이스북 라이브러리로 광고 크리에이티브만 보는 것보다는, '좋아요·댓글수'가 많은 실제 노출 광고, 그리고 랜딩 페이지·바이럴 활동을 종합적으로 체크해야 진짜 잘되는 사례인지 식별 가능합니다.
'모방 + 변주'를 통해 우리 상품도 유사한 구조(문제제기→해결→근거→이벤트)를 빠르게 적용하고, 실행 후 지표를 확인 → 성과가 올라가면 더욱 강화하는 식으로 개선 사이클을 돌리는 게 핵심입니다.
검색량 & SNS 반응(좋아요·댓글수 등) 두 가지 지표로 성장세가 있는 경쟁사를 선별 → 광고 콘텐츠·랜딩·바이럴 3박자를 모니터링 → 인사이트를 '모방·커스터마이징' 해서 우리 마케팅 전략에 즉시 반영.
 이렇게 해야 시간·자원 낭비 없이 성공 가능성이 높은 방법을 빠르게 학습·실행할 수 있습니다.
### 콘텐츠 제작 전략
#### 광고 콘텐츠 기획 방법
##### 카피
무의식, 인식, 설득 구조를 활용한 카피 작성
고객의 문화적 배경과 언어 표현 방식 반영
A/B 테스트를 위한 여러 버전의 카피 준비 (2-3가지 변수로 통제)
##### 이미지
포인트 찾아서 기획
착한구두 뒷굽 찾은 것처럼 예시 활용
시장조사 결과나 무의식 & 의식 요소 확인
화장품은 예쁜 모델을 쓰면 성과가 잘 나옴을 고려, USP를 살려서 적용 시 CTR 상승 가능성 검증
A/B 테스트 이미지 기획
한 변수만 통제하여 비교 (예: 모델 vs 제품 컷, 배경색 변화, 문구 위치 등)
명확한 가설 설정 (예: "45도 각도의 제품 이미지가 정면 이미지보다 CTR이 높을 것이다")
##### 영상
스크립트 기획
5초 내 후킹 요소 배치
문제제기 → 해결 → 근거 → 행동유도 구조 활용
목표 퍼널 단계에 맞는 메시지 집중 (인지 vs 고려 vs 전환)
커머스 편집 기술
음향: 배경음악, 효과음 전략적 활용
자막: 핵심 메시지 강조 및 무음 재생 시 이해 가능하도록 설계
컷 편집: 템포와 리듬감, 주목도 높은 장면 배치
#### 프로세스
##### 가설 수립
명확한 변수 설정 (1-2개로 제한)
측정 가능한 KPI 선정 (CTR, CVR, 체류시간 등)
예상 결과 문서화
##### 실험 설계
충분한 샘플 확보를 위한 예산 설정
컨트롤 그룹과 실험 그룹 구성
실험 기간 및 데이터 수집 방법 결정
##### 실험 및 모니터링
일일 단위 데이터 추적
이상치 발생 시 원인 분석
중간 결과에 따른 실험 계속 여부 판단
##### 분석 및 적용
통계적 유의미성 확인
성공/실패 원인 분석
학습 내용 문서화 및 다음 실험 계획 수립
성공한 방법론 확대 적용
## 부록: 주요 마케팅 용어 정리
CCFM 마케팅사업부에서 쓰는 필수 마케팅 용어를 외우고 이해하면 소통이 더 빠르고 원활해집니다.

| 용어 | 설명 |
|---|---|
| KPI (Key Performance Indicator) | 핵심 성과 지표. 조직·프로젝트의 목표 달성도를 측정하기 위한 지표. 예) 매출액, 전환율 등 |
| ROI (Return On Investment) | 투자 수익률. 투자비 대비 얼마만큼 이익을 얻었는지 나타내는 지표 |
| ROAS (Return On Ad Spend) | 광고비 대비 매출. 광고비 1원당 발생 매출을 % |
| CPC (Cost Per Click) | 클릭당 비용. 광고가 1번 클릭될 때마다 지불하는 금액 |
| CPM (Cost Per Mille) | 1000회 노출당 비용. 1000번 노출될 때마다 지불하는 금액 |
| CTR (Click Through Rate) | 노출 대비 클릭률. 노출 수 중 몇 %가 클릭으로 이어졌는지 |
| CPA (Cost Per Action/Acquisition) | 최종 전환 1건당 비용. 구매·회원가입 등 목표 액션당 지불금액 |
| CAC (Customer Acquisition Cost) | 고객 1명을 획득하는 데 드는 비용. 광고 등 모든 비용 포함 |
| LTV / CLV (LifeTime Value / Customer Lifetime Value) | 고객 생애가치. 한 고객이 우리 제품·서비스에 향후 지불하게 될 총수익 추정 |
| A/B Test | 두 가지 버전을 비교 테스트. 예) 다른 카피·배너·버튼 색상 등 전환율 비교 |
| Landing Page (랜딩페이지) | 광고 클릭 후 도착하는 1차 페이지. 전환 설계가 핵심 |
| Conversion (전환) | 광고나 웹사이트가 유도하는 최종행동. 예) 구매·회원가입·문의 등 |
| CRM (Customer Relationship Management) | 고객관계관리. 고객정보·상호작용 등을 체계화해 만족도·재구매율 높임 |
| CTA (Call To Action) | 행동 유도 문구·버튼. 예) "지금 구매하기", "상담받기" |
| USP (Unique Selling Proposition) | 제품·서비스만의 '차별화된 강점'. 고객이 우리를 선택할 이유 |
| UTM (Urchin Tracking Module) | 추적 파라미터. URL 뒤에 붙여서 광고 출처·매체·캠페인 식별 |
| Funnel (퍼널) | 고객이 관심→구매까지 흐름을 깔때기로 표현. 단계별 이탈률 확인 |
| Frequency (빈도수) | 동일 광고를 1인당 몇 번 노출했는지 나타내는 수치 |
| Reach (도달) | 광고·콘텐츠가 도달한 '고유 사용자 수' |
| Impression (노출) | 광고·콘텐츠가 화면에 표시된 총 횟수 |
| Demographics (인구통계) | 연령·성별·지역·직업 등 인구 통계학적 속성 |
| Retargeting (리타겟팅) | 방문 이력 있는 고객에게 반복 광고. 미방문자보다 전환율이 높음 |
| Lead (리드) | 잠재고객. 구매 가능성이 있는 고객 정보를 얻은 상태 |
| Conversion Rate (전환율) | 유입 대비 전환 성취 비율. 방문자 중 몇 %가 구매·문의 등 액션? |
| Engagement Rate | SNS·커뮤니티 등에서 참여도. 좋아요·댓글·공유 등 반응 비율 |
| Growth Hacking | 창의적이고 민첩한 방법으로 빠른 성장을 추구하는 마케팅 기법 |
| Segment (세분시장) | 유사 구매패턴·특성으로 묶은 고객군 |
| Positioning (포지셔닝) | 시장·소비자 머릿속에서 우리 브랜드가 차지하는 자리 |
| Persona (페르소나) | 타겟 고객을 대표하는 가상의 인물. 성별·나이·직업·취미 등 구체화 |
| Upsell 업셀 | 고객이 이미 구매한 상품보다 상위·고가 상품을 추가 구매 유도 |
| Cross-sell 크로스 셀 | 관련 상품을 함께 구매하도록 유도. 예) 신발→양말 추천 |
| SEO (Search Engine Optimization) | 검색엔진최적화. 자연 검색결과 상위 노출을 위해 콘텐츠·사이트 구조 최적화 |
| Organic Traffic | 검색엔진·SNS 등에서 '광고비 없이' 자연 유입되는 방문자 |
| Paid Traffic | 광고비를 투입하여 확보한 유입. CPC·CPM 등 지불 기반 |
| Keyword | 사용자가 검색하는 핵심 단어. SEO·검색광고 기초 |
| Earned Media | 언론 기사·바이럴·SNS 공유 등 '자발적 홍보 채널' |
| Owned Media | 자사 웹사이트·블로그·SNS 등 '직접 소유·관리' 채널 |
| Paid Media | 유료 광고 채널. TV·신문·온라인 배너광고 등 |
| SKU (Stock Keeping Unit) | 재고관리 단위. 상품식별 코드로 물류·재고 추적 |
| Remarketing | 리타겟팅과 유사. 방문·행동 이력 있는 고객에게 맞춤 광고 |
| Link Click vs. All Click | Link Click은 외부 사이트 이동 목적, All Click은 글 펼치기·페이지 이동 등 전체 클릭 |
| Outbound Click vs. On-Facebook Click | Outbound Click은 플랫폼 외 URL로 이동, On-Facebook Click은 내부 페이지로 이동 |
| Unique CTR (Link) | 고유 링크 클릭 ÷ 노출 × 100. (같은 사람이 2회 이상 클릭해도 1명 처리) |
| Cost per Unique Click | (광고비) / (Unique Click). 1명의 '최초 클릭'당 비용 |
| CPC (All) | 전체 클릭(좋아요·댓글 포함) 대비 비용 |
| CPC (Link) | 링크 클릭만 대상. Link Click / 광고비로 계산 |
| CPC (Outbound) | 플랫폼 외 사이트로 이동시키는 클릭당 비용 |
| View Content | 상세 페이지 열람 같은 콘텐츠 뷰. 전환 전 '관심 지표'로 사용 |
| CPC (View Content) | View Content 1건당 비용. 광고주가 '상품 상세조회'만큼 중요히 볼 때 |
| 2-Second Continuous Video Plays | 동영상을 2초 연속 시청한 횟수 (메타 광고 세부지표 중 하나) |
| Cost per 2-Second Video Play | 2초 연속 재생 1회당 광고비 |
| ThruPlay | 15초 이내 영상의 경우 끝까지 시청, 15초 넘는 영상은 15초 이상 시청으로 측정 |
| Cost per ThruPlay | 스킵 없이(또는 15초 이상) 재생된 뷰 한 건당 비용 |
| Video Percentage Watched | 영상을 몇 %까지 시청했는지 구간별 분석 (25%, 50%, 75%, 95%, 100%) |
| Cost per 25% View 등 | 각 시청구간별 '1회 시청 비용' |
| 광고 상기도 | 광고를 본 뒤, 광고·브랜드 기억이 어느 정도 상승했는지 추정치 (페북브랜드리프트) |
| 스펜딩 | 마케팅·광고비: 특정 채널(페이스북, 구글, 네이버 등)에 집행한 실제 지출 비용 |
| 간접전환, 간접전환 ROAS | 메타, 구글, GFA 와 같은 플랫폼에서 제공하는 매출과 ROAS 지표 |
| 직접전환, 직접전환 ROAS | GA, 카페24 접속통계 와 같은 애널리틱스 분석 툴에 UE, UTM 태그 값의 매출 기준으로 분석 하는 매출과 ROAS |
