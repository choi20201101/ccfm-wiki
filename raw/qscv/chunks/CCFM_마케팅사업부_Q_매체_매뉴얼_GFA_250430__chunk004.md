캠페인: 광고의 목적
광고의 목적을 설정하는 가장 상위 구조
대표 URL
전환 추적 대상
최적화 전환 유형
목표: 전환 / 트래픽 / 쇼핑 / 카탈로그 / 영상 등
광고 그룹: 겟팅, 지면, 소재 운영, 예산, 입찰 전략 방식 설정
타겟팅 설정과 지면, 운영 단위 설정을 담당
타겟 설정
오디언스, 컨텍스트, 상세 타겟(성별/연령/관심사/구매의도), 맞춤타겟, 잠재고객 등
타겟 전략 예시
구매의도 중심 타겟 + 맞춤 키워드 조합
리타겟 + 지역 + 고효율 지면 묶음
지면 설정
모바일 메인 / 서브 / 피드 / 스마트채널 / PC / 인스트림 등
소재 선택 방식
최적화 / 성과 가중 / 균등
예산 설정: 일 예산 or 총 예산
입찰 전략: 입찰가 한도 / 비용 한도 / 입찰가 한도 없음
광고 소재: 실제 유저에게 노출되는 이미지&영상
실제 유저에게 노출되는 광고 이미지/영상 및 문구
형식: 이미지 / 영상(숏폼, 인스트림) / 피드형
문구 작성: 타이틀 + 설명 + CTA 구성
검수: 사람 수기 검수로 진행 → 기준 유동적 (검수 팁 따로 존재)
테스트: 타겟×소재 조합으로 CTR/전환/ROAS 테스트 필수
캠페인 생성
목표 설정 (전환/트래픽 등)
광고 그룹 설정
타겟팅 + 지면 + 소재 선택 방식 설정
예산 및 입찰 방식 선택
소재 등록 및 검수
이미지 or 영상 등록 → 문구 입력 → 검수 진행
운영 & 테스트
CTR, 전환율 기반 소재 교체, 타겟 확장 or 디벨롭진행
### 예산 설정 및 입찰 전략
CPC 설정
그룹 예산 10억원으로 설정 후 CPC를 50원 단위로 계속 증액하며 1분~10분 단위로 체크합니다
노출량이 늘어나는 구간에서 유지 및 감액
그룹 예산을 5만 원 이하로 ‘입찰가 한도 없음’ 설정한 후 1분~10분 단위로 체크합니다
과금되는 CPC 체크하며 기록된 CPC의 절반 수준으로 입찰가 설정 → 5분~10분 단위로 증액 후 체크
고정 일예산 방식 + 반응 시 조정
예산 예: 50만원 고정 → CPC 점진 증액 → 소진 시작되면 노출량 체크하며 운영
그룹 예산을 10만원으로 설정 후 CPC를 50원 단위로 증액하며 1분~10분 단위로 체크합니다
반응 확인 시 그룹 예산을 5만원~10만원 씩 올리며 운영합니다
캠페인 예산을 따로 설정 후 그룹에서 자유롭게 운영합니다
그룹 예산을 10억으로 설정해도 캠페인 예산이 걸려있으면 설정된 금액을 넘기지 못합니다
입찰 전략은 정답이 아니라 조합 및 타이트한 모니터링입니다
예산, 타겟, 모수, 목표 성과에 따라 조절 가능한 여러 플랜을 준비해 두는 것이 사고를 줄이고 성과를 만들 수 있습니다
트래픽 캠페인
그룹 예산 10억 설정 후 수동 입찰 선택 기준 운영된 데이터가 없어 광고 라이브가 처음일 경우, 유사 업종 데이터를 참고하여 설정이 필요함
모수에 따라 입찰가는 변동하기 때문에 업종별 데이터보다 낮게 설정하여 운영
데이터가 없는 초기 계정일 경우
예시_ 뷰티 업종 업종별 CPC 메인 800~900원 서브 CPC 500~600원 → 설정 캠페인 메인 600원~700원 서브 300~400원
운영했던 데이터가 있거나 운영중인 계정일 경우
예시_실시간으로 입찰 경쟁이 달라지는 매체이기에 최근 14일 / 7일 기준과 당일 데이터를 참고하여 평균 CPC보다 30~50원 낮춰 세팅 진행
전환 캠페인
전환 캠페인은 수동 입찰이 아닌, 자동 입찰 학습 기반의 캠페인이기 때문에 입찰가 조정이 아닌 ‘광고그룹 예산’ 조정을 통해 학습을 유도하고 성과를 확보하는 것이 핵심입니다
초기 예산 3~5만원으로 테스트 진행 1~3일 정도 반응 체크하면서 증액 운영해야 합니다
예산 증액은 효율에 따라 유동적으로 운영하되 소진 금액 대비 전환 수, CPA, CPC 등을 기준으로 판단하는 것도 중요합니다
예시1
광고주 목표 ROAS 200%일 경우에 대한 예시
전환캠페인 예산 5만 원 설정 후 소진 2만 원 / 매출 7만원 → ROAS 350% 확인 시 예산 5만원 → 예산 10만원으로 증액 운영
추가 전환이 확인될 경우 예산 10만원 소진 후 매출 23만원 → ROAS 230% 확인 시 예산 10만원 → 예산 20만원 설정 후 효율 체크
5만원 → 10만원 → 20만원 → 50만원으로 증액하여 효율 체크가 필요합니다
 (예산 조정은 유동적으로 운영하지만 큰 폭으로 예산을 증액하면서 체크하는 게 좋음)
예산 5만 원 내 전환 5건 이상 + 목표 CPA n만 원 이하인 경우 → 10만 원 증액 고려 가능합니다
예산 10만 원 내 목표 ROAS n% 이상 or 평균 CPC 대비 많이 낮을 경우 → 증액 고려 가능합니다
단 5만 원 → 50만 원처럼 급격한 예산 증액은 학습 및 성과에 영향이 생길 가능성 있음 → 단계별 점진적 증액을 권장합니다
광고주의 전체 예산 규모가 크고,트래픽 캠페인에서 베스트 콘텐츠가 사전에 확보된 상황이라면 초기부터 30만 원 이상의 예산을 세팅하여 빠르게 학습을 유도하고, 성과 기반으로 빠르게 확장하는 전략도 가능합니다
트래픽 캠페인을 통한 고성과 베스트 컨텐츠 확보된 상태
전환 캠페인에서 학습속도 빠르게 만들기 위해 고예산 시작 가능
광고주 월 예산 여유 있음
학습 비용 감내 가능 → 저예산부터 천천히보단 빠른 테스트 목적으로 운영
세일/런칭 등 일정 압박 있음
빠른 성과 확보가 필요한 경우
전환 캠페인의 예산 설정은 정해진 정답이 아닌 “전체 예산 규모, 사전 확보된 베스트 소재의 유무 등
빠르게 확장할 수 있는 여건을 고려하여 유동적으로 운영하는 것이 효율적입니다
전환캠페인 학습 관련 설명
운영 가능
학습 시작 이후 7일 내 40회 이상의 전환수를 획득한 상태입니다.
운영 가능 (학습중)
신규 광고 그룹을 생성하거나, 학습 성능에 영향을 미치는 중대한 변경 사항이 발생할 경우, ‘학습중’ 상태가 됩니다.
학습중 상태가 반복되면 성과와 비용 효율성에 영향을 줄 수 있으며, 학습이 초기화되면 광고 최적화에 더 많은 시간이 소요될 수 있습니다
※ 학습에 영향을 미치는 중대 변경 사항
타겟팅 변경 사항 발생
그룹 내 소재의 신규 승인이 발생한 경우 (신규 소재 추가, 기존 소재 변경 등)
예산 또는 입찰가 등의 대폭 변경 (20~30% 이상 변경)
비용 관리 옵션 변경
48시간 이상 광고 그룹 집행 중단 (OFF)
[IMG]
운영 가능 (학습제한)
학습 시작 이후 7일 내 전환수(학습건수 30회) 미만의 전환수를 획득했거나, 48시간 이상 광고 그룹을 일시 중단한 경우 ‘학습 제한’ 상태가 됩니다.
학습 제한 상태에서는 효율적인 가격으로 노출이 어렵거나, 예산 소진이 원활하지 않을 수 있습니다.
단, 학습 제한 상태에서 일정 수 이상의 전환수를 획득할 경우, 다시 ‘학습중’ 상태로 변경 가능합니다.
상태 변경 기준
광고 그룹 당 1주 내 50회에서, 40회로 학습에 필요한 전환 수 기준을 변경하였습니다.
재학습(학습 요구량이 충족되었다가, 중대 변경으로 인해 다시 학습 되는 경우)의 경우 1주 내 30회의 전환이 필요합니다.
최초 학습, 또는 재학습 기간 중, 요구 학습량을 만족시키지 못하거나, 광고그룹을 48시간 이상 OFF 할 경우, 학습 제한 상태로 변경됩니다. 학습 제한 상태에서 1주 내 20회의 전환 수를 획득하면 '학습중' 상태로 변경됩니다.
학습 제한 이후, 학습중 상태로 변경된 경우, 학습 완료 이력과 관계 없이 1주 내 40회의 전환이 필요합니다
## 성과 측정 및 분석
개요: 성과 데이터 해석 및 광고 효율 진단 능력 함양
### 성과 지표 설명
캠페인 정보
캠페인 목적: 해당 캠페인이 설정된 광고 목표 (전환, 트래픽, 쇼핑 등)
캠페인 예산: 설정된 캠페인 전체 예산 금액
목적에 따라 성과 해석 지표가 달라짐
기본 성과 지표: 광고의 노출–클릭–비용 흐름을 확인할 수 있는 1차 지표
총 비용: 해당 항목에 집행된 광고비 총합
노출: 광고가 실제로 보여진 횟수
클릭: 광고를 클릭한 횟수
CTR(클릭률): 클릭 ÷ 노출, 반응도 확인
CPC(클릭당 비용: 총 비용 ÷ 클릭 수
결과: 모든 성과 지표 수치 (예: 전환 수, 장바구니 등)
결과당 비용: 결과 1건당 평균 소요 비용
CPM: 1,000회 노출당 비용 (노출 효율 파악용) / 광고비 ÷ 노출 수 × 1,000
기본 성과 지표는 전체 체크하여 확인 필요
전환 지표: 실제 액션(구매, 장바구니 등) 기반의 성과 판단 핵심 지표
구매완료수: 실질적인 성과 중심 지표
장바구니 담기수: 광고 클릭 후 상품을 장바구니에 담은 횟수
구매완료 전환 매출액: 실제 구매가 완료된 수치를 기반으로 집계된 매출 총액
장바구니 전환 매출액: 장바구니에 담긴 상품 금액의 총합 (예상 매출)
구매완료 광고 수익률: ROAS (구매완료 매출액 ÷ 광고비) × 100 / 핵심 효율 지표
GFA는 총 매출액, 총 전환율 등 “총”이 들어가면 장바구니 매출액까지 합산하기 때문에
지표를 구분해서 확인해야 합니다
재생 지표: 영상 캠페인 시 필수 확인 지표
총 재생수: 영상이 실제로 재생된 총 횟수
CPV: 동영상 재생당 평균 비용 / 총 광고비 ÷ 총 재생 수
재생 구간별 수치: 25%, 50%, 75%, 100% 재생율
자동재생 / 클릭재생: 영상 시작 방식 구분
지표는 많지만 목적에 맞게 고르는 게 핵심입니다
### 광고 매체 어드민 활용 방법
캠페인 운영 상태 확인
캠페인, 광고 그룹, 소재 별 상태와 예산 소진 현황을 실시간으로 확인할 수 있습니다
운영 상태 확인: 집행 중 / 종료 / 검수 증 / 반려 상태를 통해 광고 운영 여부 확인
예산 소진 체크: 설정 예산 대비 실제 집행 금액 및 소진 속도 확인
성과 지표 모니터링: 클릭 수, 결과 수, 전환 수 등 KPI 기반 실시간 분석 가능
실시간 확인을 통해 비효율 캠페인에 빠르게 대응하며 운영할 수 있습니다
지표 열 맞춤 설정
어드민에서 지표 항목을 선택해 목적에 맞게 데이터를 정리할 수 있습니다
캠페인 목적별 맞춤 설정: 구매완료수, 전환 매출, ROAS, CTR, 클릭 수, CPC 등
필요한 지표만 설정하여 가독성 향상
보고서 추출 전 원하는 형태로 구성 가능
지표 선택은 성과 해석의 핵심이기에 목적에 따라 꼭 필요한 지표만 골라서 분석해야 합니다
캠페인 / 광고그룹 / 소재 복사 및 편집
어드민에서 기존 캠페인이나 광고그룹, 소재를 손쉽게 복사하거나
여러 항목을 수정할 수 있고  빠른 실험과 운영 효율화가 가능한 곳 입니다
다양한 타겟-소재 조합 테스트, 빠른 스케일링, 유사 구조 반복 운영에 있습니다
검수 진행 상태 및 반려 사유 확인
소재 등록 후 검수가 진행되며, 검수 현황과 결과를 어드민에서 직접 확인할 수 있습니다
검수 상태 확인: 소재별로 승인 / 반려 여부 확인
반려 사유 확인 및 수정 대응: 검수 의견을 확인하고, 소재 문구나 이미지를 수정하여 재검수 가능
검수는 사람에 의해 이뤄지기 때문에 유동적인 기준이 존재합니다. 소재 순화 및 반복 시도도 전략입니다
리포트 다운로드 및 분석
어드민에서는 캠페인별, 날짜별, 타겟별 성과 리포트를 엑셀 형식으로 다운로드할 수 있습니다
리포트 범위 설정: 날짜, 캠페인, 그룹, 소재 단위 설정
지표 기준 선택: 열 맞춤 설정대로 추출
활용 예시: 주간 보고서 / 광고주 리포트 / 타겟별 성과 비교 / 효율 분석
리포트를 정제하여 광고주 소통 자료, 전략 수정 근거로 바로 활용할 수 있습니다.
### 보고서 생성 및 해석
보고서 생성 방법
GFA 어드민에서는 원하는 조건으로 보고서를 다운로드해 데이터를 분석할 수 있습니다
기간 선택: 일/주/월별 구간
범위 설정: 캠페인 / 광고그룹 / 소재 단위
지표 열 맞춤: 광고 목적에 맞는 핵심 지표만 선택
파일 다운로드: 엑셀 형태 추출 → 가공 및 분석 가능
※ 지표 열 맞춤 설정은 목적별로 다른 항목을 보는 것이 중요합니다
전환 퍼널 기반 분석 구성 필요
GFA는 상품 중심의 구매 유도 퍼널로 아래 세 단계로 나눌 수 있습니다
노출 → 클릭: 관심 유도 단계
목표: 관심 타겟이 반응하도록 유도
지표:
노출: 광고가 보여진 횟수
클릭: 광고를 누른 횟수
CTR: 클릭률
CPC: 클릭당 비용
CPM: 1,000회 노출당 비용
CTR이 낮다면 소재 메시지, 이미지, 지면 매칭 등을 점검이 필요합니다
CPC가 높을 경우 입찰 경쟁 높아짐과 타겟 체크가 필요합니다(CTR 낮을 경우도 체크)
CTR이 높지만 구매 없는 경우 → 랜딩, 구매 여정, CS 등 체크가 필요합니다
클릭 → 행동: 의도 확인 단계
목표: 구매 고려 유저를 식별
지표:
장바구니 담기 수
장바구니 전환 매출액
클릭은 많은데 장바구니 담기수가 적다면 → 상품 정보 혹은 랜딩 UX 개선 필요합니다
특정 지면/소재 장바구니 전환율 높다면 → 고효율 타겟 가능성 높음 → 확장 테스트 진행이 필요합니다
장바구니 → 구매완료 (전환/수익화 단계)
목표: 실 구매 발생 및 효율 분석
지표:
구매 완료수
구매완료 전환 매출액
ROAS
ROAS가 낮으면 광고비 과소진 혹은 객단가 낮은 상품 비중 확인 필요합니다
장바구니 대비 구매 이탈이 크다면 → 결제 흐름, 신뢰도, 재방문 동선을 점검해야 합니다
