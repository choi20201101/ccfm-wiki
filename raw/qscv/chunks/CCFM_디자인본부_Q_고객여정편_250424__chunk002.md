퍼포먼스 캔버스 = “단순 문서”가 아닌, 실전 지침서
단순한 사내 보고용 문서가 아닌, 실질적으로 광고에서 소비자에게 전달할 메시지를 설계하는 지침서입니다.
광고주와의 공유는 필요에 따라 일부 발췌 가능합니다.(광고 전략, 페르소나, 핵심 메시지 등)
내부 실무에서는 제작, 집행, 리포트, 피드백 모든 단계에서 캔버스를 기준으로 판단합니다.
작성 후 A/B 테스트 및 실행 결과 반영 필수
캔버스 작성 → 광고 제작 → 광고 집행 → 성과 확인 순서
광고 성과(CTR, CVR, CPA 등)를 확인 후 반드시 캔버스를 업데이트 해야합니다.
잘 먹히는 비쥬얼 요소, 소구점은 재확산 혹은 확대하여 적용 시킵니다.
성과가 낮은 요소는 원인 분석 → 개선 후 재테스트를 진행합니다.
퍼포먼스 캔버스는 일회성 작업이 아닌, 진화하는 전략맵으로 기능합니다.
퍼포먼스 캔버스는 “팀 내 공통 언어”
마케터와 디자이너, PM 모두가 “이 제품은 oo 때문에 고객이 구매할 것이다”를 동일한 수준으로 이해하기 위함입니다.
광고주 피드백, 실무 회의, 리포트 작성 시 퍼포먼스 캔버스 문구, 구조, 내용을 기준으로 대화합니다.
모든 소재, 랜딩페이지, 캠페인 기획 문서에도 캔버스 내용을 반드시 반영해야 합니다.
광고주와의 커뮤니케이션에도 활용
광고주와의 전략 미팅 또는 콘텐츠 설명 시 ex.
이번 전략의 oo USP을 바탕으로 oo 의도를 담아 제작했습니다.
이 캠페인 맞춤 oo 타겟과 oo Pain Point의 해결 방법을 oo 식으로 녹여 제작했습니다.

#### 레이아웃
각 구성요소를 제한된 공간 안에서 효과적으로 배열하는 프로세스
화면에 이미지나 텍스트를 배치할 때도 보이지 않는 ‘그리드’ 사용
그리드를 기준으로 조정하면 전체적으로 깔끔하고 정돈된 인상을 줄 수 있음
같은 정보끼리 그룹화가 되어있는지 확인
메인 / 서브 / CTA 등 각 구역별로 정돈 되어야, 보는 소비자 입장에서도 쉽게 정보 이해 가능
시선의 흐름대로 정보가 읽히는지 확인
큰 것부터 작은 것을 읽음
왼쪽에서 오른쪽으로 읽음
위에서 아래로 읽음
배치를 통해 핵심 메시지를 의도한 순서대로 전달하기 위해 흐름을 파악 필수
여백과 마진 체크
마진 : 콘텐츠를 감싸고 있는 상하좌우 여백을 뜻
마진이 넓을 수록 콘텐츠가 중앙에 몰리게 되니 주의
상하좌우를 동일한 마진으로 쓰는것이 기본적
마진을 어느 정도로 잡을지는 메세지와 콘텐츠 가이드의 비율을 통해 결정
문구와 문구사이에 충분한 여백이 있는지 확인
그룹화를 하였을 때 충분한 여백이 있어야 소비자가 편하게 메시지를 읽을 수 있음
#### 폰트
이미지와 마찬가지로 폰트 또한 소재 톤에 영향력이 큼
소재 속 폰트 통일화
명조, 고딕을 어설프게 활용하면 저하에 원인이 되므로 각진 고딕은 각직 폰트활용, 둥근 고딕은 둥근 폰트끼리 활용
필기와 디자인체는 별개로 메인 및 서브로 각 1문단 씩, 포인트를 줄 때 활용
카피와 어울리는 폰트 활용
어울리는 폰트를 사용하는 것은 소비자에게 메세지를 잘 전달하기 위함이고 가독성과 판독성을 위해 소재 하나 제작시 디테일을 체크하여 0.1초라도 시선을 머물수 있게 해야함
문구나 워딩에 어울리는 폰트를 사용하는 것 또한 소비자의 눈을 사로잡는 방법
ex. 포근포근 귀엽고 따뜻해 → 고딕x 둥근 디자인체o
ex. 무섭습니다 5일만에 기미가 녹다니 → 고딕x 명조 or 정적인 디자인체o
메인 카피 강조
정보 전달은 사람의 인식구조를 통해 이미지 → 메인카피 → 서브카피 → CTA 버튼으로 흘러감
전달하고자 하는 내용의 흐름을 소비자가 이해하기 쉽게 가독성과 판독석을 중요하게 생각해야함
자간: 글자와 글자 사이
행간: 글 줄 사이의 간격
각 그룹화를 시키며 폰트의 굵기 크기 등을 조절하여 강조 포인트를 확실하게 잡아야함

### 콘텐츠 형식
※ 자료첨부
콘텐츠 형식 교육자료
메인 후킹 포인트 강조
한눈에 제품/혜택이 들어오는 텍스트 구성하기
숫자, 강한 후킹으로 소비자의 ‘첫 시선’ 사로 잡기
사람의 시선 흐름은 좌상 → 우하로 떨어지기 때문 상단이나 좌측배치
뉴스/기사 형식
진짜 있던 기사인 것처럼 보여 신뢰도를 상승시킬 수 있음
과장된 내용으로 제작시 오히려 반감이 생길 수 있음
누끼컷 활용
광고하는 제품의 누끼를 활용하여 후킹을 주는 형식으로 제품의 특성을 잘 드러낸 누끼컷을 활용하는 것이 중요
리뷰 형식
실 사용자들의 리뷰를 풀어내어 신뢰를 얻을 수 있고 직접 촬영을 통해 이미지를 활용한다면 광고 거부감을 줄일 수 있음
AI 이미지 활용
광고주측 소스에 인물이 부족하여 빠르게 인물 테스트를 진행해야할 때 활용
착용 컷, 뷰티케어, 뷰티디바이스, 건강식품 등
예쁘고 잘생긴 이미지, 타겟별 페인포인트를 상황표현할 때 적극 활용 필수
일러스트 & 컷툰 형식
소재 내에 일러스트나 만화적인 요소를 넣어 소재를 더욱 다양하게 표현
강조, 재미, 후킹, 쉬운 이해도를 기대하여 요소 조합하기
B&A 형식
이미지를 전/후 로 배치하여 노골적이고 직관적으로 표현
GFA 매체에는 검수가이드에 활용이 불가하나 META나 GG등 매체에는 활용 추천
화장품이나 뷰티 디바이스 카테고리를 한다면 반응이 보이는 형식이니 필수 시도
게시판&커뮤니티 형식
캡쳐해온듯한 형식으로 유머글이나 정보글로 인식하여 광고 거부감 감소
비교적 글이 다른 소재형식의 비해 많으나 재미있는 요소와 실제 사례같은 느낌을 주어 흥미를 느낄 수 있도록 하기
