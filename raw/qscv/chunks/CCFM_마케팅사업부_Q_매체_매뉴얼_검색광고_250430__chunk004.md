### 검색광고 운영 기준 ( 기준, 판단, 사례 )
#### 1. 광고비 지출 규모 (기준,판단)
기준: 키워드 별 매출 대비 광고비 비율이 20~30% 이상일 경우
판단: 과다 지출 위험 → 키워드 단위 효율 점검 필요
#### 2. 광고 유입 매출(기준,판단)
기준: 광고 유입 매출이 광고비의 최소 2배 이상
판단: ROAS 200% 이상일 경우 기본 효율 달성
#### 3. 제2 매출 파악(기준,판단)
기준: 직접 매출 외 브랜드 검색 증가, 재구매, 회원가입 등
판단: 간접 효과도 성과 분석 시 포함 (리타겟팅 전략 기반)
#### 4. 목표 성과지표 (ROAS / CTR / CVR) (기준,판단)
기준: 업종 평균
ROAS 300%
CTR 3%
CVR 2%
판단: 자사 평균과 비교하여 개선 방향 도출
#### 5. 최대 전환비용(기준,판단)
기준: 객단가의 20~30% 이내
예시: 객단가 5만원 → 전환비 1.5만원 이하
#### 6. 최대 클릭비용(기준,판단)
기준: 클릭비 x CVR ≥ 목표 ROAS 충족
판단: 전환율 감안한 ROAS 유지 가능 여부 체크
#### 7. 입찰가 기준(기준,판단)
기준: 목표 전환비와 ROAS 기준
판단: 입찰가 = (객단가 × CVR) / ROAS 목표
#### 8. 키워드 ON/OFF 기준(기준)
비활성화 기준:
ROAS 150% 미만
CTR 1% 미만
CVR 1% 미만
#### 9. 지역/영역/시간 설정(기준,예시)
기준: 효율 높은 타겟 중심 노출
예시: 서울+모바일+20시~23시 → 예산 집중
#### 10. 모바일 노출 전략(기준,전략)
기준: 모바일 비중 80% 이상일 경우
전략: 모바일 전용 캠페인으로 CTR, CVR 최적화
(업종별 상황 고려 필수
#### 11. 키워드 수 관리(수량,판단)
수량: 50~200개
판단: 과소/과다 모두 효율 저하 요인
#### 12. 세부 키워드 관리
기준 : ROAS 300% / CTR 3% / CVR 2% 이상 기준으로 30일 기준 관리
기준 미만의 경우 OFF
#### 13. 광고문구 성과 측정
A/B 테스트
광고 문안 테스트 진행 시 반드시 문안 유형을 다르게 테스트
예시 : 프로모션 VS USP
#### 14. 파워링크 블로그 랜딩 전략 가이드 ( 운영목적, 콘텐츠유형 (목적,특징))

# 1. 검색 광고 계정 이관 절차
### 1. 이관 절차의 이해 및 필요성
검색 광고 플랫폼 운영을 위해서는 꼭 “이관 절차”가 되어야 합니다!
### 2. 이관 절차 정의 및 준비물 (광고주 계정 ID)
이관 절차 : 검색광고플랫폼 계정 권한 中 관리자 권한 부여 하는 절차!
이관 절차를 진행하기 위해서 필수 준비물 ! 광고주 계정 ID ( 숫자로 이루어져 있습니다 )
광고주 계정 ID 전달 받은 후 꼭 박예림 파트장에게 공유 해주세요!
공유 받은 광고주 ID를 확인 후 박예림파트장은 네이버 POS를 통해서 이관 메일 발송
이관 메일 발송 후 광고주 직접 승인 절차 후 이관 최종 완료
### 3. 이관 절차 단계 (ID 확인 > 공유 > 이관 메일 발송 > 광고주 승인)
광고주 :  광고 플랫폼 ID 확인
광고주 ID >  CCFM 공유
CCFM 네이버에 이관 메일 신청 후 최종 승인 후
광고주에게 승인 절차 안내
### 4. 광고주 계정 ID 확인 방법
접속 사이트
광고플랫폼 로그인 후 우측상단 내정보
왼쪽 기본정보칸-정보조회( 계정 ID :숫자 ) 광고주에게 공유 받기!
공유 받은 계정 ID 박예림 파트장에게 공유 주시면, 네이버POS를 통해 광고주에게 관리자 권한을 부여 할 수 있는 이관 메일 발송 !
a. 발송 시간 : 5분~ 15분 정도 소요
b. 박예림 파트장 이관 메일 갔어요 안내해주세요 하면 이관 절차 안내 시작 하시면 됩니다
