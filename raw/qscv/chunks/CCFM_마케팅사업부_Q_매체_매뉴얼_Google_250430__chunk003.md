### 결론
구글 Ads 리타겟팅은 단순히 웹사이트 방문자 전체에게 광고를 다시 보여주는 것을 넘어, 사용자의 행동과 의도를 얼마나 정교하게 파악하고 세분화하여, 각 단계에 맞는 최적의 메시지와 제안을 전달하느냐가 중요합니다.
명확한 목표 설정 아래 꾸준한 테스트와 최적화를 통해 리타겟팅 캠페인의 효과를 극대화할 수 있습니다.

## 전환 설정 및 추적
개요: 성과 분석의 기초가 되는 전환 데이터 수집 방법을 이해하고, 실전에서 정확한 전환 측정 체계를 설계할 수 있도록 안내
### 전환 이벤트 정의 및 설정 방법
전환 이벤트 정의
광고 클릭 이후의 행동(Action)을 의미
이벤트를 기준으로 구글 Ads는 성과를 측정하고 머신러닝 최적화를 수행
구글 Ads에서는 ‘구매’외에도 다양한 맞춤 전환 설정이 가능
주요 전환 이벤트
구매  (Purchase) : 제품/서비스 결제 완료, 자사몰 결제완료 페이지 도달 시
장바구니 (Add to Cart) : 장바구니 담기 버튼 클릭
결제시작 (begin_checkout) : 결제 정보 입력 페이지 도달 시
페이지 조회 (page_view) : 특정 웹페이지에 도달 시
상품 조회 (view_item) : 특정 상품 상세페이지에 들어갔을 때
회원가입 (Sign up) : 회원가입 완료 페이지 도달 시
전환 추적기간 정의
[IMG]
1) 클릭연결 전환 추적 기간
- 사용자가 광고를 클릭한 후 전환이 발생하기까지의 기간
2) 조회 완료 전환 추적 기간
- 동영상 광고를 끝까지 시청한 후 전환이 발생하기까지의 기간
3) 조회 후 전환 추적 기간
- 사용자가 광고를 본 후, 나중에 해당 사이트를 방문하여 전환이 발생하기까지의 기간
- 광고가 고객에게 노출된 후 고객이 나중에 유입되어 전환이 발생되는 경우
- 사용자가 광고를 클릭하지 않았더라도, 광고가 화면에 노출된 후 설정한 기간 내에 해당 사이트를 방문하여 전환이 발생하면 이를 조회 후 전환으로 기록
설정방법
카페 24를 통해 설치한 전환 스크립트는 영어로 전환액션 이름이 설정되어 있어, 해당 ‘Google Channel 00’ 이름으로 시작하는 전환을 위주로 세팅이 필요합니다.

전환 세부 설정
일반 구매
[IMG]
구매액이 다를 수 있기에 값은 ‘각기 다른 값을 사용’과 ‘대한민국 원(₩)’ 설정
주문 완료 창이 확인되어야 하기에 전환을 측정하는 횟수는 ‘모든 전환’으로 ‘주문 완료 창이 뜨는 모든 횟수를 전환으로 측정한다’는 의미로 설정
클릭연결 전환 추적 기간 : 조회 완료 전환 추적 기간: 조회 후 전환 추적 기간은 전환을 측정하기까지 간접전환에 대한 기여기간을 설정하는 것 신규브랜드의 경우 모수 확보를 통한 학습을 위해 전환 기여기간을 길게 잡고 감
- 신규 브랜드: 30일 / 7일 / 1일
- 기존 브랜드: 7일 / 3일 / 1일
구글 Ads에서 신규 브랜드와 기존 브랜드의 정의
'신규'와 '기존' 브랜드를 구분하는 것은 실제 브랜드의 런칭 시점 그 자체보다는, 구글 Ads 내에서 운영 히스토리와 데이터 축적 수준, 그리고 시장 내 브랜드 인지도 (특히 브랜드 검색량 등)가 얼마나 확보되어 있는가를 바탕으로 판단하는 것이 중요합니다.
신규 브랜드의 기준
구글 Ads 계정 또는 해당 제품 관련 캠페인을 운영한 기간이 짧음 (3개월 미만)
전환 데이터(특히 구매, 전환 가치)가 매우 적거나 거의 없음 (월 30~50회 미만)
브랜드 관련 키워드의 검색량이 거의 없거나 매우 낮음. (약 1,000회 미만)
기존 브랜드 기준
구글 Ads를 6개월 이상 꾸준히 운영한 이력이 있음
충분하고 안정적인 전환 데이터(월 50회 이상 꾸준히 발생)가 축적되어 있음
의미 있는 수준의 브랜드 검색량이 꾸준히 발생하며 (1,000회 이상) 브랜드 검색을 통한 전환 비중도 존재할 때
기여분석은 ‘데이터기반’으로 설정 → 전환이 일어나기까지 어떤 소재를 통해 전환이 일어났는지 시청시간, 체류시간 등 고객에게 영향을 준 비율을 측정하여 전환가치를 배분
이하 ‘네이버페이 구매’, ‘장바구니’, ‘결제시작’은 아래의 이미지와 동일하게 세팅 필요
네이버페이 구매
횟수를 제외하고 일반 구매와 동일하나 ‘네이버페이 구매’ 클릭 시 전환이 측정되기에 악의를 가지고 구매를 반복하거나, 결제 창에서 이탈이 반복될 수 있기에 횟수를 ‘전환 1회’로 세팅 필요

#### 스마트스토어
구글 Ads는 스마트스토어에 전환 추적 태그 삽입이 불가능하기 때문에, 머신러닝이 불가합니다.
그럼에도 스마트스토어로 랜딩이 필요한 경우,전환보다는 트래픽 중심 캠페인 운영이 효과적이며, 다음과 같은 방식으로 측정을 보완해야 합니다
