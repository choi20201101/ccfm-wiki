### 4. 적합한 키워드 찾기 방법
#### 주력 상품/서비스 연관 키워드 선정
광고를 통해 알리고자 하는 주력 상품 및 서비스가 무엇인지 결정하여 관련 키워드를 선정 
광고하고자 하는 사이트의 메뉴 혹은 카테고리와 관련된 키워드도 선정
#### 확장검색 활용 (검색어 보고서 분석)
확장검색으로 광고를 노출하고 검색어 보고서에서 검색유형 "확장"인 검색어의 성과를 확인하여 적극적으로 관리하고 싶은 경우, 키워드로 추가
#### 로그 분석 활용 (유입 키워드 분석)
추적 기능이나 로그 분석 등을 활용하여 내 사이트를 방문한 사용자가 어떤 키워드를 통해 유입되었는지 살펴보고, 해당 키워드 및 유사 키워드를 선정
#### 키워드 트렌드 파악 (인기 검색어 활용)
네이버 인기 검색어 및 검색창 아래의 검색어를 통해 키워드의 트렌드를 파악
#### 고효율 키워드 확장
클릭률과 전환율이 모두 좋은 고효율 키워드는 광고비 대비 매출이 높은 키워드를 의미 이미 효과가 검증된 고효율 키워드를 바탕으로 세부 키워드를 확장. 단, 취급하는 상품과 서비스가 많아 이미 등록한 키워드가 많다면 예산(광고비)을 고려하여 순차적으로 확장하는 것 추천. 광고 시작 후 데이터가 누적되지 않았다면, 월 노출 수 1,000건 이상이고 클릭률 1% 이상이며 평균 클릭 비용이 저렴한 키워드를 우선으로 검토

### 5. 소재 등록 및 작성
#### 1. 소재 정의 및 구성 요소 (제목, 설명, 연결 URL)
정의 : 사용자가 검색 후 최초로 만나는 상품이나 서비스에 대한 정보
구성 : 사이트의 제목과 설명, 광고 클릭 시 이동되는 페이지인 연결URL로 구성
제목 : 업체명을 포함한 문구 ( 띄어쓰기 포함 15글자 )
설명 : 업체 관련 설명 문구 ( 띄어쓰기 포함 45글자 )
연결 URL : 광고하고자 하는 사이트를 대표하는 URL인 '최상위 도메인(사이트 내 모든 페이지에서 공통으로 확인되는 URL)'
#### 2. 소재 등록 가이드 (그룹당 개수, 노출 방식, 작성 원칙)
소재는 광고그룹당 최대 8개(단일형 : 최대 5개 / 반응형 : 최대 3개)까지 등록
2개 이상의 소재를 등록한 경우 성과가 우수한 소재의 노출 비율을 자동적으로 조절하여 노출
이용자가 여러 광고 사이트 중 나의 사이트를 클릭하여 방문할 수 있도록 홍보하려는 상품과 서비스가 가지고 있는 장점/혜택 등 타 업체와의 차별성이 최대한 잘 드러나도록 소재를 작성추천천
단, 소재는 가이드에 맞게 작성되어야 하며 등록하고자 하는 소재가 가이드에 맞지 않는 경우 광고노출불가가
#### 3. 효과적인 소재 문안 작성 방법 (키워드 활용, 간결성, USP 강조, 액션 유도)
소재(문안)에 키워드 사용
고객은 자신이 검색한 검색어와 가장 관련이 높아 보이는 광고에 관심을 보이는 경향
검색한 문구가 들어간 광고는 그렇지 않은 광고보다 고객의 관심을 받을 가능성이 높음
제목과 설명문은 이용자의 검색어와 매칭되면 굵은 글씨로 표시
이렇게 되면 이용자가 검색한 키워드와 관련이 있는 광고가 강조되어 광고의 주목도가 높음
간결하면서도 눈길을 사로잡는 소재(문안) 사용
제품의 USP 강조
할인 및 프로모션 외에 무료배송 등의 이벤트를 기획하고 있다면 소재를 통해 전면에 제시
고객의 액션을 유도하는 내용으로 작성
주문하기나 리뷰 보기 등의 유도 문안을 통해 사용자에게 긍정적인 경험을 제공하면 전환율 상승

### 2. 광고 그룹 생성 전 키워드 전략 수립
#### 2-1. 키워드 유형 (브랜드, 제품/카테고리, 수식어/속성)
##### 브랜드키워드 (정의,예시)
정의 : 브랜드 자체를 검색하는 키워드로 브랜딩과 직접적인 유입 목적
키워드 조합 및 키워드 예시
브랜드명 단독: 메라블,메디라보,MEDILABO
브랜드명+제품명 : 유쎄라블아이크림,유쎄라블리프팅,유쎄라블리프팅900샷아이크림
브랜드명 + 세부키워드: 유쎄라블아이크림가격,유쎄라블아이크림피부과,유쎄라블아이크림내돈내산후기,유쎄라블리프팅효과
##### 상품 키워드 (정의, 예시, 실무 팁)
정의 : 특정 제품군을 포괄하는 일반키워드로, 검색량이 높고 경쟁도 높음
키워드 예시 : 아이크림, 수분크림, 레티놀크림
실무 팁 : 일반 키워드의 경우 “기본 뼈대”
여기에 가격,후기,사용후기, 증상 등을 붙여 확장
기본 → +효능 : 아이크림 추천, 주름개선아이크림
기본 → +타겟층 : 40대 아이크림, 남자아이크림
기본 → +리뷰/후기 : 아이크림 내돈내산, 아이크림실제후기
##### 수식어/속성(정의,예시)
정의 : 제품의 특징, 용도, 소재, 대상 등을 수식하는 키워드 
구매 의도가 명확한 핵심 키워드로 전환율 매우 높음
예시 : 아이크림추천,아이크림 내돈내산, 아이크림후기, 다크써클아이크림, 눈밑아이크림, 임산부아이크림
#### 2-2. 키워드 선정 시 주의 사항 (과도한 단가,무관키워드,과도한구체성,정합성,상품단가매칭,실검색량)
##### 과도한 단가
근거 : 월간 검색량이 높을 수록 경쟁심화 → ROAS 하락 가능성 증가
실제 사례 : 검색량 1만 이상 키워드 / 평균 CPC 600원 이상 ( 전환당 비용 증가, 매출대비 효율 저하 )
운영기준 : ROAS 300% 이상 유지가 어려운 키워드는 재평가
##### 무관 키워드
근거 : 제품과 연관도가 낮으면 이탈률 증가, 전환률 하락
사례 : “눈가관리법”,”노화방지법” 등 → 제품과 직접 연결 안 될 시 전환 실패
운영기준 : 랜딩페이지 제품 설명과 1:1 정합되는 키워드만 유지하며, 전환률 CVR 1% 미만 키워드 반복적 발생 시 OFF
##### 과도한 구체성
근거 : 적은 검색량으로 분리되며, 광고 노출이 잘 되지 않음
사례 : “ 40대 중년여성 눈밑 꺼짐 개선 아이크림 추천”→ 검색량 10미만 클릭 미발생
운영기준 : 3~4개 단어 이내 핵심 키워드 운영/ 구체적인 키워드는 자동완성 여부를 확인 후 등록 추천
##### 정합성
SA 노출은 “ 키워드 매칭” 기반 → 타이틀/이미지와 연관 없으면 CTR 저조
사례: “리프팅아이크림” → 클릭률 평균 이하
운영기준 : 광고 문구 및 썸네일 주요 키워드 필수 / 클릭률 CTR 1%이하 키워드는 소재- 키워드 적합성 점검필요
##### 상품 단가 매칭
근거 : 상품 단가 대비 CPC 과도한 경우 전환당비용 CPA 증가 → 마진이 남지 않아 광고비 대비 효율 저하
사례 : 상품 가격 30,000원 / CPC 1200원, CVR 2% → 전환 1건당 평균 60클릭 필요 기준
→ 전환당 비용 약 72,000원 발생 → 광고비 > 매출보다 적자
운영 기준 : 상품가 기준 CPC 상한가를 반드시 설정 필요 / ROAS 기준(300% 이상) 유지 가능한 단가만 지속 집행
실 검색량 확인
근거 : 소비자 들이 실제로 자주 검색하는 키워드 인지 확인을 하기 위하여 실제 검색량을 확인 해야 하며, 검색량의 기준의 경우 월간 검색량 1,000회 기준으로 확인 /  100회 미만의 경우 노출 제외

#### 2-3. 키워드 추출 방법 (키워드 도구, 자동완성/연관검색어, 쇼핑 인사이트, 자사몰 데이터, 경쟁사 벤치마킹)
##### 네이버 키워드 도구 활용

### 5.키워드 확인 및 제외 키워드 설정
#### 1. 쇼핑 노출 검색어 확인 방법
쇼핑검색 캠페인의 광고가 노출되었던 검색어 리스트는 광고그룹의 '제외검색어' 탭을 통해 확인
광고그룹의 ‘제외검색어’ 탭에서 [+ 제외검색어 추가] 버튼을 선택하면, 최근 30일 기준, 광고를 통해 상품이 노출되었던 검색어 리스트와 노출수, 클릭수 지표를 참고 데이터로 제공
[IMG]
#### 2. 제외 키워드 정의 및 활용 (등록 이유, 기능, 제한 개수, 구문 기능)
쇼핑검색광고는 노출할 상품소재의 정보를 기반으로 연관키워드가 자동으로 매칭되어 해당 키워드의 검색결과에 노출되는 방식
등록 이유 :상품명/카테고리 기타 정보들과 연관된 다수의 키워드에 광고가 노출되기 때문에 자동 매칭된 키워드 중 광고노출을 제한하고 싶은 키워드는 '제외키워드'로 등록하여 관리
[IMG]
제외키워드 설정은 광고 의도와 다른 키워드이거나, 비용 대비 효율이 적다고 판단되는 경우, 또는 노출/클릭이 지나치게 많아 CPC 과금이 급격하게 증가한 경우 등 광고 노출을 원하지 않는 경우에 사용하는 기능으로 해당 검색어를 제외키워드로 등록하여 광고 노출을 제한
