# 1. 퍼포먼스 마케터를 위한 키워드 이해
• 퍼포먼스 마케터는 경쟁사 대비 높은 신뢰를 확보하기 위해 다양한 시장조사와 키워드 발굴을 수행해야 합니다.
• 구매자 리뷰, 사회적 통념, 반복되는 소비자 문제, 객관적 근거 등 다양한 기준을 바탕으로 키워드를 유형화할 수 있습니다.
## 1-1. 키워드 유형 분류
### 1. 메인 키워드
#### 브랜드 키워드 (정의, 조합, 예시)
정의 : 브랜드 자체를 검색하는 키워드로 브랜딩과 직접적인 유입 목적
키워드 조합 및 키워드 예시
브랜드명 단독: 메라블,메디라보,MEDILABO
브랜드명+제품명 : 유쎄라블아이크림,유쎄라블리프팅,유쎄라블리프팅900샷아이크림판
브랜드명 + 세부키워드: 유쎄라블아이크림가격,유쎄라블아이크림피부과,유쎄라블아이크림내돈내산후기,유쎄라블리프팅효과
#### 상품 키워드 (정의, 예시, 실무 팁)
정의 : 특정 제품군을 포괄하는 일반키워드로, 검색량이 높고 경쟁도 높음
키워드 예시 : 아이크림, 수분크림, 레티놀크림
실무 팁 : 일반 키워드의 경우 “기본 뼈대”qmffh
여기에 가격,후기,사용후기, 증상 등을 붙여 확장
기본 → +효능 : 아이크림 추천, 주름개선아이크림
기본 → +타겟층 : 40대 아이크림, 남자아이크림
기본 → +리뷰/후기 : 아이크림 내돈내산, 아이크림실제후기
#### 소비자 구매의도 키워드 (정의, 예시,성과예시)
정의 : 특정 고민 해결을 위한 키워드로 구매 직전 검색 단계 키워드
예시 : 아이백크림, 레티놀아이크림, 눈가주름크림,눈밑꺼짐화장품
성과 예시
레티놀아이크림 → CTR 3.8%, 전환율 4.5%
눈밑주름크림 후기 → CTR 4.1%, 전환율 5.2%
### 2. 서브 키워드
#### 감성 키워드 (정의, 예시, 성과예시)
정의 : 소비자 감정, 분위기, 느낌을 자극하여 직접적인 기능 설명이 아닌 감정적 공감 유도
예시 : 다클서클안화,눈밑꺼짐화장품, 비건아이크림
성과 예시
다클서클 완화 키워드 활용한 da캠페인
→ 30~40대 여성 타겟 맞춤 소재 + 맘카페·SNS 스타일 후킹 문구 활용
→ ROAS 370%, CTR 2.8% 기록
카피 예시 : 밤샘 티 나는 다크서클, 하루 1번 케어로 완화된 후기 봤어요?
#### 소비 타겟층 키워드 (정의, 조합, 예시)
정의 : 특정 연령, 성별 등을 명시한 키워드
키워드 조합 : 성별 : 남/ 여 OR 연령층 키워드
예시 :4050전용,50대전용아이크림,남자아이크림
성과 예시
50대아이크림 전환율 5.1%, 단가 대비 고효율
#### 가격 키워드 (정의, 예시)
정의 : 제품의 가격대, 할인정보, 가성비 등 강조 키워드
예시 :  아이크림900원,아이크림가격, 아이크림1+1,아이크림 할인,3만8천원아이크림
주의 사항 : 브랜드명 혼재 방지→ "메라블아이크림1+1"은 브랜드 키워드로 따로 분류
#### 숫자적 근거 키워드 (정의, 예시)
정의 : 정량적 수치나 수치 기반 근거 ( 랭킹, 순위, 효능,수치 등 ) ,환불 등 포함한 소비자에게 신뢰감과 명확한 정보를 제공하는 키워드
예시 : 30년 경력 한의사, 여드름균 99.9%임상완료 ,5번만 사용해도 효과
성과 예시
30년 경력 한의사 → 클릭률 3.2%, 구매전환율 4.5%
환불 내용 : 14일 만 환불 X > 2주 사용해도 여드름 피부 개선 없는 경우 환불
#### 소비자 문제 해결 키워드 (정의, 조합, 예시)
정의 : 소비자가 겪는 특정, 고민을 해결해주는 키워드이며, 정보 취득 목적이 강한 키워드
키워드 조합 : 문제 현상 + 후기 / 방법 / 실제후기/ 솔직후기 / 내돈내산
예시 : 눈밑꺼짐개선방법, 아이백크림실체후기, 주름없애는방법
성과예시
눈밑주름개선방법 → 전환율 4.7%, 키워드 기반 블로그/DA 연계
#### 리뷰 관련 키워드 (긍정/부정) (정의, 예시)
정의 : 사용자 리뷰를 기반으로 한 키워드로 긍정적 또는 부정적 의견
긍정 예시 : 청소기 흡입력에 놀람,유튜버 보고 후기, 입소문,만족도,000맘카페반응
부정 예시 :  000보다 떨어짐, 000 사지마세요,000VS000비교
성과예시
00 맘카페 후기 → CTR 3.5%, 실제 후기로 소재 제작 시 고성과
#### 사회적 근거 키워드 (정의, 예시)
정의 : 사람들의 사용 현황이나 연구 기관에서 개발된 제품 등으로 정의
예시 : sns대란, 비건화장품, k-뷰티,화해1위, 올리브영1위
#### 사회적 통념 키워드 (정의, 예시)
정의 : 고객이 살아온 문화와 교육에 기반하여 형성
예시 : 반창고 > 데일밴드 / 초코파이 정 / 보톡스
### 3. 확장 키워드
#### 경쟁사 키워드 (정의, 조합, 예시)
정의 : 경쟁사의 주요 키워드를 분석하여 활용하는 키워드
경쟁사 이름 + 제품명 : 아모레퍼시픽 아이크림, AHC 아이크림
#### 공포 키워드 (정의, 예시)
정의 : 소비자에게 공포,두려움을 유도하는 부정적 미래 예측으로 긴장감 유도
예시 : 눈가노화방치,피부노화경고,주름방치하면
#### 시즌성 키워드 (계절 키워드) (정의, 예시)
정의 : 계절감을 담은 키워드, 계절에 따라 검색량이 증가하는 키워드
예시 : 겨울보습크림, 여름수분크림, 연말선물세트
#### 쇼핑 키워드 및 카테고리 키워드 (정의, 예시)
정의 : 실제 구매 행동에 가까운 키워드
예시 : 메라블아이크림쿠팡, 아이크림추천, 아이크림인기순위
# 2. DA(디스플레이 광고)와 SA(검색광고) 비교 및 통합 전략
## 2-1. DA(디스플레이 광고)와 SA(검색광고)의 차이점
### 비교 항목 (광고 위치, 노출 방식, 목표, 비용 구조)

|  | 검색광고(SA) | 디스플레이광고(DA) |
|---|---|---|
| 광고위치 | 검색 결과 상단 | 네이버GFA,유튜브,메타 |
| 노출방식 | 특정 키워드를 검색한 사람에게 노출 | 관심사가 있을 것 같은 사람에게 노출 |
| 목표 | 구매 의도를 가진 사람에게  판매 유치 | 브랜드 인지도 향상을 통한 브랜딩화 및 관심사를 통한 판매 유치 |
| 비용구조 | 클릭당 비용 지불 (CPC) | 노출당 비용 지불 (CPM) |

## 2-2. DA + SA 활용 전략 ('메라블' 브랜드 사례)
### '메라블' 브랜드 개요 (소개, 주요 제품, 타겟 고객)
메라블은 피부과 시술의 핵심 기술을 화장품에 담아 기존 화장품으로는 경험할 수 없던 피부 개선 효과 제공
주요 제품: 유쎄라블 리프팅 900샷 아이크림, 아크 엑소좀 트러블 라인 등
타겟 고객: 피부 탄력 & 트러블 개선을 원하는 사람들
### DA(디스플레이 광고) 활용 전략 (목적, 전략, 예시)
목적 : 브랜드 인지도 상승 및 전환 유도
전략
네이버 GFA 광고 → 뷰티 관련 블로그, 카페, 뉴스 페이지에 배너 광고
유튜브 광고 → "피부 탄력 관리법" 영상에 6초~15초 광고 송출
SNS 광고 → 인스타그램, 페이스북, 틱톡에서 피부 고민별 솔루션 콘텐츠 제공
예시
"탄력 고민? 하루 2분이면 OK!" (유쎄라블 리프팅 아이크림 배너)
"트러블 피부엔 과학적 솔루션!" (아크 엑소좀 트러블 라인 영상 광고)
### SA(검색광고) 활용 전략 (목적, 전략, 예시)
목적 : 피부 고민이 있는 사람들에게 키워드 검색을 통해 홈페이지 유입 및  전환(구매) 유도
전략
브랜드 검색광고 → "메라블 아이크림", "메라블 크림" 키워드 광고
제품 기능 키워드 → "탄력 아이크림 추천", "트러블 크림 효과" 키워드  집행
정보성 키워드  → “메라블아이크림 내돈내산, 울쎄라 크림효과 “ 등 키워드 집행
리뷰 콘텐츠 활용 → 블로그, 인플루언서 협업 후 검색광고 연결
예시
"탄력 아이크림 추천? ★4.8 후기 검증된 메라블!"
"피부과 테스트 완료, 트러블 고민 끝!
### DA + SA 연계 활용 실전 예시 (목표, 실행 플랜, 기대 효과)
목표: DA로 인지도를 높이고, SA로 전환 극대화
실행 플랜
DA 광고 → "탄력 관리 필수템" 배너 광고로 고객 유입
SA 검색광고 → 유입된 고객이 "메라블 아이크림 후기, 메라블"등  검색하면 광고 노출
DA 리마케팅 광고 → 세부키워드 등록 하여 검색 후 구매하지 않은 고객에게 추가 광고 노출
SA 리마케팅 광고 → 장바구니 이탈 고객 대상 검색 광고로 재유입
기대 효과 : 브랜드를 먼저 접한 고객이 다시 검색 → 신뢰도 증가 → 구매율 상승
### 최종 결론 (DA+SA 조합의 효과 극대화)
DA(디스플레이 광고) → 브랜드를 먼저 알리는 역할
SA(검색광고) → 관심 있는 고객을 전환(구매)시키는 역할
DA 리마케팅 + SA 리마케팅 → 이탈 고객을 다시 유입
### 전략 실행 시 기대 효과 (브랜드 인지도, 고객 관심 유도, 트래픽 확보, 전환율 상승)
브랜드 인지도 상승 → DA 광고
고객 관심 유도 → SNS, 유튜브, 블로그 활용
검색 트래픽 확보 → SA 광고
전환율 상승 → SA + DA 리마케팅
이 전략을 실행하면 메라블의 브랜드 성장과 매출 상승을 극대화
# 3. 검색광고(SA) 심층 이해
## 1. 검색광고 정의
검색광고는 소비자가 특정 검색어를 검색할 때 검색어와 연관된 광고를 노출시켜, 
고객이 광고주의 홈페이지로 유입될 수 있도록 하는 광고
## 2. 검색광고의 특징 및 중요성
### 2-1 검색광고가 중요한 이유: 마케팅 퍼널 5단계 연계
[IMG]
#### 인지 (Awareness)
고객이 처음으로 제품이나 브랜드를 접할 때는 구체적이지 않은 일반 키워드 사용
#### 관심 (Interest)
고객이 브랜드를 알게 된 후에는 제품명이나 브랜드명 등 더 구체적인 키워드를 사용하여 정보 탐색
#### 고려(Consideration)
실제 구매가 이루어지는 단계에서는 높은 전환율을 보이는 키워드를 중심으로 집중 관리하며, 
이를 더욱 효과적으로 활용하기 위해 ‘키워드확장’은 선택이 아닌 필수
#### 전환(Conversion)
실제 구매 또는 문의
#### 재구매(Retention)
브랜드에 만족해 다시 구매
### 2-2 키워드 광고 퍼널 전략의 이점
맞춤 광고 가능
퍼널의 각 단계 (인지-관심-고려-구매)에 맞게 키워드를 나눠서 광고하는 경우
 고객이 원하는 정보를 정확하게 제공할 수 있어 광고효과 상승
#### 광고 비용 효율성 증가
단계별 최적의 키워드를 선택해 집중 광고하면 불필요한 광고비 낭비를 줄이고 광고비를 효율적으로 관리
#### 브랜드 인지도 및 신뢰 상승
키워드 광고를 단계별로 적절히 제공하면 고객들이 브랜드를 계속 보게 되고, 
점차 브랜드에 대한 신뢰도가 높아져 구매 확률 상승
#### 성과 측정 용이성
단계별로 키워드를 구분하면, 어디서 광고가 잘되었고 어디서 부족했는지 쉽게 파악 할 수 있어 
광고 전략을 빠르게 개선 할 수 있음
#### 경쟁사 이탈 방지
고객이 관심을 가지고 구매를 고려할 때 지속적으로 광고에 노출되면 경쟁사로 이탈하지 않고
