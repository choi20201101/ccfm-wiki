# 영상_필수_기획_항목_Appendix

영상 필수 기획 항목 Appendix
## 광고 소재의 방향성
해당 영상 소재가 달성해야 하는 광고 목적 명확화
광고 운영 목표 점검 : 제품 판매, 브랜드 인지도, 프로모션 참여 등
예) “체취 고민 해결 중심의 전환소재” / “브랜드 인지도 증진용 참여소재”
(영상 소재 썸네일 예시 적용 필요)
이미지 소재와 영상 소재의 차별점 인지
각 스토리 구성간 영상 길이 분배 필요
첫 : 문제 제기를 통한 공감대 형성
중간 : 문제 해결에 대한 근거
마지막 : 지금 구매시 고객이 바로 얻을 수 있는 구매 혜택 안내
이미지 소재 성과 데이터, 인사이트 기반 강화 포인트 구성
이미지 소재에서 표현할 수 없는 요소
예시 : 후라이팬에 구운 모짜렐라 치즈가 3초간 늘어나는 장면
마스크팩 사용 후 5분뒤 모공이 줄어든 장면

호텔향이 나는 디퓨저를 체험하고 리뷰하는 장면
## 핵심 USP 및 페르소나 정보
이번 소재에서 반드시 표현해야 할 USP
예) “12시간 체취 차단”
메인 타겟 페르소나
USP 기반으로 메인타겟을 잡고, 상세 페르소나 설정
페르소나를 어떤 이유로 설정하였는지 근거, 이유 필요
연령/성별/라이프스타일 등 구체적인 상황 적용
예) “체취 고민 있는 30~50대 남성 직장인”
## 스토리라인 구조 제안
기본 플로우
문제 → 해결 → 근거 → 설득 → CTA 구조 설계
문제 : 고객이 공감할 수 있는 문제 또는 불편함 제시
해결 : 문제 해결에 이후 기대되는 상황 제시
근거 : 문제 해결이 될 수 밖에 없는 근거 제시
설득 : 타인의 경험을 통한 신뢰 요소 제시
CTA : 고객이 얻을 수 있는 즉각적인 이점 제시
적용 예시
문제 제기 (체취 고민 상황) → 해결(특허 성분) → 근거(임상 수치, 후기가 입증) → 설득(사용 후기, 신뢰성 강화) → CTA(구매 혜택 안내)
이미지 컨텐츠로 설득하지 못한 요소를 영상 컨텐츠에 적용
향기 / 맛 / 시간별경험 / 속도감
[음식물처리기 적용 예시]
문제 제기 : 여름철 음식물쓰레기를 매일 버리지 않으면 날파리가 생김
해결 : 음식물처리기를 사용하면 음식물쓰레기를 갈아버리기때문에 매일 버리지 않아도 날파리가 안생김
근거 : 강력한 BLDC 모터로 고온처리와 함께 4시간만에 건조 분쇄 가능, 냄새가 안남. 부피 90% 이상 감소
설득 : 와디즈 펀딩 9억, 실제 리뷰 1,200개를 통한 고객 경험 간접 체험
CTA : 지금 구매시 40% 선착순 할인
