## 상단 고정 리뷰 통제
Why?
판매자가 10번 USP를 이야기하는 것보다, 소비자가 1번 USP를 이야기하는 것이
소비자에게 더욱 강한 신뢰감을 줄 수 있습니다.
상세페이지 최상단에서 소비자들의 리뷰로써 USP를 객관적으로 강조해야합니다
페이지 최상단에 리뷰 위젯이 노출 되는가?
모바일 환경으로 접속 시 상세페이지 최상단 리뷰 위젯 배치 확인
쇼핑몰 고정 리뷰 내 USP 강조를 통한 신뢰도 증진
리뷰 수량(500~1,000개 수준 이상)을 통한 신뢰도 증진
리뷰 상단 배치가 안 되어있을 경우 상단 배치 방안 제안
리뷰 솔루션을 활용하고 있지 않을 경우
리뷰 솔루션 활용 제안 - 알파 리뷰, 크리마 리뷰, 스냅 리뷰 등
리뷰 솔루션 활용 시 장점
리뷰 작업 수행 가능
소비자 오가닉 리뷰 작성 접근성 확대
리뷰 관리 및 통제 가능
리뷰 솔루션을 활용하고 있을 경우
각 리뷰 솔루션 내 문의 진행 후 적용 제안
활용 리뷰 솔루션 파악
파악 방법 : 아이보스 스크립트 분석기
동일 솔루션사 브랜드 레퍼런스 기반 제안
상단 고정 리뷰(쇼핑몰 추천 리뷰) 내 광고와 동일한 USP가 담겨있는가?
모바일 환경 내 리뷰 열람 시 추천순으로 정렬 되는지 확인
상단 고정 리뷰 내 포토 및 동영상 리뷰가 배치되는지 확인
포토 및 동영상 리뷰 + 긴 텍스트 리뷰 내 USP가 드러나는지 확인
B&F가 시각적으로 잘 드러나는 이미지 및 동영상 배치
텍스트 내 광고와 동일한 USP를 강조

## 소비자 유입 초기 통제
Why?
상세페이지 내에서 객관적인 USP에 대한 신뢰도를 얻지 못한
유입 초기 소비자의 동선이 불편할 경우,
소비자는 상세페이지를 조회하기 전 초기 단계에서 이탈할 것입니다.
불필요한 팝업 등 상세 콘텐츠 노출이 방해되지 않는가?
팝업 노출 확인
초반부 2개 이상의 팝업이 노출될 경우
USP 신뢰도에 도움이 되지 않는 단순 홍보성은 삭제 제안
필요한 팝업이라면, 상세페이지 조회 중 노출되도록 위치 수정 제안
조회 중인 제품과 연관 없는 정보가 노출될 경우
삭제 및 정보 수정 제안
플로팅 배너 노출 확인
플로팅 배너 : 특정 위치 내 스크롤 시 자연스럽게 등장하는 배너
의도한 위치 및 내용이 적절히 강조되는지 점검 후 제안
고정 노출 배너일 경우, 조회에 방해가 되지 않는지 점검
상세 콘텐츠를 가리거나 침범하지 않는지 점검
CTA(문의, 최상단 이동 등)일 경우 원하는 위치로 랜딩 되는지 점검
페이지 로드 속도가 콘텐츠 조회에 영향을 미치는가?
페이지 로드 속도 확인
모바일 환경(WIFI / 모바일 데이터) 내 접속 속도 확인
접속 속도가 느릴 경우 제안
상세페이지 콘텐츠 중 경량화 할 수 있는 부분 체크
불필요한 GIF > 이미지 배너로 변경
영상 콘텐츠 > 용량 축소 및 임베드 링크 활용
## CTA(Call-To-Action) 구성
Why?
CTA 버튼 클릭이란, 소비자가 상세페이지 유입 후
보다 심화된 액션으로 전환됨을 뜻합니다.

리뷰 조회, 구매, 회원가입 등 소비자의 의지로 클릭하는
CTA 버튼은 소비자에게 더욱 매력적으로 보여야합니다
CTA(구매 버튼, 신청 버튼)가 스크롤 상단 및 주요 구간에 적절히 배치되어 있는가?
네비게이션 바 CTA 확인
심화된 액션(리뷰 조회, 구매 등)을 유도하고 있는지 점검
리뷰 수량이 강조 되고 있는지
구매보다 장바구니등 하위 액션이 더 주목도있게 노출되진 않는지

## 모바일 환경 최적화
Why?
온라인 소비자의 절대다수는 PC가 아닌 모바일 환경 속에서 광고에 노출됩니다.

자연스럽게 랜딩페이지도 모바일 환경으로 접속되니 모바일 환경이 최적화되어야합니다.
랜딩페이지가 모바일 화면에서도 가독성, 인터페이스가 우수한가?
모바일 디자인 최적화 확인
모바일 가독성
PC 대비 화면이 작은 모바일 환경 내 가독성 점검
텍스트 문단, 줄바꿈 등 이상이 없는지 점검
디바이스 별 점검
IOS / Andriod 환경으로 교차 점검
영상 자동 재생 여부, 로딩 및 오류 점검
CTA, 상품 설명, 이미지가 모바일 환경에 최적화되어 있는가?
모바일 소비자 동선 점검
UI 확인
옵션 드롭다운 내 텍스트 점검
옵션 선택 후 가격 정보 및 구매 CTA 버튼 점검
배너 디자인 요소 확인
PC로 작업된 배너 모바일 환경 내 디자인 점검
호스팅사 반응형 설정으로 자동 PC / 모바일 최적화 점검
