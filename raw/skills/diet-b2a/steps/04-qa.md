# Step 04 — 품질 검증 (QA)

## 목적
3개 최종 영상이 `PLAN.md § 6 Definition of Done` 을 모두 충족하는지 기계 검증.

## 실행
```bash
python scripts/qa_check.py --config config/<project>.json
```

## 검사 항목
| 항목 | 방법 | 실패 처리 |
|---|---|---|
| 3개 mp4 파일 존재 | fs | abort |
| 해상도 1080×1920 | ffprobe | abort |
| 길이 범위 | ffprobe (영상1 8–11s, 영상2·3 9–11s) | warn |
| fps ≥ 30 | ffprobe | warn |
| 오디오 aac 스트림 포함 | ffprobe | warn (무음 허용 설정 시 skip) |
| 얼굴 모자이크 픽셀화 감지 | 영상1/2/3 한 프레임 추출, 지정 박스 영역 엣지 에너지 ≤ 임계값 | fail → step03 재실행 유도 |
| 컷 지점(영상2·3 @ 4.0s) 존재 여부 | scene cut 감지(lavfi select='gt(scene,0.4)') | warn |

## 출력
- `output/qa_report.json` (각 항목 pass/fail/metric)
- `output/qa_report.md` (사람이 읽는 요약)
