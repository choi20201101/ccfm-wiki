# HOOK — 훅 카피 & 자막 템플릿

> 같은 포맷이라도 훅 카피 한 줄이 조회수를 좌우한다. 이 문서는 플랫폼별로 검증된 자극 카피 패턴과 스킬의 자막 슬롯 매핑을 제공한다.

---

## 1. 자막 슬롯 (스킬에서 노출되는 편집 포인트)
| 슬롯 키 | 위치 | 기본값 | 교체 규칙 |
|---|---|---|---|
| `title` | 영상1 하단 중앙 | `160cm / 골격 스트레이트` | `{키} / {체형}` or 유행 해시태그 훅 |
| `label_before` | 영상1 좌반 상단 | `Before` | `전`, `D-0`, `시작` 등 |
| `label_after`  | 영상1 우반 상단 | `After`  | `후`, `D+90`, `성공` 등 |
| `date_before`  | 영상2·3 체중계 위 | `2025년 12월` | 실제 시작 월 |
| `date_after`   | 영상2·3 체중계 위 | `2026년 3월` | 실제 완료 월 |

## 2. 훅 카피 3단 구조 (Hook / Value / CTA)
1. **Hook** (0~1초, 스크롤 멈춤): 숫자·충격·호기심
2. **Value** (중간): "변신"의 근거 — 기간/방법/식단
3. **CTA** (마지막 0.5초): 팔로우/프로필/댓글 유도

### 2.1 후킹 타이틀 30선 (영상1 하단 또는 영상2·3 시작 자막으로 주입 가능)
#### 숫자형
- `3개월만에 -20kg, 골격 스트레이트가 이게 되네`
- `88kg → 44kg 실화? 키 160`
- `반년에 인생 갈아엎었습니다`
- `44kg, 내가 나를 못 알아봄`
- `현실 직장인 다이어트 3개월 기록`

#### 대비형
- `같은 사람 맞습니다`
- `왼쪽: 자기혐오 / 오른쪽: 자기애`
- `뚱뚱할 땐 몰랐던 것들`
- `옷장 싹 다 버렸습니다`

#### 공감/고백형
- `체중계 무서워서 못 올라갔던 사람`
- `거울 보고 울던 내가`
- `친구가 먼저 몰라봄`
- `소개팅 취소당하고 결심했습니다`

#### 질문형 (댓글 유도)
- `이 방법 궁금한 사람?`
- `-20kg 어떻게 했게?`
- `식단? 운동? 뭐가 먼저일까`

### 2.2 Before/After 라벨 변형
| 톤 | Before | After |
|---|---|---|
| 표준 | Before | After |
| 한국어 | 전 | 후 |
| 기간 | D-0 | D+90 |
| 감정 | 숨김 | 해방 |
| 체중 | -88.0kg | -44.0kg |
| 의지 | 다짐 | 증명 |

### 2.3 날짜 자막 변형 (영상2·3)
- `2025년 12월` → `2026년 3월` : 기본
- `시작` → `3개월 후` : 기간 강조
- `D-Day` → `+90일` : 챌린지 뉘앙스
- `BEFORE 66.4` → `AFTER 44.8` : 숫자 그대로

## 3. 업로드 캡션 템플릿 (릴스 본문)
```
{Hook line}

{기간} 동안 한 것:
1. {식단 원칙 1줄}
2. {운동 원칙 1줄}
3. {습관 원칙 1줄}

저처럼 할 수 있어요.
궁금한 거 댓글 👇
#다이어트 #비포애프터 #골격스트레이트 #헬린이 #다이어트기록
```

## 4. 훅 A/B 테스트 가이드
- 동일 영상을 카피만 3개(숫자형 / 공감형 / 질문형)로 바꿔 업로드 → 24h 후 저장수·공유수 비교
- `config.json` 의 `copy` 블록만 교체하면 A/B 바리에이션을 스킬이 자동 생산

## 5. 금지 카피 (리젝/신고 위험)
- 특정 약/시술 브랜드 언급
- 극단적 단식/굶기 권장
- 의료 효과 단정 (`반드시 살 빠집니다` 등)
- 타인 신체 비하 비교

## 6. 자막이 들어갈 수 없는 데드존
- **영상1**: 체중계 박스 (각 중심 270/810, y=245~495), 인물 얼굴선 ± 모자이크 영역
- **영상2**: 우상단 체중계 박스(중심 780,430 / 400폭), 인물 몸통선
- **영상3**: 좌상단 체중계 박스(중심 260,430 / 380폭)

자막은 이 영역 밖 — **영상1은 하단/각 라벨 영역, 영상2·3은 체중계 박스 위쪽 여백 또는 하단 여백** 에만 배치.
