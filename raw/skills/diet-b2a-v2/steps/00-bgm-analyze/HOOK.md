# step-00 HOOK — BGM을 잘 고르는 법

## 왜 BPM 분석이 훅이 되는가
- 릴스는 **첫 1초 BGM**이 스크롤 멈춤의 핵심. BPM이 곧 주목력.
- BPM 95 = 감성·고백 톤, BPM 129 = 챌린지·승리 톤.
- 드롭 지점 = 썸네일 큰 움직임이 시작되는 위치 = 알고리즘의 "retention 가중점".

## 이 프로젝트의 BGM 전략
- 느린 2곡(DUX/DWnms)은 **변화를 천천히 보여주는 포맷**(좌우분할, 좌측체중계)에 배정
- 빠른 1곡(DW3)은 **가장 역동적인 포맷**(우측체중계 + 후반 춤)에 배정
- 컷 지점 = 드롭 지점과 맞춰서 "음악이 빵 터질 때 몸이 바뀐다" 착시 극대화

## 다른 프로젝트 전이 규칙
- BPM < 100 → 감성형 2~3초 인트로 + 중단 컷 1번
- BPM 100~130 → 표준 드롭 컷 (현 프로젝트 영상2)
- BPM > 130 → 더 잦은 컷 + 후반 루프 속도 1.4x
