# M1 입력 패키지 (Windows)

실행 방법: 웹앱에서 제출 후 `/m1v2` 실행. 인자로 폴더 경로 지정 가능: `/m1v2 [폴더경로]`

---

## 경로 고정값 (절대 변경 금지)

```
DESKTOP_INPUTS  = C:/Users/{WINDOWS_USER}/Desktop/클로드/m1-inputs
DESKTOP_WORK    = C:/Users/{WINDOWS_USER}/Desktop/클로드/m1-work
NAS_INPUT       = //192.168.0.103/01_마케팅사업부_직원/{NAS_USER}/3팀 영상 자동화/M1/input/사용자 입력 MD/input
NAS_DONE        = //192.168.0.103/01_마케팅사업부_직원/{NAS_USER}/3팀 영상 자동화/M1/input/사용자 입력 MD/done
NAS_OUTPUT      = //192.168.0.103/01_마케팅사업부_직원/{NAS_USER}/3팀 영상 자동화/M1/output
NAS_BRAND_MD    = //192.168.0.103/01_마케팅사업부_직원/{NAS_USER}/3팀 영상 자동화/M1/input/브랜드 분석 MD
NAS_VIDEO_MD    = //192.168.0.103/01_마케팅사업부_직원/{NAS_USER}/3팀 영상 자동화/M1/input/영상 분석 MD
```

---

## STEP 0: 입력 폴더 탐색 — 반드시 Bash 툴로 실행

```bash
DESKTOP_INPUTS="C:/Users/{WINDOWS_USER}/Desktop/클로드/m1-inputs"

if [ -n "$1" ]; then
  SRC_FOLDER="${1%/}"
else
  SRC_FOLDER=$(ls -td "$DESKTOP_INPUTS"/*/ 2>/dev/null | grep -v '브랜드명_날짜' | head -1 | sed 's|/$||')
fi

if [ -z "$SRC_FOLDER" ] || [ ! -d "$SRC_FOLDER" ]; then
  echo "웹앱에서 먼저 입력을 제출해주세요."
  exit 1
fi

LATEST_TEXT=$(ls -t "$SRC_FOLDER"/m1_text_*.txt 2>/dev/null | head -1)
LATEST_IMG=$(ls "$SRC_FOLDER"/m1_img_* 2>/dev/null | head -1)
BRAND_NAME=$(basename "$SRC_FOLDER" | sed 's/_[0-9]\{8\}$//')
DATE_STR=$(basename "$SRC_FOLDER" | grep -oE '[0-9]{8}' | tail -1)

if [ -z "$LATEST_TEXT" ]; then
  echo "텍스트 파일이 없습니다: $SRC_FOLDER"
  exit 1
fi

echo "브랜드: $BRAND_NAME / 날짜: $DATE_STR"
```

- Read 툴로 `$LATEST_TEXT` 읽어서 파싱 (브랜드명, Q1, Q2)
- 레퍼런스 이미지 있으면 Read 툴로 시각 확인

---

## STEP 1: Desktop 작업 폴더 생성 — 반드시 Bash 툴로 실행

```bash
DESKTOP_WORK="C:/Users/{WINDOWS_USER}/Desktop/클로드/m1-work"
NAS_OUTPUT="//192.168.0.103/01_마케팅사업부_직원/{NAS_USER}/3팀 영상 자동화/M1/output"

# 출력 폴더명 결정 (YYYYMMDD_브랜드명_N)
i=1
while [ -d "${NAS_OUTPUT}/${DATE_STR}_${BRAND_NAME}_${i}" ] || [ -d "${DESKTOP_WORK}/${DATE_STR}_${BRAND_NAME}_${i}" ]; do
  i=$((i+1))
done
OUTPUT_BASE="${DATE_STR}_${BRAND_NAME}_${i}"
LOCAL_WORK="${DESKTOP_WORK}/${OUTPUT_BASE}"

mkdir -p "$LOCAL_WORK"
echo "로컬 작업 폴더 생성: $LOCAL_WORK"
```

---

## STEP 2: NAS → Desktop 로컬 작업 폴더로 파일 수집

### 2-1: 사용자 입력 MD 생성 (Write 툴로 로컬에 저장)

파일 경로: `$LOCAL_WORK/03_${OUTPUT_BASE}.md`

```markdown
# M1 사용자 입력 패키지 — [브랜드명]
생성일: YYYY-MM-DD

## 브랜드명
[브랜드명]

## Q1. 강조 포인트
[Q1 답변]

## Q2. 분위기/연출
[Q2 답변]

## 레퍼런스 이미지
[파일명 또는 없음]
```

레퍼런스 이미지가 있으면 `컨셉이미지.{확장자}` 이름으로 복사:
```bash
if [ -n "$LATEST_IMG" ]; then
  IMG_EXT="${LATEST_IMG##*.}"
  cp "$LATEST_IMG" "$LOCAL_WORK/컨셉이미지.${IMG_EXT}"
fi
```

### 2-2: 브랜드 에셋 MD 복사 — 반드시 Bash 툴로 실행

```bash
NAS_BRAND_MD="//192.168.0.103/01_마케팅사업부_직원/{NAS_USER}/3팀 영상 자동화/M1/input/브랜드 분석 MD"
BRAND_ASSET=$(ls "$NAS_BRAND_MD" 2>/dev/null | grep -i "$BRAND_NAME" | head -1)

if [ -n "$BRAND_ASSET" ]; then
  cp "$NAS_BRAND_MD/$BRAND_ASSET" "$LOCAL_WORK/01_${BRAND_ASSET}"
  echo "✅ 브랜드 에셋 복사: 01_${BRAND_ASSET}"
else
  echo "⚠️ 브랜드 에셋 없음 — WebSearch로 자동 생성 진행"
fi
```

브랜드 에셋 파일이 없으면 중단하지 말고 아래 순서로 자동 생성:
1. WebSearch 툴로 `[브랜드명] 제품 성분 효능` 검색 (2~3회)
2. 검색 결과 + Q1/Q2 종합해 브랜드 에셋 MD 작성:
   ```
   # 브랜드 자산 규격화: [브랜드명]
   > ⚠️ 자동 생성 — 공개 정보 기반 추론
   ## Section 1. 브랜드 식별자
   ## Section 2. 핵심 메시지 재료 (USP 리스트, Before→After)
   ## Section 3. 타겟 & 상황
   ## Section 4. 크리에이티브 고정값
   ```
3. Write 툴로 `$LOCAL_WORK/01_${BRAND_NAME}_brand_asset.md` 로컬 저장
4. Write 툴로 `$NAS_BRAND_MD/${BRAND_NAME}_brand_asset.md` NAS에도 저장 (다음 사용을 위해)

### 2-3: 영상 분석 MD 랜덤 복사 — 반드시 Bash 툴로 실행

```bash
NAS_VIDEO_MD="//192.168.0.103/01_마케팅사업부_직원/{NAS_USER}/3팀 영상 자동화/M1/input/영상 분석 MD"

VIDEO_FILES=($(ls "$NAS_VIDEO_MD" 2>/dev/null | grep -iv "$BRAND_NAME"))
VIDEO_COUNT=${#VIDEO_FILES[@]}

if [ "$VIDEO_COUNT" -eq 0 ]; then
  echo "❌ 영상 분석 MD 없음 — 중단"
  exit 1
fi

RANDOM_IDX=$((RANDOM % VIDEO_COUNT))
SELECTED_VIDEO="${VIDEO_FILES[$RANDOM_IDX]}"

cp "$NAS_VIDEO_MD/$SELECTED_VIDEO" "$LOCAL_WORK/02_${SELECTED_VIDEO}"
echo "✅ 영상 분석 MD 복사: 02_${SELECTED_VIDEO}"
```

### 2-4: 로컬 작업 폴더 검증

```bash
echo "=== 로컬 작업 폴더 확인 ==="
ls "$LOCAL_WORK"
```

✅ 검증: `01_` `02_` 파일 + `03_${OUTPUT_BASE}.md` 3종 존재 확인. 미달 시 해당 단계 재실행.

---

## STEP 3: 로컬 작업 폴더 → NAS output 복사 — 반드시 Bash 툴로 실행

```bash
NAS_OUTPUT="//192.168.0.103/01_마케팅사업부_직원/{NAS_USER}/3팀 영상 자동화/M1/output"

cp -r "$LOCAL_WORK" "$NAS_OUTPUT/"
echo "✅ NAS output 복사 완료: $NAS_OUTPUT/$OUTPUT_BASE"
```

---

## STEP 4: 입력 폴더 → NAS done 이동 + 로컬 정리 — 반드시 Bash 툴로 실행

```bash
NAS_INPUT="//192.168.0.103/01_마케팅사업부_직원/{NAS_USER}/3팀 영상 자동화/M1/input/사용자 입력 MD/input"
NAS_DONE="//192.168.0.103/01_마케팅사업부_직원/{NAS_USER}/3팀 영상 자동화/M1/input/사용자 입력 MD/done"

# 원본 입력 폴더 → NAS done (cp+rm, SMB mv 오류 방지)
mkdir -p "$NAS_DONE"
cp -r "$SRC_FOLDER" "$NAS_DONE/"
rm -rf "$SRC_FOLDER"
echo "✅ done 이동 완료"

# 로컬 작업 폴더 삭제
rm -rf "$LOCAL_WORK"
echo "✅ 로컬 작업 폴더 정리 완료"
```

## STEP 5: 완료 안내

아래 형식으로 보고:

```

브랜드: [브랜드명]

✓ 컨셉이미지.[ext]                    (컨셉 이미지, 있을 경우)

다음: /m2 [브랜드명] 실행
```
