# M3 패키징 스킬

## 설정

```
NAS         = //192.168.0.103/01_마케팅사업부_직원/{NAS_USER}/3팀 영상 자동화
M2_OUTPUT   = {NAS}/M2/output
M2_DONE     = {NAS}/M2/output/done
M3_OUTPUT   = {NAS}/M3/output
M3_PRODUCTS = {NAS}/M3/제품이미지
M4_INPUT    = {NAS}/M4/input
M5_INPUT    = {NAS}/M5/input
LOCAL       = C:/Users/{WINDOWS_USER}/Desktop/클로드/m3-work
```

> NAS 접근은 다운(Step 3) + 업(Step 7) 두 번만. 중간 작업은 전부 LOCAL.

---

## Step 1. 실행 모드 결정

| 호출 형식 | 동작 |
|---|---|
| `/m3 브랜드명` | 해당 브랜드 폴더 1개만 처리 |
| `/m3` | M2/output 미처리 폴더 전체 날짜 오름차순 처리 |

---

## Step 2. 미처리 폴더 감지

```bash
ls "{M2_OUTPUT}/"
ls "{M2_OUTPUT}/done/" 2>/dev/null
ls "{M4_INPUT}/" 2>/dev/null
```

- `M2/output/done/` 또는 `M4/input/`에 이미 있으면 처리 완료
- 어디에도 없으면 미처리
- 미처리 없으면 "처리할 M2 아웃풋이 없습니다" 안내 후 종료

---

## Step 3. NAS → 로컬 복사

```bash
mkdir -p "{LOCAL}/{날짜_브랜드}"
cp -r "{M2_OUTPUT}/{날짜_브랜드}/." "{LOCAL}/{날짜_브랜드}/"
ls "{LOCAL}/{날짜_브랜드}/"
```

복사 후 파일 확인:
- `04_*_기획안.md` ✓
- `scenes.json` ✓
- `m1_img_*` (컨셉이미지, 있을 경우)

---

## Step 4. 브랜드명·제품명 확정

폴더명 `{날짜_브랜드명}` 또는 `{날짜_브랜드명_제품명}` 에서 파싱.
- `_`로 분리 후 날짜(8자리 숫자) 제외
- 예: `20260511_나잇사렌_세럼` → 브랜드=`나잇사렌`, 제품=`세럼`
- 예: `20260511_나잇사렌` → 브랜드=`나잇사렌`, 제품=`null`

이후 모든 작업은 로컬 폴더 기준.

---

## Step 5. 스크립트 TXT 생성

`scenes.json`의 `script` 필드를 `id` 순으로 추출 → `{날짜_브랜드}.txt` 로컬 저장.

```python
import json, pathlib

local_dir = pathlib.Path(r"C:/Users/{WINDOWS_USER}/Desktop/클로드/m3-work/{날짜_브랜드}")
scenes = json.loads((local_dir / "scenes.json").read_text(encoding="utf-8"))
scripts = [s["script"] for s in sorted(scenes, key=lambda x: x["id"])]
(local_dir / "{날짜_브랜드}.txt").write_text("\n".join(scripts), encoding="utf-8")
```

생성 후 파일 내용 확인 (한 줄씩 스크립트만 출력).

---

## Step 6. 제품 이미지 복사

제품이미지 폴더 파일명은 `브랜드_제품_번호` 형식. 한글 또는 영문일 수 있으므로 둘 다 검색.

**한글↔영문 매핑 테이블:**

| 한글 | 영문 |
|------|------|
| 모미차 | momicha |
| 나잇사렌 | nightsaren |
| 로쌩 | lossang |
| 메디티엔 | meditn |
| 레피넬 | refinelle |
| 엔케이365 | NK365 |
| 프리우먼 | freewoman |

**검색 패턴 결정:**
- `제품명 = null` → `{브랜드명}_*` 패턴으로 검색 (브랜드 전체)
- `제품명` 있음 → `{브랜드명}_{제품명}_*` 패턴으로 검색

```bash
# 제품명 있을 때: 브랜드_제품 패턴
PATTERN="{브랜드명}_{제품명}"   # 또는 제품명 null이면 "{브랜드명}"

# 1차: 한글 패턴으로 검색
MATCHES=$(ls "{M3_PRODUCTS}/" | grep -i "$PATTERN" | grep -iE "\.(png|jpg|jpeg)$")

# 결과 없으면 매핑 테이블에서 영문명 조회 후 재검색
if [ -z "$MATCHES" ]; then
  MATCHES=$(ls "{M3_PRODUCTS}/" | grep -i "{영문패턴}" | grep -iE "\.(png|jpg|jpeg)$")
fi
```

매칭 파일 전체 로컬 복사 (Thumbs.db, .ini 제외). 매칭 결과 없으면 "제품이미지 없음" 경고 출력 후 계속 진행.

```bash
for f in $MATCHES; do
  cp "{M3_PRODUCTS}/$f" "{LOCAL}/{날짜_브랜드}/"
done
```

## Step 8. 파일명 정리

로컬 폴더의 이미지 파일을 표준 이름으로 변경.

**① 컨셉이미지 → `컨셉이미지.[ext]`**
```bash
for f in "{LOCAL}/{날짜_브랜드}"/m1_img_*; do
  [ -f "$f" ] || continue
  ext="${f##*.}"
  mv "$f" "{LOCAL}/{날짜_브랜드}/컨셉이미지.$ext"
done
```

**② 제품이미지 → `제품이미지_{브랜드명}.[ext]` 또는 `제품이미지_{브랜드명}_{제품명}.[ext]` (여러 개면 `_1`, `_2` 순번)**

파일명 접미사 결정:
- `제품명 = null` → `제품이미지_{브랜드명}`
- `제품명` 있음 → `제품이미지_{브랜드명}_{제품명}`

```bash
PREFIX="제품이미지_{브랜드명}"  # 제품명 있으면 "제품이미지_{브랜드명}_{제품명}"

i=1
for f in "{LOCAL}/{날짜_브랜드}"/제품이미지원본파일*; do
  [ -f "$f" ] || continue
  ext="${f##*.}"
  if [ "$i" -eq 1 ]; then
    mv "$f" "{LOCAL}/{날짜_브랜드}/${PREFIX}.$ext"
  else
    mv "$f" "{LOCAL}/{날짜_브랜드}/${PREFIX}_${i}.$ext"
  fi
  i=$((i+1))
done
```

rename 후 `ls "{LOCAL}/{날짜_브랜드}/"` 확인. 이미 표준 이름이면 skip.

## Step 9. NAS 업로드 (3곳 동시)

### M3/output (전체 보관)
```bash
mkdir -p "{M3_OUTPUT}/{날짜_브랜드}"
cp -r "{LOCAL}/{날짜_브랜드}/." "{M3_OUTPUT}/{날짜_브랜드}/"
```

### M4/input (TTS용)
```bash
mkdir -p "{M4_INPUT}/{날짜_브랜드}"
cp "{LOCAL}/{날짜_브랜드}/04_*_기획안.md" "{M4_INPUT}/{날짜_브랜드}/"
cp "{LOCAL}/{날짜_브랜드}/scenes.json" "{M4_INPUT}/{날짜_브랜드}/"
cp "{LOCAL}/{날짜_브랜드}/{날짜_브랜드}.txt" "{M4_INPUT}/{날짜_브랜드}/"
```

### M5/input (이미지 생성용)
```bash
mkdir -p "{M5_INPUT}/{날짜_브랜드}"
cp "{LOCAL}/{날짜_브랜드}/scenes.json" "{M5_INPUT}/{날짜_브랜드}/"
cp "{LOCAL}/{날짜_브랜드}/제품이미지_{브랜드명}"* "{M5_INPUT}/{날짜_브랜드}/"
cp "{LOCAL}/{날짜_브랜드}/컨셉이미지."* "{M5_INPUT}/{날짜_브랜드}/" 2>/dev/null || true
```

업로드 후 확인:
```bash
ls "{M3_OUTPUT}/{날짜_브랜드}/"
ls "{M4_INPUT}/{날짜_브랜드}/"
ls "{M5_INPUT}/{날짜_브랜드}/"
```

---

## Step 10. done 이동 및 로컬 정리

```bash
# M2/output → done (cp+rm, SMB mv 오류 방지)
mkdir -p "{M2_DONE}"
cp -r "{M2_OUTPUT}/{날짜_브랜드}" "{M2_DONE}/"
rm -rf "{M2_OUTPUT}/{날짜_브랜드}"

# 로컬 작업 폴더 → done
mkdir -p "{LOCAL}/done"
mv "{LOCAL}/{날짜_브랜드}" "{LOCAL}/done/{날짜_브랜드}"
```

