## 🔁 설득 구조 5단 템플릿

모든 29개 영상이 아래 구조 반복:

```
[HOOK 0-4s]
이목 끌기 + 타겟 페인 공감
→ ex: "10년 묵은 기미가 7일만에 녹아내린 한국 크림"

[PAIN 4-10s]
기존 실패 증폭
→ ex: "피부과에 300만원 썼는데 소용없었다"

[PRODUCT 10-20s]
제품명 + 성분 소개
→ ex: "메라블 루비알엔 피코샷, 루비 PDRN + 골드필름"

[PROOF 20-30s]
B&A + 통계 + 유명인 증거
→ ex: "1일차 vs 7일차 / 100명 중 98명 효과 / 한국 연예인도 난리"

[CTA 30-35s]
긴급성 + 링크
→ ex: "24번 완판됐으니 서두르세요"
```

## 🎭 톤 매트릭스

| 톤 | 적합 타겟 | 장점 | 단점 |
|---|---|---|---|
| 에너제틱 | 20~30대 활기찬 | 높은 리텐션 | 50대엔 과할 수 있음 |
| 고백형 | 감성 타겟 | 깊은 공감 | 느린 템포 |
| 전문가 | 성분 의식형 | 신뢰 |ᅟ 딱딱함 |
| 가십 | 가벼운 시청자 | 재미 | 제품 이미지 저하 가능 |
| 싸시/드라마 | Z세대 | 공유 유발 | 브랜드 톤 어긋날 수 |
| 뉴스 앵커 | 중년 | 객관성 느낌 | 창의성 낮음 |
| 따뜻한 가족 | 40~60대 | 구매 전환 | 반복되면 신선도↓ |

## 💡 재활용 팁

1. **같은 훅 × 다른 페르소나** = 4~6배 확장 가능
2. **같은 페르소나 × 다른 훅** = A/B 테스트 용도
3. **훅만 바꿔서 30초짜리 기존 제품 재활용 가능** (Avatar 유지, 자막/카피만 변경)
