# 01. 기획서 — 외국인 인플루언서 리뷰 영상 자동화

## 🎯 비즈니스 목표

**40~60대 한국 여성 타겟의 기미 크림 광고**를 외국인 인플루언서 포맷으로 제작. 한국 소비자가 "해외에서도 인정받는 한국 제품"이라는 사회적 증거를 체감하도록 유도.

## 👥 타겟 세그먼트

| 1차 타겟 | 2차 타겟 |
|---|---|
| 40~60대 여성 | 30대 후반 예비 기미 걱정층 |
| 10년 이상 기미 고민 | 피부과 시술 실망 경험자 |
| 피부과 비용 부담 | 해외/한국 모두 관심 |

## 🌍 페르소나 다양성 전략

**"우리 제품이 전 세계 인플루언서들에게 통한다"** 라는 메시지를 29개 독립 페르소나로 반복 주입.

### 페르소나 축 (Dimensions)
```yaml

ethnicity:     [백인, 흑인, 황인-한국계, 황인-동남아, 라틴계, 중동계, 혼혈-흑백, 혼혈-아시아-백인]
age_range:     [20대 초반, 20대 후반, 30대 초반, 30대 후반]
vibe:          [에너제틱 뷰티, 시크 패션 에디터, 사사한 GRWM, 스킨케어 덕후, Y2K 아이돌,
                샤이 ASMR, 헬스 브로, 소프트 에스테틱, 드래그 퀸 에너지]
setting:       [화장대, 욕실 거울, 차 운전석, 호텔방 침대, 카페 창가, 공원 벤치,
                해변, 엘리베이터, 호텔 루프탑, 뉴욕 길거리, 도쿄 시부야, 파리 발코니]

product_hold:  [손에 들고 가까이, 테이블 위 포인팅, 얼굴 옆 포즈, 손등에 바르기]
```

## 📐 영상 포맷 표준

| 항목 | 규격 |
|---|---|
| 해상도 | 1080×1920 (9:16 세로) |
| 길이 | 30~45초 (레퍼런스 평균 44초) |
| FPS | 30 |
| 코덱 | H.264 (비디오) + AAC (오디오) |
| 비트레이트 | ~2.6Mbps |
| 자막 | 한국어 위(크게) + 영어 아래(작게·노랑 이탤릭) |
| 상단카피 | 검정 바 520px + 빨강(위)/노랑(아래) 2줄 Arial Black |

## 🎬 스토리 구조 (5단 HOOK-PAIN-PRODUCT-PROOF-CTA)

| 단계 | 시간 | 비중 |
|---|---|---|
| **HOOK** | 0~4s | 충격/공감 발언 |
| **PAIN** | 4~10s | 기존 실패 경험 |
| **PRODUCT** | 10~20s | 제품 소개 + 성분 |
| **PROOF** | 20~30s | B&A + 통계 + 유명인 |
| **CTA** | 30~35s | 링크/품절/긴급성 |

## 🎨 시각 다양성 전략

각 영상 = 메인 립싱크 아바타 + **4개 씬 전환 B-roll**:
1. **PAIN 씬**: 욕실/화장대에서 걱정스러운 클로즈업
2. **PRODUCT 씬**: 침대/의자에 앉아 제품 자랑
3. **APPLY 씬**: 화장대에서 크림 바르기
4. **CTA 씬**: 복도/전신 샷에서 윙크/포인팅

## 📊 측정 지표 (향후 광고 운영 시)

| KPI | 측정 |
|---|---|
| CTR | 썸네일 클릭률 |
| Hook retention | 3초 이탈률 |
| VTR | 완료율 (30초) |
| CVR | 링크 클릭→구매 전환 |
| ROAS | 광고비 대비 매출 |

## 🔄 반복 생산 가능성

**배치 파이프라인 1회 구축 완료** → 신규 제품/브랜드에 재사용 가능:
- USP 엑셀 파싱 로직 재사용
- 페르소나 풀 확장 가능
- 스크립트 YAML만 교체하면 N편 자동 제작
