# 04. 자막 가이드 — 한국어 중심 이중 자막

## 🎯 기본 구조

```
┌─────────────────────────────┐  1080×1920 (9:16)
│  [검정 바 520px]            │
│   빨강 카피 (Arial Black 86) │  상단 고정 카피 1줄
│   노랑 카피 (Arial Black 86) │  상단 고정 카피 2줄
├─────────────────────────────┤
│                             │
│                             │
│       아바타 영상 1080×1080  │
│       (또는 B-roll)          │
│                             │
│                             │
├─────────────────────────────┤
│   한국어 자막 (Malgun 78)    │  씬별 이중 자막
│   English sub (Arial 56)    │  (씬당 2~3 청크 분할)
└─────────────────────────────┘
```

## 🎨 상단 고정 카피

| 속성 | 값 |
|---|---|
| 위치 | 검정 바 내부 (y=0~520px) |
| 폰트 | Arial Black |
| 크기 | 86px |
| 1줄 색 | 빨강 (&H000000FF) - 상단 (MarginV=200) |
| 2줄 색 | 노랑 (&H0000FFFF) - 하단 (MarginV=298) |
| 외곽선 | 검정 3px |
| 행간 | 98px (1줄 → 2줄 간격) |
| 노출 | 영상 전체 기간 (고정) |
| 자수 | 줄당 20~25자 |

## 🎙️ 씬별 이중 자막

| 속성 | 한국어 (주) | 영어 (보조) |
|---|---|---|
| 폰트 | Malgun Gothic | Arial |
| 크기 | 78px (기준 68 × 1.15) | 56px (기준 68 × 0.82) |
| 색상 | 흰색 + 검정 외곽 4px | 노란색 + 검정 외곽 2px |
| 스타일 | Bold | Italic |
| 위치 | 하단 중앙 (MarginV=380) | 하단 중앙 (MarginV=280) |
| 노출 | 해당 음성 구간만 | 동일 |

### 청크 분할 규칙 (Option B)

**씬 단위로 2~3개 청크 분할**:

```yaml
# Scene: 오디오 4.5초, 영어 "POV: I walk into my derm office with the same damn spots."
chunks:
  - en: "POV: I walk into my derm office"
    ko: "POV: 피부과 또 갔어요"
    # 시간 = 4.5 × (32/62) = 2.32초
  - en: "with the same damn spots."
    ko: "같은 기미를 안고서"
    # 시간 = 4.5 × (30/62) = 2.18초
```

타임스탬프는 **청크 글자수 비례 분배** (Whisper 없이 근사치).

## 📝 한국어 번역 원칙 — 중년(50~60대) 친화

### ✅ 사용 OK
- 존댓말 완성형: `~이에요`, `~습니다`, `~하세요`
- 표준어 명사: 크림, 효과, 피부, 기미
- 한국 단위: `6만원`, `300만원` (not `$60`, `$3000`)
- 브랜드/성분: 메라블 루비알엔, PDRN, 나이아신, 트라넥사믹

### ❌ 금지

| 스킨 | 피부 |
| 인박스 폭발 | 반응이 뜨거웠어요 |

## 🖥️ FFmpeg 자막 burn

```bash
ffmpeg -i padded.mp4 \
  -filter_complex "[overlays...];[last]ass='subs.ass'[v]" \
  -map "[v]" -map 0:a \
  -c:v libx264 -crf 20 -c:a aac \
  final.mp4
```

**핵심**: ASS burn을 filter_complex 체인의 **마지막**에 배치 → overlay가 자막을 가리지 않음.

## 🎨 타겟별 권장 자막 강도

| 타겟 | 한국어 크기 | 영어 크기 | 상단 카피 |
|---|---|---|---|
| 40~60대 | **크게** (78+) | 작게 (56) | 매우 크게 (86+) |
| 20~30대 | 중간 (64) | 중간 (54) | 크게 (72) |
| 해외 영어 | 작게 (54) | 크게 (68) | 영어로 |

## 💡 실전 팁

1. **씬당 1줄 권장** — 2줄 넘으면 시청자 따라가기 힘듦
2. **한국어 15자 이내** — 짧게 끊어 빠른 리듬 유지
3. **Bold 필수** — 배경 위 가독성 위해
4. **노란색 영어** — 흰 바탕 오브젝트에서도 구분됨
5. **MarginV 여유** — 하단 플로팅 PIP과 겹치지 않게 380/280 조정
